# 11장. 전투
# Chapter 11. Combat — 교전 규칙(交戰規則)

HP는 체력이 아닙니다. 총알이 스쳐가고, 파편을 간신히 피하고,
폭발에서 굴러나온 것입니다. HP가 0이 되는 순간 —
비로소 진짜 맞기 시작합니다.

---

## 11-1. 전투의 원칙

- 차원계 사망 = 드랍아웃. 물질계 사망 = 진사(영구 퇴장).
- 전투는 최후의 수단. 현명한 러너는 싸우기 전에 준비합니다.
- NPC는 죽기 싫어합니다. 사기(Morale)가 꺾이면 도주합니다.
- 전투는 3라운드 안에 끝나야 합니다. 안 끝나면 시계가 찹니다.

---

## 11-2. 전투 라운드

1라운드 = 약 6초.

1. 기습 판정 — 무방비 시 기습 라운드. Notice 체크.
2. 선공 — 진영별 1d8 + 최고 신(REF) 수정치. PC 동점 승.
3. 행동:
   - 주행동(Main) 1회 — 공격, 태그 활성화, 스킬 사용.
   - 이동(Move) 1회 — 10m. 주행동 포기 시 더블 무브.
   - 부수행동(On Turn) 다수 — 짧은 말, 엎드리기.
   - 즉시행동(Instant) — 운(LCK) 토큰 사용. 타인 턴에도.
4. 라운드 종료 — 시계 업데이트. 사기 체크 트리거 확인.

---

## 11-3. 공격과 피해

### 공격 판정

> 공격 = 1d20 + AB + 스탯 수정치 + 전투 스킬 >= 대상 AC
>
> 근접: 체(BOD) + Stab 또는 Punch
> 사격: 신(REF) + Shoot
> 신기: 감(SEN) + 신호(Signal)
>
> 자연 1 = 항상 빗나감. 자연 20 = 항상 명중 + 자동 치명 판정.

- 기본 AC = 10 + 신(REF) 수정치 + 갑옷 + 임플란트.
- 엄폐: +2(반엄폐) / +4(완전엄폐).
- 환경 태그 활성화: +2 또는 적 -2 (11-7절 참조).

### 피해

명중 시 무기 피해 주사위 + 수정치.
- 근접: 피해 + 체(BOD) 수정치.
- 사격: 피해만(수정치 없음).
- 신기: 피해 + 감(SEN) 수정치 x (BP% / 100).

### Shock(충격 피해) — 빗나가도 주는 압박

근접 무기만 적용. 빗나갔어도 대상 AC가 무기 Shock AC 이하이면
Shock 수치 + 체(BOD) 수정치만큼 피해. 치명타 유발 안 함.

### 치명타 — 폭발하는 한 발의 도박

> 명중 시 치명 주사위(Trauma Die)를 추가로 굴림.
> 치명 주사위 >= 치명 목표치(기본 6) → 치명타 확정.
>
> 치명 주사위는 폭발(Exploding)한다.
> 최고값이 나오면 재굴림하여 가산. 연쇄 가능.
> 예: d6 → 6! 재굴림 → 4 → 합산 10. 치명타 확정.
>
> 치명타 시: 피해 x 치명 배수(통상 x2, 대형 무기 x3).
> "평범한 권총 한 발이 전투를 뒤집을 수 있다."

## 11-4. HP=운, 돌파, 사망

### 피해 적용 흐름

명중 → 피해 → 갑옷 HP에서 먼저 차감 → 초과분이 캐릭터 HP에.

- 갑옷 HP: 전투 종료 후 자동 복원 (다음 전투 시 전량).
- HP > 0: "운으로 피했다." 전투 지속.
- HP = 0: 돌파(Breakthrough).

### 돌파 — 진짜 맞기 시작

HP 0 → 즉시 취약(Shaken) 상태.
이후 피해는 체(BOD) 스탯에서 직접 차감.

- 체(BOD) >= 1: 허약(Frail). 전투 지속 가능하지만 자연 치유 불가.
- 체(BOD) = 0: 치명 판정.

### 치명 판정

- 물질계 → 치명상(Mortal Wound). 6라운드 안정화. 실패 시 진사.
- 차원계 → 드랍아웃. 코핀 귀환. HP 1, 에고 -1, 배제 1개월.

### 취약(Shaken) 상태

다음 턴 시작 시 정신 내성 체크. 성공 시 해제.
실패 시 이번 턴 주행동 불가(이동+부수만).
동료의 Lead 체크(난이도 8)로도 해제 가능.

### 안정화

혜(INT)/Heal 체크. 난이도 = 8 + 경과 라운드. 메딕키트 없으면 +2.
성공 → 10분 후 HP 1. 허약 상태. 실패 → 6라운드 후 사망.

### 중상 테이블 (같은 전투에서 치명타 + 돌파가 겹칠 때)

체력 내성 실패 시 1d12:

| 결과 | 효과 |
|------|------|
| 1 | 즉사 |
| 2 | 내부손상(코마) |
| 3 | 뇌손상 |
| 4 | 시력상실 |
| 5 | 내장파열(BOD-1) |
| 6-7 | 사지파괴 |
| 8-9 | 임플란트 파손 |
| 10 | 에고손상(에고-2 영구, 매팅+1) |
| 11-12 | 피부만 찢어짐 |

## 11-5. 잡졸과 네임드 — NPC 이원 체계

### 잡졸(Extra) — "이름이 없으면 잡졸"

- HP 1. 어떤 피해든 1점 이상이면 즉시 전투 불능.
- 치명타/중상 없음. 맞으면 쓰러지고 끝.
- 사기 체크 대상. 실패 시 즉시 도주/항복.
- 떼 공격: 4명까지 합동. +2 명중 / +1 피해 (인당, 최대 +6/+3).
- 예: 거리 깡패, 하급 경비, 갱단 졸개.

### 네임드(Wild Card) — "이름이 있으면 네임드"

- 정규 HP. 치명타, 중상, 안정화 전부 적용.
- 고유 능력 보유 가능. 사기치 9+ 또는 면제.
- 예: 수호사, 보스급 흑전사, 유명 신수사, 미명존재.

잡졸이 극적 행동을 하면 GM이 네임드로 승격 가능.

---

## 11-6. 사기(Morale) — 적은 죽기 싫어한다

NPC만 해당. PC는 면제.

체크 시점: 첫 아군 사망 / 절반 쓰러짐 / 리더 쓰러짐 / 압도적 위협.

2d6 >= 사기치. 실패 → 도주 또는 항복.
- 잡졸 기본: 7. 네임드: 9. 수호사/군인: 10. 엘리트: 11.
- 광폭화/기계/미명존재: 면제.
- 수정: 수적 우위 +1, 리더 생존 +1, 절반 쓰러짐 -2, 리더 쓰러짐 -2.

---

## 11-7. 환경 태그(Aspect) — 전장은 무기다

전투 시작 시 GM이 환경 태그 1–3개 선언.
미션 태그(Book 3, 14장)에 따라 자동 부여되는 태그도 있음.

태그 흐름: 발견(Notice) → 활성화(스킬 체크) → 효과(+2 또는 -2)

- 발견: Notice 또는 관련 스킬(난이도 8). 야전 분석은 자동 발견.
- 활성화: 관련 스킬 체크(난이도 10)를 주행동으로 수행.
- 효과: 아군 +2 또는 적 -2. 1전투 1태그 1회.
- 새 태그 생성: 환경 변화 시 GM이 추가. 즉시 활성 가능.

예시:
- [구형 신기 배선] → 신호 체크 → 조명 폭발 → 적 사격 -2.
- [좁은 복도] → 폭발물 피해 +2 또는 방패 진형 AC +2.
- [흑공 잔류] → 현화술 체크 → 적 매팅+1 강제 (시전자도 위험).
- [폐차 바리케이드] → Exert 체크 → 1–2명에게 1d6+엎드림.

## 11-8. 시계(Clock) — 시각적 긴장 장치

원형을 N칸으로 나누고 조건마다 1칸 채움. 다 차면 발동.
테이블 중앙에 공유 배치. 모든 플레이어가 볼 수 있음.

| 시계 | 칸수 | 채움 / 발동 |
|------|------|------------|
| 증원 시계 | 4 | 총소리/폭발 +1 / 증원 도착 |
| 흑공분출 시계 | 6 | 현화/신기 과잉 +1 / 분출 발생 |
| 패러독스 시계 | 8 | 능력/인터렉션 +1 / 먹개비 판정 |
| 신폭 시계 | 4 | 신기 과잉집중 +1 / 신폭 발생 |
| 탈출 시계 | 4 | 매 라운드 +1 / 탈출로 봉쇄 |

미션 태그 연동: "흑공분출" → 분출 시계 자동. "잠입" → 증원 시계 자동.

## 11-9. 푸시(Push) — "더 밀면 더 위험해진다"

스킬 체크(2d6) 실패 시 "푸시" 선언. 재굴림하되 대가 1개 발생.
- 1턴 1회. 공격 판정(d20)에는 불가. 스킬 체크에만.
- 결과와 관계없이 대가는 반드시 발생.

| 행위 유형 | 대가 |
|----------|------|
| 임플란트/장비 사용 | 시스템 부하 +1 |
| 신기 사용 | 셀 소모 x2 (이번 행위만) |
| 정신적 행위 | 에고 -1 |
| 차원계 행위 | 패러독스 +1 (시계 +1) |
| 물리적 행위 | HP -1d4 |
| 사회적 행위 | NPC 호감도 -1 |

> "성공했지만 대가가 따른다" = FEWK의 핵심 테마.

## 11-10. 운(LUCK) 토큰

운(LCK) 토큰(이하 '운 토큰').
세션 시작 시: 운(LCK) 수정치 + 2개 수령 (최소 1개).
미사용 토큰은 세션 간 이월 불가.

| 용도 (즉시행동) | 효과 |
|----------------|------|
| 재굴림 | d20 또는 2d6 1회 재굴림 |
| 피해 반감 | 피격 시 피해 절반 |
| 치명타 무효 | 치명 성공을 강제 실패로 |
| 패러독스 상쇄 | 패러독스 1점 즉시 소거 |
| 사기 강제 | 적 네임드 1체 즉시 사기 체크 |
| 배제 단축 | 드랍아웃 배제 기간 1주 단축 |

1턴 1토큰 한정. GM도 네임드 NPC에 1–2개 부여 가능.

## 11-11. 신기 전투

- 셀 소모: 미세(100) / 소규모(1,000) / 중규모(10,000) / 대규모(100,000) / 재앙급(1,000,000).
- 신기 무기 강화: 주행동. 피해 + 감(SEN) 수정치. 1,000셀/라운드.
- 의도적 신폭: Lv3+. 10,000셀. 반경 5m. 3d6+감 수정치.
- 과잉 사용(1라운드에 용량 50% 이상) → 신폭 시계 +1.
- 생체순도 보정: 신기 피해 x (BP% / 100).
- 신수 전투: 신수 HP = 신기용량 / 100. 유지 1,000셀/라운드. 신수 소멸 시 신수사 정신 내성(실패 → 1라운드 기절).

## 11-12. 이원 사망 — 드랍아웃과 진사

### 차원계 사망 = 드랍아웃(Drop-out)

코핀 귀환. HP 1. 에고 -1(1일). 시스템 부하 = 최대.
해당 계 배제 1개월. 배제 중 재진입 → 패러독스 +5 + 먹개비 위험.

### 물질계 사망 = 진사(True Death)

영구 사망. 캐릭터 퇴장.

### 네이티브 사망 = 진사

인공계 고유 주민. 드랍아웃 불가. 돌아갈 곳이 없음.

### 동시 전투 (물질계 + 차원계)

물질계 육체 피격 → 차원계 에고 -1/피격.
물질계 사망 → 강제 드랍아웃 + 진사.

---

## 11-13. 사회적 전투 — 에고 대미지

물리 전투와 같은 프레임워크.
- 공격: 격(PRE) + Talk(설득/기만) 또는 Lead(위압/명령).
- 방어: 정신 내성 또는 격(PRE) + 관련 스킬.
- 피해: 에고에 적용. 1d4(일반) – 1d8(전문 심리전).
- 에고 0 = 코마 또는 완전 굴복.
- 환경 태그 적용: [공개 장소] 위협 → +2.

---

## 11-14. 차량 전투와 추격

- 차량 HP, AC 별도. 탑승 무기: 주행동 사격.
- 추격: 혜(INT)/Drive 대항 체크. 3회 성공 시 따라잡기/도주.
- 오토바이 도심 추격: Drive +1.

---

## 11-15. 치유와 시스템 부하

- 자연 치유: 1박 후 레벨 수치 HP 회복. 허약 시 불가.
- 체(BOD) 피해: 1주 요양으로 1점 회복.
- 응급 처치: Heal 체크 → 1d6+Heal 레벨 HP. 시스템 부하 +1.
- 시스템 부하: 최대 = 체(BOD). 1박 -1. 최대 도달 → 치유/약물 불가.
- 허약(Frail): 1주 요양 + Heal 체크(난이도 10)로 해제.

## 11-16. 추격전 확장 규칙 Extended Chase Rules

기본 추격(11-14절)은 Drive 대항 체크 3회이지만,
차량 위 격투/차량 간 점프/폭발 등은 이 확장 규칙을 사용합니다.

### 추격 라운드 구조

1. 조종사 체크: 혜(INT) 또는 신(REF) + Drive >= 상대 결과.
   성공 → 추격 거리 1단계 접근(또는 이탈).
   실패 → 거리 변화 없음. 대실패(자연 1) → 사고 판정.
2. 탑승자 행동: 주행동 1회 (사격, 점프, 해킹, 신기 등).
   - 이동 중 사격: 사격 -2 페널티.
   - 차량 간 점프: 체(BOD) + Exert >= 10. 실패 → 1d10 낙하 피해.
   - 전투 중 차량 폭발: 탑승자에게 3d6 피해. 체력 내성으로 절반.
3. 환경 이벤트 (GM이 d6으로 결정, 선택적):
   - 1: 장애물 — 급제동 Drive 체크(난이도 10).
   - 2: 좁은 길 — 대형 차량 진입 불가. 소형만 통과.
   - 3: 보행자 — 회피 Drive 체크. 실패 시 히트 +2.
   - 4: 연료/셀 경고 — 3라운드 후 정지.
   - 5-6: 특이사항 없음.

### 추격 거리

원거리(3) → 중거리(2) → 근거리(1) → 접촉(0).
- 원거리: 사격 -4. 육안 식별 어려움.
- 중거리: 사격 -2. 기본 추격 거리.
- 근거리: 사격 페널티 없음. 차량 간 점프 가능.
- 접촉: 충돌 가능. 근접 공격 가능. 탑승자 간 격투.

### 추격 종료 조건

- 거리 0에서 "제압" — 접촉 후 충돌/정지/탑승.
- 거리 4 이상 — 완전 이탈. 추격 종료.
- 추격 시계(4칸): 경찰/수호사 추격 시 매 라운드 +1. 4칸 차면 도로 봉쇄. 새로운 상황 전환.

## 11-17. 흑공분출 중 전투 Combat During Tenebre Eruption

미션 태그 "흑공분출"이 걸린 전투에서 사용합니다.
흑공분출 시계(6칸)가 자동 활성화됩니다.

### 흑공분출 환경 효과

매 라운드 종료 시 GM이 d6 환경 변이 테이블을 굴림:

| d6 | 효과 |
|----|------|
| 1 | 지형 변이 — 바닥이 늪, 벽에서 가시, 천장 녹음. 이동력 절반. 새 환경 태그 [변이 지형] 자동 부여. |
| 2 | 현화 파동 — 반경 10m 모든 존재 현화 판정. 기연 내성 실패 시 매팅 트랙 +1. |
| 3 | 신기 역류 — 신사의 신기용량이 일시 x2(1라운드). 하지만 다음 라운드 용량 0으로 리셋. |
| 4 | 괴물 출현 — 야생 신수 또는 현화 괴물 1d4체 출현. 잡졸(HP 1). 무차별 공격. |
| 5 | 시야 차단 — 흑연기 확산. 10m 이상 시야 0. 사격 불가(근거리만). 1d4 라운드 지속. |
| 6 | 소강 — 이번 라운드 특이사항 없음. 시계 +1 대신 면제. |

### 흑공분출 시계 만땅(6칸) 시

대규모 분출. 모든 존재 즉시 기연 내성 체크.
- 성공: 1d6 피해 + 매팅 트랙 +1.
- 실패: 3d6 피해 + 매팅 트랙 +2 + 환경 태그 전부 리셋.
- 분출 후 시계 0으로 리셋. 새 시계 시작.

### 레인저/야전교수의 역할

- 레인저: 분출 시계 진행 시 "대피 유도" 가능. Lead 체크(난이도 10) 성공 → 잡졸 NPC 전원 안전 대피.
- 야전교수: "분출 분석" 가능. Know 체크(난이도 10) 성공 → 다음 환경 변이 테이블 결과를 미리 알림. 대비 가능.

## 11-18. 최후의 연주 Last Performance

도시급 흑공분출에 맞서는 최후의 방어선.
이것은 전투가 아닙니다. 이것은 공연입니다.

### 발동 조건

- 도시급 이상의 흑공분출이 감지됨 (흑공분출 시계 6칸 x 3회 이상).
- 도시의 가장 높은 건물의 공연 설비가 가동됨.
- 최후의 연주자 라이선스 소지자가 무대에 섬.

### 연주 판정

주행동으로 매 라운드 연주합니다.

> 연주 체크: 2d6 + 격(PRE) + Perform >= 난이도
>
> 난이도 10: 소규모 흑공 소멸 (반경 10m).
> 난이도 12: 중규모 (반경 100m).
> 난이도 14: 대규모 (반경 1km). 도시급 위기 해소.
>
> 성공마다: 흑공이 음악에 반응하여 격렬하게 소멸.
> 실패마다: 연주자에게 2d6 피해 + 매팅 트랙 +1.
>
> 연주자는 매 라운드 피해를 받습니다.
> 흑공이 소멸하는 과정에서 연주자의 존재와 반응하기 때문.
> 라운드당: 1d6 피해 (난이도 10) / 2d6 (난이도 12) / 3d6 (14).
>
> 연주자의 HP가 0이 되면 → 돌파. 체(BOD)에서 직접 피해.
> 체(BOD)가 0이 되면 → 진사. 하지만 연주는 완성됩니다.
>
> "그 과정에서 연주자가 살아남을 가능성은... 없어 보입니다."

### 동료의 지원

- 신기사가 에너지 공급: 연주자의 HP 피해를 1d6 경감/라운드. 10,000셀/라운드 소모.
- 수호사가 수호 선언: 연주자 대신 피해를 받을 수 있음.
- 다른 연주자가 합주: 난이도 -2. 하지만 피해를 나눠 받음.
- 교수가 분석 지원: 연주 체크에 +1.

### 성공 시

흑공이 음악에 반응하며 폭발하듯 퍼져 소멸합니다.
도시는 살아남습니다. 연주자는... GM 판단.

이 장엄한 광경은 수많은 예술작품으로 오마주되었으며,
모든 문명 거주자들이 예술인을 존경하는 기반을 마련했습니다.

### RPG에서의 사용

이 규칙은 캠페인의 클라이맥스에서 사용합니다.
일반 미션에서 사용하는 규칙이 아닙니다.
GM은 이 순간이 올 때까지 충분한 서사적 준비를 해야 합니다.
PC가 연주자라면, 이 장면은 그 캐릭터의 마지막 장면이 될 수 있습니다.
