## 9-1. 인터렉션이란 무엇인가 What Is An Interaction

이 세계에서는 "줌으로 먼 곳을 본다"는 하나의 행위입니다.
타고난 능력으로 볼 수 있고, 훈련으로 배워서 볼 수 있고,
사이버아이를 달아서 볼 수 있습니다.
결과는 동일합니다. 먼 곳이 보입니다.
다른 것은 "어떻게 그걸 갖게 되었는가"와 "쓸 때 뭘 치르는가"뿐.

> 인터렉션 = 무엇을 할 수 있는가 (WHAT)
> 슬롯     = 그것을 어디에 꽂았는가 (WHERE)
>
> 인터렉션은 중립적이고 보편적입니다.
> 효과는 어떤 슬롯에 꽂든 동일합니다.
> 슬롯이 결정하는 것은 취득 비용과 사용 비용뿐입니다.

### 인터렉션의 4가지 분류

1. 능력(能力, Power) — 타고나거나 각성으로 열린 것. 돈으로 살 수 없음. 도구 없이 맨몸으로 작동. 에고를 치름.
2. 기술(技術, Skill) — 배워서 익힌 것. 교육기관, 도제 수련, 흑공샘플 학습. 가장 안정적. 체크 필요.
3. 기능(機能, Function) — 장비를 달아서 얻은 것. 임플란트 이식. 돈으로 구매. 가장 빠르고 안정적. 생체순도 대가.
4. 현상(現象, Phenomenon) — 환경이 일으키는 것. 캐릭터의 슬롯이 아님. 흑공분출로 지역 전체에 신기감지가 생기거나, 현화 폭풍으로 전원 괴수화 위험에 처하는 것. GM이 운용.

캐릭터가 인터렉션을 꽂는 슬롯은 1능력, 2기술, 3기능의 3가지.
이 구조를 '3슬롯(Three Slots)'이라 부릅니다.
같은 인터렉션이 다른 슬롯에 꽂히면 효과는 동일하고 비용만 다릅니다.

귀걸이에도 공격+1이 붙고, 신발에도 공격+1이 붙을 수 있듯이 —
같은 "줌 Lv1"이 능력 슬롯에 꽂히든, 기능 슬롯(사이버아이)에 꽂히든
결과는 동일합니다. 먼 곳이 보입니다. 다른 것은 비용뿐.

| | 능력(Power) | 기술(Skill) | 기능(Func) |
|---|---|---|---|
| 취득 | 선천/각성 | 학습/훈련 | 구매/이식 |
| 오얏 비용 | 없음 | 낮음 | 높음 |
| 사용 대가 | 에고 | 시간/도구 | 셀/부하 |
| 전투 중 속도 | 주행동 | 주행동 | 부수행동 |
| 체크 필요 | 예 | 예 | 대부분 아니오 |
| BP 영향 | 없음 | 없음 | 하락 |
| 희귀도 | 희귀 | 보편 | 보편(돈) |

예: 재생사(능력으로 심상재생) vs 심상프로세서 임플란트(기능으로 심상재생)
효과 동일. 재생사는 에고를 치르고 맨손으로. 임플란트는 BP와 셀을 치르고 자동으로.
이것이 재생사가 쇠락한 이유. 능력이 기능으로 대체됨.

인터렉션 상세 목록은 별도 문서(인터렉션 재설계서)를 참조하십시오.
현상(4)은 환경 태그(11-7절)와 시계(11-8절)로 처리합니다.

RPG 규칙에서:
- 기술(Skill) = 스킬 레벨 0–4. 2d6 체크에 가산. 아래 9-2절 참조.
- 능력(Power) = 클래스 고유 능력(7장) + 인터렉션 목록(별도).
- 기능(Function) = 임플란트 효과(10장). 장비 장착으로 자동 적용.
- 현상(Phenomenon) = GM이 환경으로 부여. 캐릭터 시트에 없음.

## 9-2. 스킬 상세 목록 Skill Details (23종)

스킬 레벨: 0(기초) / 1(전문) / 2(베테랑) / 3(도시 최고) / 4(세계 최고)
체크: 2d6 + 프라임 스탯 수정치 + 스킬 레벨 >= 난이도.
스킬 미보유 시 -1 페널티. 일부 전문 스킬은 미보유 시 시도 불가.

### 범용 스킬 (19종)

Administer (행정) — 혜(INT) — 조직 관리, 서류 처리, 법률 지식, 관료제 파악. 대사국이나 기업과의 공식 업무에 필수.

Connect (인맥) — 격(PRE) — 유용한 사람 찾기, 인맥 활용, 암시장 접근, 외국 문화 이해. Talk 대신 사용 가능(인맥 경유).

Drive (조종) — 신(REF) — 차량, 선박, 비행선, 드론 조종 및 정비. 오토바이 추격, 흑해선 항해, 신단 엔진 선박 운용.

Exert (체력) — 체(BOD) — 달리기, 점프, 수영, 등반, 던지기. 투척 무기 사용 시 전투 스킬로도 적용.

Fix (수리) — 혜(INT) — 장비 제작/수리. 임플란트 정비. 메카트로닉스 장비 진단. 파워셀 호환 개조.

Heal (치료) — 혜(INT) — 의료/심리 치료. 안정화, 질병 진단. 임플란트 이식/유지보수. 바이오트로닉스 시술. 야메(사설 임플란터)들의 핵심 스킬.

Know (지식) — 혜(INT) — 학술/과학 지식. 역사, 희귀 정보 파악. 교수 클래스의 핵심 스킬. 전공별로 보너스 가능.

Lead (지휘) — 격(PRE) — 부하/동료를 이끌기, 사기 유지, 전투 중 지시. Talk은 설득, Lead는 실행시키는 차이.

Notice (관찰) — 감(SEN) — 수색, 매복 탐지, 감정 읽기, 환경 이상 감지. 신기/흑공 감지의 기본 스킬.

Perform (공연) — 격(PRE) — 노래, 연주, 연기, 연설. 최후의 연주자(Last Performancers) 라이선스와 연결. FEWK에서 예술은 사회적 존경과 실용적 가치를 모두 가짐.

Program (프로그래밍) — 혜(INT) — 계산기/컴퓨팅 장치 조작, 해킹, 통신 장비 운용, 암호 해독. FEWK에는 전자 컴퓨터가 없음. 흑공 기반 계산기 조작.

Punch (격투) — 체(BOD) — 맨손 전투. 무투가의 핵심 스킬. 무투가: Punch 레벨을 피해에 가산 + 기(氣) 연동.

Shoot (사격) — 신(REF) — 총기, 투척 무기, 원거리 무기 전투. 돌칩 권총부터 대구경 총까지.

Sneak (은밀) — 신(REF) — 잠입, 변장, 소매치기, 전자 잠금 물리적 해제, 은신.

Stab (근접무기) — 체(BOD) 또는 신(REF) — 근접 무기 전투. 나이프, 검, 신기 무기 등. 투척 무기에도 적용.

Survive (생존) — 감(SEN) — 도시 생존술 또는 야외 생존술. 흑공 분출 지역 대피, 노마드 영역 생활, 식량/물 확보.

Talk (대화) — 격(PRE) — 설득, 매력, 속이기. 상대방이 믿게 만드는 것. 행동하게 만드는 건 Lead.

Trade (거래) — 혜(INT) — 매매, 시세 파악, 희귀품 유통 경로, 밀수, 사업 운영. 다중 화폐 환전에도 사용.

Work (직업기술) — 다양 — 포괄 직업 스킬. 취득 시 직업 지정. 변호, 요리, 회계, 재봉, 농업 등. 해당 직업 수행 및 인맥.

### FEWK 고유 스킬 (4종)

신호 (Signal) — 감(SEN) — 신기 회로 연결/복원/조작. 신단석 배치, 돌칩 수리, 신단 기능 조정. 신사 계열의 핵심 기술 스킬. 신호연결사의 생업.
- Lv0: 돌칩 기본 조작. Lv2: 신단 복원. Lv4: 신단 설계 보조.
- 미보유 시 시도 불가 (전문 스킬).

현화술 (Manifest Art) — 감(SEN) — 의도적 현화 유도, 현물 제조, 도안(Doan) 마킹, 배틀 마킹. 도아너, 배틀마커, 현물제조사의 핵심 스킬.
- Lv0: 기본 도안. Lv2: 전투용 마킹. Lv4: 대규모 현화 설계.
- 사용 시 흑공 또는 오주사 소모. 실패 시 매팅 트랙 +1 위험.
- 미보유 시 시도 불가 (전문 스킬).

러닝 (Running) — 감(SEN) — 차원이동 기술. 코핀 조작. 패러독스 관리, 차원 적응, 월드 템플릿 전환. 러너 클래스의 핵심이지만 다른 클래스도 습득 가능.
- Lv0: 코핀 기본 사용. Lv2: 비표준 차원 진입. Lv4: 자유 러닝 보조.

심상 (Image) — 감(SEN) — 심상기록/재생/전이. 복제기 조작, 영화상 카드 재생, 정밀 이미지 출력. 재생사의 핵심 스킬. 심상 프로세서 임플란트로 대체 가능.
- Lv0: 기본 재생. Lv2: 심상전이(타인에게 이미지 전달). Lv4: 인공계 수준의 심상 공간 일시 생성.

## 9-3. 능력 목록 (Power) — 각성으로 얻는 초능력

능력은 스킬 레벨이 아니라 "보유/미보유"로 관리됩니다.
획득 경로: 샤이닝 트랙 완료, 매팅 트랙 생존, 신내림, 선천.
각 능력은 클래스 레벨에 따라 위력이 성장합니다.

### 능력 표준 양식

- 능력명(한자, English)
- 원천: [샤이닝/매팅/신내림/선천/기타]
- 관련 스탯: [체크에 사용되는 프라임 스탯]
- 효과: [기본 효과 + 레벨별 성장]
- 비용: [셀/에고/패러독스 등]
- 제약: [사용 조건, 부작용]

### 프로토타입 능력 목록

- 심상재생(心象再生) — 원천: 선천/학습. 감(SEN). 심상 매체를 정신으로 재생. 재생사의 기본 능력.
- 심상전이(心象轉移) — 원천: 선천. 감(SEN)+격(PRE). 타인에게 이미지 전달. 소규모 인공계 일시 생성.
- 시간도보(時間徒步) — 원천: 각성. 감(SEN)+운(LCK). 앞걸음=미래, 뒷걸음=과거. 인과율 유지하며 시간이동. 비용: 패러독스 +2/사용.
- 재생성(再生成) — 원천: 매팅 생존. 체(BOD). 신체 손상 시 자동 재구성. 사실상 불사. 제약: 재구성 형태가 정확하지 않을 수 있음.
- 수납(受納) — 원천: 현화. 체(BOD). 체내에 물건을 수납. 정해진 공간과 육체가 연동. 군용 기술로도 존재. 직업: 집사(執事).
- 배관 인식(Pipe Perception) — 원천: 불명. 감(SEN). 물질계의 인과율 코드를 읽거나 조작. 극히 희귀. 획득 경로 불명.
- 몽상(夢想) — 원천: 선천/각성. 감(SEN)+운(LCK). 흑공에 의지를 직접 전달. 이론상 만능. 비용: 매팅 트랙 +1/사용. 기연 내성 체크 필요.
- 리듬필링(Rhythm Feeling) — 원천: 학습/선천. 감(SEN). 대지/신체의 맥동 감지. 흑공분출 예감, 치유 보조. RPS(바이오리듬 필러) 공무원의 필수 능력.
