## 빠른 참조표 (Quick Reference)

- 판정: 2d6 + 스탯 수정치 + 스킬 >= 난이도
- 난이도: 6(쉬움) / 8(보통) / 10(어려움) / 12(매우) / 14(불가능급)
- AC 기본: 10 + 신(REF) 수정치
- 공격: 1d20 + AB + 스탯 수정치 + 스킬 >= AC
- 치명: 치명 주사위(폭발) >= 목표치(6). 성공 시 피해 x 배수.
- 내성: d20 >= 내성치 (15 - 레벨/2)
- HP = "맞지 않는 운." HP 0 → 돌파. 이후 체(BOD) 직접 피해.
- 파워셀: 일만셀=1오얏 – 이십억셀=50만오얏
- 패러독스: 1-3 미미 / 4-6 불운 / 7-9 위험 / 10-12 재난 / 13+ 먹개비
- XP: 성공 3 / 실패(생존) 1 / 레벨업 = 현재Lvx3 누적
- QP 비용: 1st=2 / 2nd=3 / 3rd=5 / 4th=8

---

## 피해 흐름 요약

명중 → 피해 → 갑옷HP 흡수 → 초과분 캐릭터HP → HP 0 = 돌파
→ 이후 피해 체(BOD)에서 → BOD 0 = 물질계:진사 / 차원계:드랍아웃

---

## 전투 행동 요약

- 주행동 1회: 공격 / 태그 활성화 / 스킬 사용 / 해킹 / 인터렉션
- 이동 1회: 10m (주행동 포기 시 더블 무브)
- 부수행동: 짧은 말 / 엎드리기 / 기능 슬롯 인터렉션
- 즉시행동: 운(LCK) 토큰 사용 (타인 턴에도)

---

## 운(LCK) 토큰 용도

재굴림 / 피해 반감 / 치명 무효 / 패러독스 상쇄 / 사기 강제 / 배제 단축

---

## 인터렉션 슬롯 비용 요약

- 능력(Power): 에고 소모. 주행동. 체크 필요. 도구 불필요.
- 기술(Skill): 시간/도구. 주행동. 체크 필요. 푸시 가능.
- 기능(Function): 셀/시스템부하. 부수행동. 체크 대부분 불필요.
