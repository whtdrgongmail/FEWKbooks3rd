## 8-1. 신사 계열 (Prototype A: 에너지 운용형)
### 神士 SINSA CLASS GROUP

신사(神士)는 신기(神氣)를 사용하는 모든 직업군을 부르는 이름입니다.

신사들은 신단(神壇)에서 뿜어지는 신기를 몸에 저장하고, 이 에너지를
능력과 기술, 신기 임플란트, 신기 현물, 장비를 통해 직업적 역할을
수행합니다. 신사가 능력을 사용할 때 몸을 둘러싸는 오색빛의 기운은
신사의 상징과 같은 이미지이며, 이 오색빛을 '감로(甘露)'라고
부르는 이들도 있습니다.

공통 제약: 생체순도(BP) 50% 미만이면 신기용량 반감.
임플란트가 많을수록 신기 효율 저하.
공통 능력: 신기 조작(Sinki Control) — 7장 Prototype A 참조.

### 신사 계열의 인터렉션 슬롯 구조

- 신기조작 → 능력 슬롯 (선천/각성). 에고 소모.
- 신호연결 → 기술 슬롯 (학습). 체크 필요.
- 신기감지 → 기술 슬롯 또는 기능 슬롯 (임플란트 대체 가능).
- 신사의 딜레마: 능력 슬롯은 BP를 지키지만 에고를 치르고, 같은 효과를 기능 슬롯(임플란트)로 달면 BP가 줄지만 빠르고 안정적.

### 신전사(神戰士, Sinsa Warrior)

> "신기가 흐르는 칼날 앞에서는, 가장 두꺼운 장갑도 종이와 같다."

세계관: 신기 기반의 전투기술을 물리적 접촉으로 구사하는 전사.
신기가 흐르는 물리적 무기와 방패를 기본으로 사용하며, 실력자는
의도적 신폭으로 엄청난 위력의 전투를 수행합니다.

- 자격: 신사 Lv1+ / 체(BOD) 10+
- QP 비용: 하위직업(추가 2 QP)

고유 능력 — 신기 전투(Sinki Combat):
- Lv1: 근접무기에 신기 주입 — 피해 + 감(SEN) 수정치. 1,000셀/라운드.
- Lv3: 신기 방패 — 주행동으로 AC + 감(SEN) 수정치. 1,000셀/라운드.
- Lv5: 의도적 신폭 — 주행동. 10,000셀. 반경 5m/3d6+감 수정치.
- Lv7: 감로 검기(甘露劍氣) — 신기 원거리 참격. 15m/2d8+감. 3,000셀.
- Lv10: 전장 신단화 — 발밑 반경 10m를 일시적 신단으로 변환. 신기 무한 공급 (10라운드). 100,000셀 사전 충전 필요.

추천 스킬: Stab, 신호(Signal), Exert, Notice
시작 장비: 신기 장검 1자루, 돌칩 3개
대표 인물: 동쟁 4대 신수사들은 모두 신전사이기도 했습니다.

### 신기사(神氣士, Sinki Knight)

> "나이트라 불리는 이유는, 신단 엔진 위에서 세상을 달리기 때문이다."

세계관: 신기를 몸에 모으고 뿜어내는 배터리 역할 특화. 일반 신사의
20–100배 용량을 축적하며, 신단 엔진을 다룰 수 있어 선박, 비행선,
열차, 오토바이 등을 운용합니다. 서국의 명칭을 따 '나이트(Knight)'
라고도 불립니다.

- 자격: 신사 Lv1+ / 감(SEN) 14+
- QP 비용: 하위직업(추가 2 QP)

고유 능력 — 신기 충전(Sinki Charge):
- Lv1: 신기용량 x20. 타인/장비에 신기 충전 가능.
- Lv3: 신기용량 x50. 신폭 억제(10m 반경, 주행동).
- Lv5: 신단 엔진 운용 숙련. Drive 체크에 +2 (신단 탈것).
- Lv7: 신기용량 x100. 도시급 에너지 공급 가능(단시간).
- Lv10: 이동 신단 — 본인이 걸어다니는 신단이 됨.

추천 스킬: 신호(Signal), Drive, Fix, Exert
시작 장비: 대형 돌칩 5개(각 천만셀), 신단 엔진 접근 도구
대표 인물: 가야(배터리 소녀) — 아버지 가보에 의해 인간 신단이 됨. 도시급 시설 에너지 공급 가능.

### 신수사(神獸士, Sinsu Master)

> "아이들에게 가장 인기 있는 직업. 어른들에게 가장 두려운 직업."

세계관: 신기가 존재화된 신수(神獸)를 다루는 직업. 여러 인연으로
신수를 얻습니다. 현대에는 신수 자체가 희귀해진 탓에 유명한 신수사는
대부분 동쟁 시대 인물이고, 현역 러너로는 청호협려 진이 있습니다.
4대 신수: 첨작, 백무, 적룡, 청호.

- 자격: 신사 Lv3+ / 감(SEN) 14+ / 신수 조우 이벤트 필수(인게임)
- QP 비용: 하위직업(추가 3 QP) — 희귀

고유 능력 — 신수 유대(Sinsu Bond):
- Lv1: 신수 소환. HP = 신기용량/100. 유지 1,000셀/라운드.
- Lv3: 신수 강화 — 신기를 주입하여 신수 스탯 일시 상승.
- Lv5: 신수 합일(合一) — 1전투 1회, 신수와 합체. 모든 스탯 +1.
- Lv7: 대신수 — 신수 크기/출력 대폭 증가. 건물급.
- Lv10: 전설급 신수 — 지역급 영향력. 동쟁 괴수대전 재현 수준.

추천 스킬: 신호(Signal), Notice, Lead, Survive
시작 장비: 신수 부적(소환 매개), 돌칩 3개
대표 인물: 적룡화(赤龍花, 적룡의 신수사, 15세에 호올네트 독립), 청호협려 진(Jin, 벙커233에서 청호를 해방시킨 러너)

신수 데이터 예시 — 청호(靑虎):
HP 100 / AC 16/16 / AB +5 / 이동 20m
공격: 발톱 2d8+3, 신기 포효 1d10/15m 범위.
특수: 신기 방호 면역. 진의 신기용량에서 유지.

### 신법사(神法士, Sinsa Ritualist)

> "노란 종이에 붉은 글씨로, 세계의 규칙을 다시 쓴다."

세계관: 신기를 주술적 체계에 의해 사용. 오주사를 노란 종이에 마킹하여
즉시 사용하는 직업. 신법을 신단석에 새긴 것이 돌칩입니다.
팝티콘과 혼동되기도 하지만, 신법사는 신기 기반입니다.

- 자격: 신사 Lv1+ / 혜(INT) 12+
- QP 비용: 하위직업(추가 2 QP)

고유 능력 — 신법(Sinbeop):
- Lv1: 기본 신법 3개 습득. 오주사 부적 즉시 사용.
- Lv3: 신법 5개 추가. 전투 중 부적 즉발 사용(주행동).
- Lv5: 신법 설계 — 새로운 효과의 부적을 직접 설계.
- Lv7: 대신법 — 건물/지역 규모 효과. 고비용.
- Lv10: 신법 마스터리 — 오주사 없이 신법 발동 가능.

추천 스킬: 신호(Signal), 현화술(Manifest Art), Know, Notice
시작 장비: 오주사 10g, 황지(黃紙) 20장, 마킹 도구

### 신호사 계열 (神号士, Signal Specialists)

신사의 가장 많은 직업군. 신단석과 신기 회로를 다루는 기술자들.

- 신호연결사(神号連結士) — 가장 흔한 신사 직업.
  신기 회로를 연결, 복원하며 약간의 테크닉을 구사.
  교육기관이 많아서 꽤 많은 수가 있습니다.
  자격: 신사 Lv1+ / 혜(INT) 10+. QP +1.
  추가 능력: 신호(Signal) 스킬에 항상 +1 보너스.
  대표 인물: 진(Jin) — 신호연결사에서 시작하여 신수사가 됨.

- 신호설계사(神号設計士) — 신단 개조, 돌칩 제작.
  보통 (주)기업 소속. 자격: 신사 Lv3+ / 혜(INT) 14+. QP +2.
  추가 능력: 돌칩 제작 가능. Fix+신호 합산 체크.

- 신호탐지사(神号探知士) — 신기 탐지 특화.
  임플란트로도 대체 가능하여 독립 직업보다는 기본 소양.
  숙련자는 바이오리듬필러를 겸직. 자격: 신사 Lv1+. QP +1.
  추가 능력: Notice+신호 합산으로 50m 내 신기/흑공 자동 감지.

- 신호조각사(神号彫刻士) — 인게임 전용. 극히 희귀.
  '신내림'이라는 현상이 일어나야 가능한 직업.
  신단을 직접 만들 수 있는 유일한 존재.
  자격: 신사 Lv3+ / 신내림 이벤트 필수 / 혈통 기반.
  QP +3 (최고 비용).
  추가 능력: 신단 직접 제작. 신호 스킬 체크에 항상 자동 성공(Lv5+).
  팀에 신호조각사가 있다는 것만으로도 임무가 고평가.
  대표 인물: 세린(Serine, "달의 여신"), 마리아(크리스의 어머니, 소녀시절 '리브스트레이트' 제작 후 능력 상실).

- 신단주사(神壇朱士) — 오주사+신기 회로 설계.
  신호조각사와 신호설계사를 합친 느낌. 돌칩 제작 가능.
  자격: 신호연결사 분기 보유 / 신사 Lv3+. QP +2.

### 바이오리듬 필러 (RPS, Rhythm Perception Specialist)

> "흑공분출 경보와 괴물출현주의 경보는 이들에 의해 이뤄집니다."

세계관: 대지나 신체의 맥동을 감지하는 직업. 공식적으로는 흑공분출
감지 공무원만 RPS라고 부르지만, 신맥동 인터렉션으로 진맥, 치료,
신물 수리를 하는 기술자들도 많습니다.

- 유형: 라이선스 (대사국 연합 발행)
- 자격: 감(SEN) 12+ / 리듬필링 능력 보유 / 대사국 채용
- QP 비용: 2 QP

고유 능력 — 리듬 감지(Rhythm Detection):
- Lv1: 1km 반경 흑공/신기 농도 이상 감지. 1일 3회.
- Lv3: 5km 반경. 분출 예상 시간 추정(+-1시간).
- Lv5: 인체 맥동 감지 — Heal 체크에 +2. 에고 상태 진단.

추천 스킬: Notice, Heal, 신호(Signal), Survive

## 8-2. 흑사 계열 (Prototype B: 장비 확장형)
### 黑士 HEUKSA CLASS GROUP

흑사(黑士)는 흑공을 기반으로 하는 모든 자들을 통칭합니다.
일반적으로 서국의 기술 기반 임플란트와 장비를 사용하는 직업군.
민간에서 흑사는 "임플란트를 단 사람"을 의미하기도 합니다.

### 흑사 계열의 인터렉션 슬롯 구조

- 강화/줌/신기감지 → 기능 슬롯 (임플란트). 가장 빠르고 안정적.
- 흑전사가 전투에서 강한 이유: 돈으로 기능 슬롯을 채움. 체크 불필요, 부수행동으로 발동. 대가는 BP와 시스템 부하.

---

### 흑전사(黑戰士, Heuksa Warrior)

> "고위 흑전사는 탈것을 방불케 하는 임플란트를 탑재하고 있다."

- 자격: 흑사 Lv1+ / 체(BOD) 12+. QP +2.

고유 능력 — 중화기 운용(Heavy Ordinance):
- Lv1: 임플란트 무기 AB +1. 대구경 무기 반동 페널티 무시.
- Lv3: 임플란트 오버차지 — 1전투 1회, 임플란트 무기 피해 x2.
- Lv5: 전투 플랫폼 — 체(BOD) 14+이면 중화기 2개 동시 운용.

대표 인물: (주)신성 본사 경비 흑전사들.

### 흑번제사(黑燔祭士, Heuksa Priest)

> "시작은 화학자, 지금은 오컬티스트. 헤레틱을 다루는 자들."

세계관: 흑공 정제를 맡던 화학자 중 오컬트적으로 변모한 일파.
종교적 색채. '헤레틱(Heretic)'이라는 흑공 기반 원시 에너지 사용.
난초대사국과 연관이 깊으며, 비밀조직과 기업조직망 보유.
신기와 반응하여 헤레틱이 더 원활히 추출되므로 신사 소양을
갖추고 있는 경우가 많습니다.

- 자격: 흑사 Lv3+ / 감(SEN) 10+ / 난초국 계열 콘택트. QP +3.

고유 능력 — 헤레틱 술식(Heretic Ritual):
- Lv1: 흑공용액 기반 주술. 소규모 현화 유도 (통제적).
- Lv3: 헤레틱 추출 — 신기 접촉 시 헤레틱 에너지 생성.
- Lv5: 소환진 설계 — 대규모 흑공 구조물/존재 소환.
- Lv7: 흑공거인 소환 (동쟁급 병기. 극고비용).

제약: 사용 시 매팅 트랙 +1 위험. 대사국 감시 대상.
대표 인물: 흑해적 관련 흑번제사들, 난초국 비밀조직원.

### 흑루사(黑樓士, Heuksa Builder)

> "하루만에 성을 만들어 올리고, 터널을 뚫고, 도로를 깐다."

세계관: 심상기록으로 설계를 기록하고, 흑공말통의 흑공을 이용하여
토목/건설을 하는 기술자. 흑공 메카트로닉스 분야에서 빛나는 기술.

- 자격: 흑사 Lv3+ / 혜(INT) 14+. QP +2.

고유 능력 — 흑공 건설(Tenebre Construction):
- Lv1: 소규모 구조물(벽, 바리케이드) 즉석 건설. 이억셀 소모.
- Lv3: 건물 1동 1일 건설. 설계 심상 필요.
- Lv5: 터널/도로 건설. 대규모 토목. 이십억셀 소모.

추천 스킬: Fix, 심상(Image), Know, Exert
대표 인물: 모스(Moss) — 미모국 흑루사. 루쳄트레일로드역 건설.

## 8-3. 전사 계열 (Prototype C: 전투 특화형)
### 戰士 JEONSA CLASS GROUP

### 유용한 추가 역할

- 수호사 — 대사국 공식 임무 시 필수 참여 규정.
- 집사 — 장비 공급 혁신. 있으면 작전 자유도 급상승.
- 신기사 — 신사 팀의 배터리. 에너지 지속력 대폭 향상.
- 배틀마커 — 대형 임무 전 파티 전원 강화.

### 조합 시너지 예시

- 신전사 + 신기사: 신기 무한 콤보. 신폭 걱정 없이 연속 전투.
- 교수 + 러너: 미지 차원 탐사 최적 조합. 분석 + 이동.
- 흑전사 + 집사: 화력과 탄약 보급의 완벽한 조합.
- 신수사 + 야전교수: 신수 전투 + 즉흥 분석 지원.
- 도아너 + 수호사: 뚝(방어진) + 수호 선언 = 철벽 방어.
