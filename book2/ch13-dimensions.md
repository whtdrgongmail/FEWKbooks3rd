# 13장. 차원 이동과 러닝
# Chapter 13. Dimensional Travel & Running

## 13-1. 러닝의 기본 규칙 Running Basics

차원을 건너 이동하는 것을 '러닝(Running)'이라 합니다.
러닝을 업으로 삼는 자들이 '러너(Runner)'입니다.

### 러닝의 3요소

1. 바이트 비용 — 이동 대상의 바이트에 비례하는 에너지 소모.
2. 패러독스 누적 — 차원 간 이동 시 불가피하게 쌓이는 인과율 오류.
3. 적응 — 목표 차원의 자연법칙에 맞추어 존재를 재구성하는 과정.

## 13-2. 이동 방식 Methods of Transit

### 코핀(Coffin) — 표준 방식

다이버 코핀은 러닝의 표준 장비입니다.
육체(그릇)를 코핀에 남기고 주체(21자 바이트)만 이동.
- 바이트 비용: 약 600자 (주체 21자 x 약 30배)
- 패러독스: +1/러닝 (기본)
- 적응: 자동 (인공계). 러닝(Running) 체크 필요 (오지계).
- 장점: 저렴, 안전, 드랍아웃 시 코핀에서 복귀.
- 단점: 육체가 물질계에 무방비로 남음. 피지컬 러너의 표적.

### 게이트(Gate) — 물리적 통로

두 차원 사이에 고정된 물리적 통로.
- 바이트 비용: 없음 (게이트가 부담).
- 패러독스: +0–+1 (안정적 게이트) / +2–+3 (불안정).
- 적응: 불필요 (게이트가 자동 변환).
- 장점: 비용 없이 육체째 이동 가능.
- 단점: 고정 위치. 양방향 개방이라 침입 위험.
- 예시: 빅포레스트 거리의 카페 뒷문, 새우거리 건물.

### 히어로폰(Herophone) — 약물 이동

자가 투여 약물로 강제 차원 전이를 유도.
- 바이트 비용: 약물이 대체 (히어로폰 1회분 = 오얏 500)
- 패러독스: +3/사용. 고위험.
- 적응: 불안정. 러닝(Running) 체크 난이도 +2.
- 장점: 코핀/게이트 없이 어디서든 러닝 가능.
- 단점: 부작용 (시스템 부하 +2, 에고 -1 일시). 중독 위험.
- "크리스란 미친 놈은 히어로폰을 퍼마시고 있다."

### 리추얼(Ritual) — 의식 이동

신사/흑번제사가 의식을 통해 차원문을 여는 방식.
- 바이트 비용: 의식 재료 (돌칩, 오주사, 흑공말통 등)
- 패러독스: +1–+2.
- 적응: 의식 설계에 따라 다름.
- 장점: 다인원 동시 이동 가능. 육체째 이동 가능.
- 단점: 준비 시간 (수 시간–수일). 재료비 고가. 전문가 필요.

### 광전(光銭) — 완전 이동

광제 시절 금화 1개 = 한 사람의 육체 완전 이동.
- 바이트 비용: 광전이 내장 (60–90만자).
- 패러독스: +1.
- 적응: 자동.
- 장점: 바이트 계산 불필요. 완전한 육체 이동.
- 단점: 극고가. 1회용. 비상 탈출 수단.

### 인신공양(人身供養) — 금지된 차원이동법

히어로폰 이전, 고대에 사용된 차원이동법.
다른 존재의 바이트를 대가로 지불하여 차원이동하는 방식.
대상의 혼백을 분해하여 이동 비용으로 전환합니다.

현재 모든 대사국에서 불법이며, 발각 시 사형추방(영구 드랍아웃).
하지만 나이트 러너와 일부 흑번제사 사이에서 비밀리에 행해집니다.

RPG에서: GM이 "금기 기술"로 제시 가능. PC가 사용하면:
- 대상 1인 즉사 (물질계) 또는 강제 드랍아웃 (차원계).
- 시전자: 패러독스 +5. 매팅 트랙 +2.
- 차원이동 성공 보장 (패러독스 무시).
- 대가: 히트(Heat) +5. 모든 수호사 조직이 추적 개시.

## 13-3. 패러독스 규칙 Paradox Rules

### 패러독스 누적

러닝, 인터렉션 사용, 시간여행, 인과율 위반 시 누적.
PC의 패러독스 수치는 캐릭터 시트에 기록합니다.

누적 원인:
- 코핀 러닝: +1/회
- 히어로폰: +3/회
- 시간도보(시간여행): +2/회
- 인터렉션(능력 사용): +0–+1 (능력에 따라)
- 현화술 실패: +1–+2
- 드랍아웃 후 같은 차원 재진입 시도: +5

### 패러독스 해소(Paradox Resolution)

패러독스는 대자연이 강제로 보정합니다. 해소 단계:

| 수치 | 효과 |
|------|------|
| 1–3 | 미미 — 두통, 손저림, 악몽. 롤플레이 효과만. |
| 4–6 | 불운 — 기연 내성 체크 1회/일. 실패 시 사소한 사고. 머피의 법칙. 중요 체크에 -1 페널티. |
| 7–9 | 위험 — 질병, 급살 위험. 기연 내성 체크 1회/시간. 실패 시 1d6 피해(회피 불가). 에고 -1/일. |
| 10–12 | 재난 — 개인 급살. 기연 내성 실패 시 즉사 위험. 주변 지역에 소규모 재난(지진, 화재 등) 유발 가능. |
| 13+ | 먹개비 — 먹개비 조우 판정. 2d6 >= 패러독스 수치이면 먹개비 출현. 먹개비에게 잡히면 존재 소멸. |

### 패러독스 감소 방법

- 자연 감소: 1박마다 -1 (안정적 환경에서).
- 운(LCK) 소모: 1 LUCK = 1 패러독스 즉시 소거.
- 러너 클래스 능력: Lv3에서 러닝당 -1 감소. Lv7에서 대폭 감소.
- 오주사 정화: 오주사 1g 소모 = -2 패러독스. Heal 체크 필요.
- 시간 경과: 차원이동 없이 1주일 = -3 추가 감소.

## 13-4. 차원 적응과 월드 템플릿 Adaptation & World Templates

### 차원 적응

새로운 차원계에 진입하면 존재가 해당 계의 기본 물질로 재구성됩니다.
시각/청각/촉각은 물질계와 동일하게 느껴집니다.

적응 체크: 2d6 + 감(SEN) + 러닝(Running) >= 차원 난이도.
- 인공계: 자동 성공 (난이도 없음).
- 안정 차원(광제사직계 등): 난이도 6.
- 불안정 차원(개척 차원): 난이도 10.
- 오지 차원: 난이도 12–14.
- 실패 시: 1d6 라운드 적응 불량 (모든 체크 -2).

### 월드 템플릿(World Template)

적응 완료한 차원계에 대해 '월드 템플릿'이 생성됩니다.
이후 해당 차원 재진입 시 적응 자동 성공.

월드 템플릿 내용:
- 해당 차원의 자연법칙 특이사항
- 스탯 변환 규칙 (있을 경우)
- 사용 가능/불가능한 장비 목록
- 패러독스 발생 조건

예: 플랫마운틴(오크계) 월드 템플릿 —
"메카트로닉스 장비 사용 불가. 현물만 통용. DDOGG 로컬 질서 활성.
어린이 눈에 띄면 DDOGG가 로컬 패러독스 발생시킴."

## 13-5. 시간여행 특수 규칙 Time Travel

FEWK에서 시간여행은 가능하지만 극히 위험합니다.

- 루쳄트레일로드역: 상행=내일, 하행=어제. 가장 안전한 방법. 패러독스: +1/이동. 개찰구에서 먹개비 즉시 출현 가능.
- 시간도보(時間徒步): 능력자 전용. 앞걸음=미래, 뒷걸음=과거. 패러독스: +2/사용. 인과율 흐름 유지가 핵심.
- 시간의 한계:
  - 검은 수평선(과거 한계) — 이보다 과거로 갈 수 없음.
  - 검은 태풍(미래 한계) — 이보다 미래로 갈 수 없음.
- 인간의 주사위: 시간 분기점에서 어떤 경로를 타는지는 운(LCK)으로 결정. "인과율의 분기에서 길을 잃지 않는 것이 핵심."

## 13-6. 드랍아웃 상세 규칙 Drop-out Details

차원계에서 HP 0 → 사망 판정 → 사망:

1. 즉시 물질계의 코핀/그릇으로 귀환.
2. HP 1. 에고 -1 (1일 회복). 시스템 부하 = 최대치.
3. 해당 차원계에 '배제(Exclusion)' 상태 부여.
   - 배제 상태에서 재진입 시도 시: 패러독스 +5.
   - 기연 내성 체크 실패 시 먹개비 즉시 조우.
4. 배제 해제: 1개월 경과 또는 오주사 정화 의식.

>  어떤 보수로도 상쇄되지 않는다." — 러너들의 격언.
