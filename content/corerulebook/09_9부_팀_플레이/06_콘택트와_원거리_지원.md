# 09-6. 콘택트·원거리 지원

---

모든 PC가 매 순간 함께 있는 것은 아닙니다. 차원 다이브 중 물질계 동료와 통신할 때, 브로커가 멀리서 의뢰를 중개할 때, 해커가 세이프하우스에서 원격 지원할 때. **Lead Check**는 원거리 동료·콘택트의 지원을 판정으로 처리합니다.

## 6-A. 콘택트 개념

**콘택트 (Contact)**: PC의 직업적·사회적 네트워크 속 인물. PC가 연락해 정보·지원을 받을 수 있는 NPC.

- 시작 시 3~5명 (태그·배경에 따라)
- 세션 진행으로 확장 가능
- 각 콘택트: 이름·분야·신뢰도·현재 관계

## 6-B. 콘택트 카드 서식

```
이름: ___
분야: ___ (브로커·픽서·정보상·의료인·기술자·신사·흑사 등)
신뢰도: +1 ~ +5
현재 관계: ___ (호의·중립·경계)
주 스탯: ___
주 스킬: ___
연락 수단: ___ (직접·뉴로덱·통신기 등)
```

## 6-C. Lead Check — 원거리 지원 판정

### 조건

- 통신 가능 (뉴로덱·무선·기타)
- 정보 전달 시간 허용 (실시간 or 지연)
- 관련 정보 전달 가능

### 판정

```
Lead Check 주 판정:
주 판정자(현장 PC): 기본 Grade Die 판정
원거리 지원자: 관련 스킬 +N 제공

효과:
+1 Die (주 판정자 Grade에 추가)
또는 Position 완화
또는 정보 제공 (별도 판정 생략)
```

### 비용

- 콘택트 신뢰도 일시 소모 (관계 −1)
- 의뢰 대가 (오얏·호의·빚)
- 시간 지연 (한 턴 소모)

### 예시

> **상황**: 민이 ㈜신성 내부 해킹 중. 원격 해커 수덕(집)이 지원.
>
> 민: "수덕, 백도어 좀 열어줘."
> 수덕 (원거리): INT 7 + 해킹 +3 vs TN 15. 굴림 성공.
> **효과**: 민의 다음 해킹 판정에 +1 Die.

## 6-D. 주요 콘택트 유형

### 브로커 (Broker)

- 의뢰 중개
- 신뢰도 높으면 고급 의뢰 접근
- 비용: 수수료 10~20%

**FEWK 예**: 페이 — 서민구 브로커.

### 픽서 (Fixer)

- 문제 해결 전문
- 법·사회적 어려움 해결
- 비용: 오얏 (10~50 오얏)

### 정보상 (Information Broker)

- 흑백 시장 정보 판매
- 특수 지식 접근
- 비용: 오얏·거래·호의

### 의료인 (Medic)

- 위급 치료
- 불법 개조·야메 수술
- 비용: 오얏·비밀 유지 (BP% 대가 시)

### 기술자 (Tech)

- 장비 수리·개조
- 특수 부품 확보
- 비용: 오얏·재료

### 신사 (Priest)

- 신기 지원
- 정화 의식
- 비용: 서약·의무

### 흑사 (Dark Agent)

- 흑공 조작
- 어두운 지식
- 비용: BP% 감소·사회적 대가

## 6-E. 콘택트 확장

### 세션 중 획득

- 새 NPC와 의미 있는 관계 → 콘택트 추가 가능
- GM 승인 + 서사적 근거

### 포인트 소비

- 새 콘택트 획득: 5p (8부 §01 포인트)
- 콘택트 신뢰도 +1: 3p

## 6-F. 커뮤플레이·PBeM 활용

비동기 플레이(13부 §04·§05)에서 Lead Check는 특히 유용:

- **PBeM**: GM이 원거리 동료 판정을 이메일 내 동시 처리
- **커뮤플레이**: 다른 파티의 PC를 Lead Check 대상으로 활용
- **게시판 플레이**: 원거리 지원 판정을 별도 스레드로 처리

비동기 환경에서는 "모두 같은 시간에 같은 장소"가 어려우므로 Lead Check로 유연성 확보.

## 6-G. 예시 — 복합 콘택트 활용

> **상황**: 민이 기업 본사 침투 중. 여러 콘택트 동원.
>
> **준비 단계**:
> - 페이(브로커): 내부 경비 일정 정보 (대가: 수수료 10%)
> - 기술자 제리: 뉴로덱 업그레이드 (대가: 오얏 15)
> - 정보상 카이: 임원 스케줄 (대가: 정보 교환)
>
> **작전 중**:
> - 민 해킹 판정 실패 → 수덕 Lead Check → 성공 → 재시도 허용
> - 민 부상 → 의료인 원격 조언 → 응급 처치 판정 +2
>
> **결과**: 혼자라면 불가능했을 작전. 콘택트 네트워크의 힘.
