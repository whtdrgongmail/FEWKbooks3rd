# 07-4. 탄약·d6 Pool 소모

---

FEWK에서 **탄약은 d6 Pool로 추상화**됩니다. 개별 탄환 개수를 세지 않고, 한 번의 사격·폭발에 필요한 d6 Pool 소모로 표현합니다.

## 4-A. 탄약 계열

무기 6계열별 탄약 (§01 무기 카탈로그):

| 계열 | 탄약 | 특징 |
|---|---|---|
| M건 | 물리 탄 | 표준·저렴 |
| V건 | 진동 탄 | 방어 관통 |
| S건 | 전자기 탄 | 임플란트 추가 피해 |
| 광선포 | 에너지 셀 | 재충전 가능 |
| 전격총 | 전격 셀 | 충전식 |
| 서국 화약 | 구식 화약 | 차원 체류 중 작동 |

## 4-B. 사격 유형별 d6 Pool 소모

| 유형 | 소모량 | 효과 |
|---|---|---|
| **정확 사격** | 1d6 | 단발, 공격 판정 +0 |
| **조준 사격** | 2d6 | 1라운드 조준, 판정 +1 |
| **연사** | 3~5d6 | 3발 이상 빠르게 |
| **자동 사격 (범위)** | 5~15d6 | 범위 제압 |
| **기관포 소사** | 10~15d6 | Scale B 대규모 |
| **저격** | 2d6 | 거리 극대화, 판정 +2 |

d6 Pool Hit 수 = 피해 배수 or 범위 (§3-3 d6 Pool 참조).

## 4-C. 특수 탄약

일반 탄 외에 특수 탄약:

| 탄약 | 비용 | 효과 |
|---|---|---|
| 관통 탄 (AP) | 2배 | 방어 수정치 0 |
| 확장 탄 (Expanding) | 1.5배 | 피해 +1d4 |
| 소이탄 | 2배 | 출혈 상태 부과 |
| EMP 탄 | 3배 | 임플란트·전자 장치 일시 불능 |
| 신기 교란 탄 | 5배 | 샤이닝 시전 방해 |

특수 탄약은 **상시 소지량 제한** (PC당 3~5발 수준).

## 4-D. 재장전

### 기본 재장전

- **부수 행동 1회**: Pool에서 1d6 → 5d6 재충전
- **주요 행동 1회**: Pool 절반 재충전

### 관련 스탯 장전

```
장전 판정: 관련 스탯 Grade 굴림 ÷ 5 올림 = 충전 d6 개수
```

- 일반 장전: 별도 판정 없이 부수 행동
- 긴급 장전: 판정 + 부수 행동 (Full 이상 시 +2d6)

### 장전 불가 상황

- 탄약 완전 고갈 (새 탄창 필요)
- 집중 공격 중 (부수 행동 불가)
- 근접 교전 (1m 이내)

## 4-E. 탄약 부족

### Pool 0 상태

- 사격 불가
- 근접 공격만 가능
- 탄약 구매·재고 보충 필요 (Downtime)

### Pool 경고 (30% 이하)

- GM이 "탄약 부족 경고" 선언
- 플레이어는 장전 또는 이탈 결정

## 4-F. 탄약 경제

### 구매 비용

| 탄약 | 가격 |
|---|---|
| 표준 탄 (풀 탄창) | 1 오얏 |
| 특수 탄 (5발) | 3~15 오얏 |
| 에너지 셀 | 10 오얏 |
| 신기 탄 (돌칩 연동) | 5 오얏 + 돌칩 |
| 크로코다일 전용 탄 | 5 오얏 (5발) |

### 월 표준 러너 탄약 예산

약 20~30 오얏/월. A급 이상은 특수 탄약 예산 추가.

## 4-G. Pool 관리 전략

### 플레이어

- 세션 시작 전 Pool 상태 확인 (BOD + PRE 한도의 50%)
- 전투 전 장전 필수 (자유 장전 기회)
- 특수 탄약 비축 (결정적 순간)

### GM

- Pool 소모 추적 (전투 중)
- 재장전 타이밍 제공 (엄폐 이용 기회)
- 탄약 고갈 서사 활용 (긴장 고조)
