# 2부. 캐릭터 (Characters)

**예상 분량**: 55~66p (B5)

---

2부는 "한 명의 PC를 만든다"에 해당합니다. 6 Prime Stats로 뼈대를 세우고, 스킬·태그·인터렉션으로 몸을 붙인 다음, 프리셋과 라이프패스로 서사를 얹어 완성합니다. 플레이어가 가장 많이 펴드는 부입니다.
