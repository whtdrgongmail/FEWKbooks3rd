# 02-3. 스킬 — 0~7 선형 보너스

---

스탯이 캐릭터의 **재능**이라면, 스킬은 **훈련**입니다. 스탯은 d4~d20의 주사위가 되지만, 스킬은 그 위에 **플랫 보너스 +0~+7**로 얹힙니다. 같은 스탯을 가진 두 PC도 스킬 구성에 따라 완전히 다른 전문가가 됩니다.

## 3-A. 스킬의 역할

```
판정값 = (스탯 Grade) + 스킬 + 수정치
```

스킬은 Grade Die에 **고정값**으로 더해집니다. 굴림과 무관한 확정 보너스입니다.

**예시**:
> INT 7 (5d6) 해킹 스킬 +5 해커.
> 굴림: 3+5+**6**+2+4 = 20 → 6 폭발 → 재굴림 5 → 누적 25
> 스킬 +5 → **판정값 30**

스킬이 없으면 같은 스탯도 평범한 일반인 수준에 머뭅니다. 전문가와 아마추어를 가르는 것이 스킬입니다.

## 3-B. 스킬 비용 — (X+1)p 선형

```
스킬 X → X+1: (X+1) p

0 → 1: 1p
1 → 2: 2p
2 → 3: 3p
3 → 4: 4p
4 → 5: 5p
5 → 6: 6p
6 → 7: 7p
0 → 7 누적: 28p
```

스킬 비용은 **완벽한 선형**입니다. 한 단계 오를 때마다 정확히 +Np 비용. 100p 예산 중 28p를 쏟으면 한 스킬을 장인 수준(+7)까지 올릴 수 있지만, 그러면 다른 모든 축을 포기해야 합니다.

실전 권장: **주력 스킬 2~3개를 +4~+6으로**, **보조 스킬 4~5개를 +1~+3으로** 분산.

## 3-C. 스킬 상한 — 스탯이 재능의 천장

```
상한 = min(스탯 ÷ 3 올림 + 3, 7)
```

| 스탯 | 스킬 상한 |
|---|---|
| 1~3 | 4 |
| 4~6 | 5 |
| 7~9 | 6 |
| 10+ | 7 |

**의미**: 낮은 재능으로 극한 훈련 불가. 해킹 스킬 +7을 찍으려면 INT 10 이상 필요. BOD 3짜리 PC가 격투 스킬 +7을 찍을 수는 없습니다 — 몸이 따라가지 못합니다.

이 규칙은 "포인트만 쏟으면 무엇이든 될 수 있다"의 함정을 막습니다. 극한 전문가가 되려면 **재능 + 훈련** 둘 다 필요합니다.

## 3-D. 스킬 카탈로그 (Core 기본 + Pack 확장)

### 물리계

| 스킬 | 주 스탯 | 용도 |
|---|---|---|
| 격투 | BOD·REF | 근접 전투 |
| 운동 | BOD·REF | 뛰기·오르기·수영 |
| 사격 | REF | 원거리 무기 |
| 은밀 | REF | 숨기·기습·잠입 |
| 저항 | BOD | 독·질병·극한 환경 버티기 |

### 지각계

| 스킬 | 주 스탯 | 용도 |
|---|---|---|
| 감지 | SEN | 함정·숨은 자·미세 변화 발견 |
| 조사 | INT·SEN | 현장 단서 수집·분석 |
| 추적 | SEN | 발자국·흔적 따라가기 |

### 지식계

| 스킬 | 주 스탯 | 용도 |
|---|---|---|
| 해킹 | INT | 컴퓨터·네트워크 침투 |
| 공학 | INT | 장비·기계 수리·개조 |
| 의학 | INT | 치료·독·해부학 |
| 지식 | INT | 일반 지식·연구·기록 |

### 사교계

| 스킬 | 주 스탯 | 용도 |
|---|---|---|
| 사교 | PRE | 환대·신뢰 쌓기 |
| 설득 | PRE | 논리적 설득 |
| 위협 | PRE·BOD | 공포를 통한 강제 |
| 기만 | PRE·INT | 거짓말·속임수 |

### 운·감각계

| 스킬 | 주 스탯 | 용도 |
|---|---|---|
| 도박 | LCK·INT | 게임·확률 판단 |
| 직감 | SEN·LCK | 순수 직관 |

### FEWK Cyberpunk Pack 확장

- **차원 조작**: INT·PRE (코핀 다이브·뇌네 조작)
- **신기 조작**: SEN·PRE (신단 활성·신기 시전)
- **흑공 읽기**: SEN (흑공 농도·흐름 감지)
- **러너 생리학**: INT·SEN (매팅·드랍아웃 진단)

Pack 고유 스킬은 FEWK 세계에서만 의미가 있지만, Core 스킬과 동일한 비용 규칙을 따릅니다.

## 3-E. 스킬 조합 예시 — 어떤 전문가가 되나

### 표준 러너 (전투·잠입 양면)
- 사격 +4 · 격투 +3 · 은밀 +4 · 감지 +3 · 사교 +2 · 지식(거리 문화) +1
- 총: 17p

### 전담 해커
- 해킹 +6 · 조사 +4 · 지식 +4 · 공학 +3 · 사교 +1 · 감지 +2
- 총: 20p

### 사회 공작원
- 사교 +5 · 설득 +5 · 기만 +4 · 위협 +3 · 지식 +3 · 감지 +2
- 총: 22p

### 전담 신사
- 신기 조작 +6 · 감지 +5 · 흑공 읽기 +4 · 설득 +3 · 지식 +2 · 사교 +2
- 총: 22p

각 빌드가 예산 약 20p로 **특정 역할의 전문가**를 만듭니다. 나머지 포인트는 태그·인터렉션·장비에.

## 3-F. 스킬 vs 인터렉션

스킬과 비슷해 보이지만 다른 개념이 **인터렉션** (능력·기술·기능)입니다. 다음 장에서 다룹니다.

| 구분 | 스킬 | 인터렉션 |
|---|---|---|
| 효과 | 판정 플랫 보너스 | 특수 효과·장르 액션 |
| 비용 | 선형 (X+1)p | 누진 (Lv별) |
| 습득 | 학습 | 재능·훈련·임플란트 조합 |
| 개수 | 제한 없음 | 3 슬롯 제한 (능력·기술·기능) |

단순히 "잘하게 되는" 건 스킬, "할 수 있게 되는" 건 인터렉션입니다. 해킹 스킬은 해킹을 더 잘하게 하지만, "시간 지연 해킹" 능력은 해커가 할 수 있는 행동 자체를 확장합니다.
