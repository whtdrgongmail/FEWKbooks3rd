# 02-4. 태그 — 육체·성격·배경 3종 + 러너 태그 30종

---

스탯과 스킬이 "PC가 얼마나 할 수 있는가"를 정한다면, **태그**는 "PC가 누구인가"를 정합니다. 체격, 성격, 출신, 별명 — 태그는 서사적 색깔을 기계 효과로 변환합니다.

## 4-A. 3 범주 태그

| 범주 | 비용 | 역할 |
|---|---|---|
| **육체 태그** | 3p | 신체·물리 특성 |
| **성격 태그** | 2p | 내면·심리 특성 |
| **배경 태그** | 1~5p (규모별) | 출신·소속·역사 |

각 범주에서 PC당 **1~3개** 정도 권장. 너무 많으면 서사가 흩어지고, 너무 적으면 캐릭터가 평면적이 됩니다.

## 4-B. 육체 태그 (Core 기본 7종, 3p)

| 태그 | 효과 |
|---|---|
| **강인함** | HP +5 |
| **민첩함** | Initiative +1 Die (임시) |
| **감각 예리** | 감지 판정 +1 |
| **언어 능력** | 사교 판정 +1 |
| **회복력** | 자연 휴식 회복 +1d6 HP |
| **고통 저항** | 빈사 저항 +1 |
| **장수** | 노화 지연 (서사 효과) |

Pack이 효과를 커스터마이징할 수 있습니다. 예: FEWK Cyberpunk Pack은 "강인함"에 "흑공 농도 저항 +1" 추가 가능.

## 4-C. 성격 태그 (Core 기본 예시, 2p)

성격 태그는 **자유 명명** 가능합니다. 아래는 권장 목록:

- **냉정함** — 공포·공황 저항 +1
- **호기심** — 조사 판정 +1
- **집착** — 특정 목표에 대한 집중 판정 +1 (다른 목표 −1)
- **친화력** — 첫 만남 사교 판정 +1
- **의심** — 기만 저항 +1
- **신중함** — Position 협상 DSR 비용 −1 (세션당 1회)
- **충동** — 즉시 반응 판정 +1 (장기 계획 −1)

성격 태그의 **단점**을 같이 쓰는 것이 FEWK의 권장 스타일입니다. 장점과 단점이 공존하는 PC가 서사적으로 풍부합니다.

## 4-D. 배경 태그 (1~5p, 규모별)

| 규모 | 비용 | 예시 |
|---|---|---|
| 소형 | 1p | 특정 동네 주민·작은 가족 |
| 표준 | 3p | 특정 기업 출신·갱단 말단·종교 집단 |
| 대형 | 5p | 대사국 귀족·23기업 임원 후보·신디케이트 간부 |

배경 태그는 **관계망**을 제공합니다. 기업 출신이면 그 기업 내부 정보에 접근 가능, 종교 집단 소속이면 해당 종교 NPC가 호의적. 규모가 클수록 비용과 함께 **연결된 NPC 수**가 늘어납니다.

## 4-E. FEWK 러너 태그 30종 (Pack 03 SSOT)

FEWK Cyberpunk Pack은 러너 캐릭터 고유의 **직업적 태그** 30종을 제공합니다. 세 범주와 별개의 **러너 태그 슬롯**이 있으며, 기본 1~2개 획득 가능합니다.

### 유형 태그 (10종, Pack 03 §3-1 SSOT)

| 태그 | 효과 | 약점 |
|---|---|---|
| **피지컬** | BOD +1 Die·비무장 +1단계·HP +5·중량 완화 | 정밀 작업 −1 |
| **플라이어** | LCK 판정 +1 Die·토큰 +1/세션·생사 체인 | 장기 계획 −1 Die |
| **배틀마커** | Critical +5·Initiative +1 Die·1R Effect +1 | 비전투 사회 −1 Die |
| **수호사** | 매팅 저항 +5·대신받기·재굴림 체인 | BP 30+ 임플란트 시 −50% |
| **에고부스터** | 에고 +2·PRE 판정 +1 Die·팀 Pool +1d6·회복 +1 | 단독 행동 +1 Die 상실 |
| **신전사** | Effect +1·트랙 5↓ +1 Die·재굴림 | BP% 자연 −1%/10세션 |
| **흑사** | 흑공 조작 +1·BP% 저항 +1 | 신기 판정 −1·신사와 대립 |
| **해커 전담** | 해킹 판정 +1 Die·뉴로덱 −1 Lv 비용 | 물리 전투 −1 |
| **드라이버** | 차량 조작 +1 Die·Initiative +1·리브 스트레이트 무료 | 차량 밖 지구력 −1 |
| **브로커** | 사교 +1 Die·정보 수집 +1·콘택트 +2 | 직접 전투 −1 |

세부 효과·약점·시너지는 Pack 03 §3-1~§3-10에서 정의합니다.

### 분류 태그 (5종)

| 태그 | 효과 |
|---|---|
| **D급 러너** | 시작 보수 낮음·접근 가능 의뢰 많음 |
| **C급 러너** | 표준 보수·중간 의뢰 |
| **B급 러너** | 보수 +50%·엘리트 의뢰 접근 |
| **A급 러너** | 보수 +100%·희귀 의뢰·명성 +1 |
| **S급 (전설)** | Pack 특수 — 이야기 주인공급 |

### 출신·소속 태그 (15종)

6스타일지구·23기업·갱단·종교재단·대사국별 태그. 예:
- **궁정구 출신** · **자본구 출신** · **노마드 출신** · **서민구 출신**
- **㈜신성 소속** · **㈜액살 소속** · **㈜뻑더레스 소속**
- **피스메이커 갱단** · **식스센스 요원** · **오펀나지 소속**
- **신성재단 신사 견습** · **와치벨 기계교 속** · **해골 성당 속**
- **미모국 시민** · **이화국 시민**

각 태그는 관련 NPC 접근권·사회 수정치·특수 의뢰 경로를 부여합니다. 세부는 Pack 03 §3-2~§3-5.

## 4-F. 태그 획득·상실

### 시작 획득

- 캐릭터 생성 시 포인트로 구매
- 프리셋(§06)이 기본 태그 1~2개 포함

### 서사 중 획득

- 장기 캠페인 서사 이벤트로 새 태그 획득
- 8부 성장 포인트 소비 (태그 비용과 동일)

### 상실

- 서사 사건 (기업에서 해고 → 기업 소속 태그 상실)
- BP% 하락 극단 (흑사 태그 획득 시 신사 태그 상실 등)
- 드랍아웃 후유증 (6부 §07)

태그는 **변화합니다**. 시작 시의 태그가 캠페인 끝까지 그대로인 경우는 드뭅니다.

## 4-G. 태그 시너지

태그 조합이 특별한 효과를 내는 경우가 있습니다.

- **피지컬 + 배틀마커**: 1라운드 Initiative +2 Die, HP +10
- **에고부스터 + 신전사**: 팀 Pool 공유 시 Effect +1 보정
- **수호사 + 궁정구 출신**: 왕실 경호 의뢰 전용 접근권
- **브로커 + 자본구 출신**: 23기업 네트워크 특수 접근

시너지는 Pack 03 §3-8에 일부 정리되어 있으며, GM의 재량으로 세션 중 새 시너지가 인정될 수 있습니다.

## 4-H. 예시 — 태그 구성 3종

### 표준 러너 (11p)
- **강인함** (육체, 3p)
- **냉정함** (성격, 2p)
- **서민구 출신** (배경, 1p)
- **피지컬** + **C급 러너** (Pack 러너 태그, 비용 5p)

### 해커 전담 (13p)
- **감각 예리** (육체, 3p)
- **호기심** (성격, 2p)
- **자본구 출신** (배경, 3p)
- **해커 전담** + **B급 러너** (Pack, 5p)

### 신사 견습 (14p)
- **언어 능력** (육체, 3p)
- **신중함** (성격, 2p)
- **신성재단 신사 견습** (배경, 4p)
- **신전사** + **C급 러너** (Pack, 5p)

태그 20p 안팎이 일반적 분배입니다.
