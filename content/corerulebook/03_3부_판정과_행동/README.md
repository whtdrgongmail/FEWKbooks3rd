# 3부. 판정과 행동 (Resolution)

**예상 분량**: 35~45p (B5)

---

3부는 "주사위를 굴리고 결과를 읽는다"의 심화 단원입니다. 1부가 개괄이었다면 3부는 Grade Die·d100·d6 Pool 세 체계 각각의 정밀 규칙과 Position × Effect 매트릭스, 대립·저항·협상의 모든 경우를 다룹니다. 전투(4부)·자원(5부)이 이 단원을 전제로 설명됩니다.
