# 3부. 판정과 행동 (Resolution)

> 성수기 타이안의 고원 호텔. 괴물출현주의 경보로 모든 예약이 취소되었지만 민은 가족여행을 강행한다. 호텔측의 스위트 룸 업그레이드 서비스에 눈이 멀어.
>
> 도착했을 때쯤엔 주의가 경보로 격상되어, 무인호텔 시스템이 작동되고 있었다. 손님이라곤 민의 가족들뿐이었다.
>
> 객실에 짐을 푼 후 인공지능 직원을 따라 호텔을 안내 받던 중 — 민은 호텔 복도에서 엉뚱하게도 전사 능력 각성 상태에 들어가버렸다.
>
> 능력자 관리부서에서 말단공무직으로 일하는 딸 소이는 끔찍한 미래를 걱정하고, 부인은 안정된 수입을 기대한다. 흑사들이 부르는 단어를 떠올렸다.
>
> "그걸 뭐라고 부르더라?"
>
> "샤이닝…"
>
>
> — *FEWK Core #0006 「샤이닝」*

---

3부는 "주사위를 굴리고 결과를 읽는다"의 심화 단원입니다. 1부가 개괄이었다면 3부는 Grade Die·d100·d6 Pool 세 체계 각각의 정밀 규칙과 Position × Effect 매트릭스, 대립·저항·협상의 모든 경우를 다룹니다. 전투(4부)·자원(5부)이 이 단원을 전제로 설명됩니다.
