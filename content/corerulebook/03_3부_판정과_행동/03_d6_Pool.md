# 03-3. d6 Pool — 연료 탱크

---

d6 Pool은 **"얼마나 퍼부을 수 있나"**에 답하는 주사위입니다. 한 번의 판정이 아니라, 한 번에 많은 양을 소모하는 대규모 행동에 씁니다. 탄약, 에너지, 집중력, 신기의 연료 — 모두 d6 Pool의 이름입니다.

## 3-A. 저장 한도와 세션 시작

```
Pool 저장 한도 = BOD 수치 + PRE 수치
세션 시작 시: 50% 충전
초과분: 저장 불가
```

**예시**:
> BOD 8, PRE 6 → 한도 14개, 시작 7개 충전.
> BOD 7, PRE 3 → 한도 10개, 시작 5개 충전.

한도 이상은 가질 수 없습니다. 장전 행동으로 한도까지 채울 수 있지만, 넘지는 못합니다. "여분 탄창 하나 더"는 한도 바깥의 개념이 아니라 **Pack 특수 장비**로 해결합니다.

## 3-B. 장전·축적 — Pool 보충 행동

### 기본 장전

**행동**: 1 Action 또는 부가 행동 (전투 중 선택)
**판정**: 관련 스탯 Grade Die 굴림
**공급량**: (굴림 결과 ÷ 5) 올림 = 충전되는 d6 개수

**예시**:
> INT 9 (5d8) 해커가 "집중 축적" 선언.
> 5d8 굴림 = 27 → 27÷5 = **6개 올림** → Pool에 6d6 추가.

### 세션 경계 회복

세션 시작 시 **50% 충전**. 세션 마무리(애프터매스)에서 충전되는 건 아니고, 다음 세션 열릴 때 자동 충전입니다.

**Pack 특수 회복**:
- **카페·긴 휴식 (Pack 7-2)**: 전액 회복
- **신단 30m 내 (Pack 6-2)**: 신기 Pool 전액
- **차원 체류 특수**: 라운드당 +Nd6 (Pack 지정)

### 운 토큰을 통한 임시 공급

운 토큰 1 → d6 Pool에 +1d6 임시 추가. **소모 시 1회성**, 저장 불가. 세션 중 긴급한 순간의 안전 장치입니다(Spec §6-1).

## 3-C. 소모 — 한꺼번에 N개

### 규칙

1. Pool에서 원하는 개수 N만큼 꺼냅니다.
2. N개 d6을 **한꺼번에** 굴립니다.
3. **5~6이 나온 개수 = Hit 수**.
4. Hit 수가 용도의 효과 배수가 됩니다.
5. 소모된 d6은 세션 경계까지 **복구되지 않습니다**.

### 특수 결과

- **모든 d6가 6**: 자동 크리티컬. Hit 수의 **2배** 효과.
- **모든 d6가 1**: 펌블. 연료 낭비 + 역효과.

### 용도·Hit 해석 (기본)

| 용도 | 소모량 | Hit 해석 |
|---|---|---|
| 기관포 연사 | 5~15d6 | Hit 수 = 피해 배수 |
| 범위 공격 (수류탄·범위 신기) | 10d6 | Hit 수 = 영향 반경(격자) |
| 차원력 신기 발동 | 가변 | Hit 수 = 현상 강도 |
| 대량 해킹 (IC 돌파) | 8~20d6 | Hit 수 = 침투 깊이 |
| 집단 돌격 지원 | 팀 합산 Pool | 분배 가능 |
| 드랍아웃 전환 (Pack) | 15d6 | Hit 수 = 안착 정밀도 |

각 용도의 세부 해석은 7부(장비)·6부(FEWK 고유 규칙)에서 다룹니다.

## 3-D. d6 Pool은 Grade Die 판정과 조합될 수 있다

Pool은 단독으로도 쓰이지만, 더 자주는 **Grade Die 판정과 함께** 쓰입니다.

**예시 — 범위 수류탄**:
1. **정확도 판정**: Grade Die (REF + 사격 스킬 vs TN 15)
2. 성공 시 **위력 판정**: d6 Pool 10개 소모 → Hit 수 = 영향 격자 수

두 판정은 **독립**입니다. 하나는 정확도(능력), 하나는 위력(대량 소비). 능력의 성공과 위력의 강도는 별개의 이야기입니다.

## 3-E. Pool은 1개씩 보너스로 쓰지 않는다

FEWK 규칙의 **기본 금지 사항**입니다.

> **금지**: d6 Pool에서 1개만 꺼내 "+1d6 판정 보너스"로 쓰는 것.
>
> 그건 **운 토큰**의 영역입니다 (→ [5-2. 운 토큰](../05_5부_자원과_트랙/02_운_토큰.md)).

역할 분리는 FEWK 규칙의 명료함을 보장합니다. 이 규칙이 무너지면 **"대량 소비 자원"과 "판정 보조 자원"이 섞여** 플레이가 끈적해집니다.

## 3-F. Pool ≠ 장르 자원(DSR)

d6 Pool과 장르 자원(DSR)은 **서로 다른 자원**입니다. 이름이 비슷하게 들리지만 축이 다릅니다.

| 구분 | d6 Pool | 장르 자원 (DSR) |
|---|---|---|
| 한도 | BOD + PRE | 8 + PRE |
| 단위 | d6 주사위 | 정수 점수 |
| 용도 | 대량 소비 판정 | Position 협상·Effect 강화·플래시백 |
| 세션 시작 | 50% 충전 | 0 리셋 |
| 가득 차면 | 문제 없음 | 트라우마 트리거 |

Pool은 탄약 탱크, DSR은 서사 개입 점수. 상세는 [5-4. 장르 자원 DSR](../05_5부_자원과_트랙/04_장르자원_DSR.md).

## 3-G. 예시 — 민의 기관포 연사

> **상황**: 민이 기관포로 갱단 4명을 소사. Pool 10개 보유 중.
>
> **민**: "Pool에서 10d6 꺼내 굴릴게요."
> 굴림: **6**, 5, 2, **6**, 4, 5, 3, 1, **6**, 5 → Hit 6개
> **GM**: "기관포 기본 피해 1d6 × Hit 6 = 6d6. 4명에게 분배하세요."
> **민**: "세 명에게 2d6씩, 한 명에게 0... 아니 한 명에게는 자폭 안 하는 걸로." (분배 결정)
> **결과**: 남은 Pool = 0. 다음 장전 행동까지 기관포 사용 불가.

## 3-H. Pool 관리 팁

- **세션 중반 체크**: 절반 이하로 떨어지면 장전 행동을 계획하세요.
- **전투 전 장전**: 전투 직전 자유 라운드에 장전해두면 유리합니다.
- **팀 지원**: 두 PC가 합산 Pool로 대형 공격을 할 수 있습니다(GM 승인 필요).
- **Pack 회복 사용**: 카페·신단·안전 지대에서 전액 회복되는 기회를 놓치지 마세요.
