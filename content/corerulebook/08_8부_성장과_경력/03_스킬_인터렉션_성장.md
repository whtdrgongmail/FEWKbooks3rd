# 08-3. 스킬·인터렉션 성장

---

스탯 승급이 **재능의 도약**이라면, 스킬·인터렉션 성장은 **훈련의 누적**입니다. 자주 일어나고, 꾸준히 쌓입니다.

## 3-A. 스킬 성장

### 비용 공식

```
스킬 X → X+1 = (X+1) p 선형

0 → 1 = 1p
3 → 4 = 4p
6 → 7 = 7p
0 → 7 누적 = 28p
```

### 성장 조건

- 세션 중 해당 스킬 사용 누적 (3회 이상 권장)
- 포인트 지불
- GM 승인 (서사적 정당성)

**예시**:
> 민의 해킹 스킬 +5. 세션 10~15에서 해킹 판정 7회 사용. 세션 15 애프터매스에 해킹 +6 선언 (비용 6p).

### 상한 검증

스킬 상한 = min(스탯 ÷ 3 올림 + 3, 7). 상한 초과 불가. 스탯이 낮으면 스킬도 못 올림 (2부 §03).

## 3-B. 인터렉션 Lv 업

### 비용 공식 (Pack 05·06·07)

```
Lv 1 → 2 = 3p
Lv 2 → 3 = 5p
Lv 3 → 4 = 8p
Lv 4 → 5 = 12p
Lv 1→5 누적 = 28p
```

스킬(선형 (X+1)p)과 달리 **누진 비용**입니다. Lv 상승이 어려울수록 비용이 가파르게 오릅니다.

### 성장 조건

- 해당 인터렉션 사용 누적 (5회 이상 권장)
- 포인트 지불
- 서사적 정당성

### 기능 Lv 업 주의

기능(임플란트) Lv이 오르면 **BP% 추가 감소**. Lv 1→2: 추가 −5%. Lv 2→3: 추가 −5%. 누적 계산 필수.

**예시**:
> 뇌네 인터페이스 Lv 2 (−10% BP) → Lv 3 (−15% BP). 추가 −5%. BP% 70% → 65%.

## 3-C. 새 인터렉션 획득

### 같은 슬롯에 추가 획득

- 능력 슬롯에 새 능력 Lv 1 획득: 3p
- 기존 능력과 **동시 활성 불가**. 슬롯 내 활성 전환 필요.

**예시**:
> 능력 슬롯에 "샤이닝 발현" Lv 2 + "에고 집중" Lv 1. 라운드 시작 시 활성 1개 선언.

### 새 슬롯 추가 (드물게)

- 기본 3슬롯 외 추가 슬롯은 **Pack 특수 옵션**
- 서사적 대가 요구 (특수 수련·의례·임플란트)
- 비용: 10p 이상

## 3-D. 임플란트 교체

기능 슬롯 임플란트 교체:
- **비용**: 원 임플란트 비용 + 신 임플란트 비용
- **BP% 조정**: 기존 임플란트 해제 → BP% 절반 회복. 새 임플란트 장착 → Lv별 감소.
- **세션 시간**: 전문 의료 방문 Downtime 1 회기

**예시**:
> 민의 반사 부스터 Lv 2 (−10%) 해제. BP% +5% 회복.
> 시각 강화 Lv 3 (−15%) 새 장착. BP% −15%.
> 총 BP% 변화: −10%.

## 3-E. 태그 획득·상실

태그도 성장 대상입니다 (2부 §04).

### 태그 획득
- 서사 이벤트 기반 (새 출신·새 소속)
- 비용 = 태그 비용 (육체 3p, 성격 2p, 배경 1~5p)
- Pack 러너 태그: 5p (유형) + 3p (분류)

### 태그 상실
- 서사 이벤트 (기업 해고 등)
- 비용 환불 없음 (자연 소멸)
- 빈 태그 슬롯 재사용 가능

## 3-F. 예시 — 민의 성장 총론

> **세션 1 민**: 해킹 +5 / 인터렉션: 영감 번뜩임 Lv 1, 해킹 심화 Lv 1, 뇌네 인터페이스 Lv 2.
>
> **세션 10 민**: 해킹 +6 (+6p) / 뇌네 인터페이스 Lv 3 (+5p, BP% −5% 추가) / 새 기술 "취조 전문" Lv 1 (+3p).
>
> **세션 20 민**: 해킹 +7 (+7p) / 해킹 심화 Lv 2 (+3p) / 새 능력 "장비 소환" Lv 1 (+3p).
>
> **세션 30 민**: 해킹 심화 Lv 3 (+5p) / 영감 번뜩임 Lv 2 (+3p) / 취조 전문 Lv 2 (+3p).
>
> 세션당 평균 4p 획득 + 이정표 보너스로 약 300p 누적 사용 가능.
