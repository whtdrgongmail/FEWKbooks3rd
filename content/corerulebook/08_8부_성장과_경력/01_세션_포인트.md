# 08-1. 세션 포인트 체계

---

세션마다 PC는 **포인트**를 획득합니다. 이 포인트를 모아 스탯·스킬·인터렉션·장비·기능을 업그레이드합니다. FEWK는 레벨 대신 포인트 성장을 씁니다.

## 1-A. 포인트 획득

### 세션 기본
```
세션당 4p (Consistency Check 전수 일치)
```

참여 + 완주로 자동 획득. 특별한 활약 없이도 받습니다.

### 이정표 보너스

| 이정표 | 보너스 |
|---|---|
| **소형 성취** | +10p |
| **대형 성취** | +20p |
| **서사적 절정** | +30p |

- **소형 성취**: 한 회 임무 완수·짧은 아크 종결·주요 NPC 첫 만남
- **대형 성취**: 한 아크 완결·핵심 적 타격·중요 전환점
- **서사적 절정**: 대아크 완결·캠페인 결정적 순간

GM이 이정표 도달을 선언합니다.

## 1-B. 포인트 소비 — 우선순위

PC는 자유롭게 포인트를 배분하지만 일반적 우선순위:

1. **주 특기 스탯 Tier 승급** (1→2→3 ... 승급점)
2. **주 스킬 +1~+2**
3. **새 태그 획득**
4. **인터렉션 Lv 업**
5. **새 인터렉션 슬롯**
6. **부 스탯 개선**
7. **장비·임플란트 구매**

각 소비의 비용은 해당 부(§02·§03)의 비용표를 따릅니다.

## 1-C. 포인트 저장

- **무제한 저장 가능**: 세션 간 누적
- **저장 후 한 번에 큰 승급**: Tier 4→5 승급 (+20p) 준비에 좋음
- **위험**: 저장 중 PC 은퇴·진사 시 전액 소실

## 1-D. 예시 — 민의 성장 로드맵

> **세션 1~5**: 기본 4p × 5 = 20p. 소형 성취 1회 +10p = **30p 누적**.
> 사용: 해킹 +1 → +2 (5p), 은밀 +1 → +2 (5p), 뇌네 인터페이스 Lv 2 → 3 (5p) = 15p 사용. **저장 15p**.
>
> **세션 6~10**: 20p + 소형 성취 +10p = 30p. 총 저장 45p.
> 사용: INT 9 → 10 승급 (52 − 45 = 7p, 현재 45p로 INT 9유지 불가. 포기 저장).
>
> **세션 11~15**: 20p + 대형 성취 +20p = 40p. 총 저장 85p.
> 사용: INT 9 → 10 승급 (52p 누적 비용, 9에서 10 = +7p). 스탯 비용 45p 쌓인 상태에서 45+7=52p 도달. INT 10 달성. **잔여 33p**.
>
> **세션 16~20**: 20p + 서사 절정 +30p = 50p.
> 사용: 해킹 +6 → +7 (7p, 누적 28p), 기술 해킹 심화 Lv 2 → Lv 3 (8p). **잔여 35p**.

## 1-E. GM 포인트 관리

- 세션마다 4p 자동 지급 (양식 확인)
- 이정표 도달 시 선언 + 보너스
- 세션 종료 시 애프터매스에서 포인트 공지
- 캐릭터 시트에 총 획득·사용·잔여 기록
