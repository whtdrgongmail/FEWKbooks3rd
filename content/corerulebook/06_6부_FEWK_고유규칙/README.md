# 6부. FEWK 고유 규칙 (FEWK Specifics) ★★ 책의 심장

**예상 분량**: 60~70p (B5)

---

6부는 FEWK 세계를 FEWK답게 만드는 규칙들이 모입니다. 흑공 농도·신기 판정·샤이닝/매팅/현화 트랙·코핀으로 차원을 넘는 방법·파이오니아·드랍아웃·인과율·시간도보까지. 1~5부가 "어떤 세계에 대해서도 쓸 수 있는 엔진"이었다면, 6부는 "이 세계이기 때문에 존재하는 규칙"입니다.
