# 부록 J. Player Handout — in-world 인쇄용 자료 텍스트

> GM이 세션에서 PC들에게 직접 건네줄 수 있는 in-world 자료의 *텍스트 원본*. 그래픽 디자인은 별도 작업이며, 본 부록은 텍스트만 제공. 5종 양식 + 추가 7종 향후 작업.

---

세션의 PC가 손에 진짜 종이를 받는 순간 몰입이 한 단계 깊어집니다. Call of Cthulhu·Trophy·World of Darkness가 50년 가까이 핸드아웃 전통으로 쌓아온 강점입니다. 본 부록은 FEWK용 핸드아웃의 *텍스트 원본*을 제공합니다 — GM이 그대로 출력해 PC에게 건네거나, 그래픽 디자이너와 협업해 디자인 후 인쇄할 수 있습니다.

라이선스는 본문과 동일 FWOL v2.1 (CC BY 4.0). 인쇄·재배포 자유.

---

## J-1. (주)신성 영화상 카드 광고

세션 도입에서 PC에게 *광고 전단지*로 배포. 종이 한 장 양식.

```
═══════════════════════════════════════════
        ┃  (주)신성 영화상 카드  ┃
═══════════════════════════════════════════

       당신의 가장 깊은 비밀을
       가장 안전하게 보관하는 방법

           — SINSUNG MOVIE CARD —

  영화상 카드는 단순한 기억 저장이 아닙니다.
  그것은 *재생되지 않는 한 봉인된 진실*입니다.

  ◆ 한 번의 다이브로 평생의 한 장면을 저장
  ◆ (주)신성 인증 재생사만 안전하게 재생 가능
  ◆ 광전 두 닢부터 — 평생 한 번의 투자

   "다른 카드를 가지고 가십니까?
        — 어떤 재생사가 그것을 본 적이 있습니까?
       (주)신성 카드는 재생사가 보지 못하도록
            설계된 유일한 카드입니다."

   문의: 새우거리 (주)신성 지사 / 자본구 본사
   재생 보증: 주식회사 신성 ⓒ 144년 (현재)

═══════════════════════════════════════════
        ⚠ 본 카드는 본인에게만 재생 가능
            타인 재생 시 효력 무효
═══════════════════════════════════════════
```

## J-2. 러너 의뢰서 양식

GM이 PC에게 새 일거리를 *의뢰서 양식*으로 건네줌. 빈 양식이라 GM이 사전에 채워넣음.

```
═══════════════════════════════════════════
           RUNNER CONTRACT FORM
              러너 의뢰서 (#______)
═══════════════════════════════════════════

[기본 정보]
의뢰 번호  : _______________________
브로커명  : _______________________
의뢰일    : _______________________
완료 기한 : _______________________

[의뢰 종류]
□ 운반   □ 회수   □ 호위   □ 조사
□ 잠입   □ 탈출   □ 방어   □ 탐색
□ 중개   □ 기타: _________________

[표적 / 대상]
대상명    : _______________________
위치      : _______________________
세부 사항 : _______________________
            _______________________

[보수]
선금 : ___________ 오얏 / 광전 ____닢
완수 : ___________ 오얏 / 광전 ____닢
부수 : _______________________

[알려진 위험]
☐ 흑공 농도 (예상) : ____ 단계
☐ 적대 세력         : __________
☐ 차원 이동 필요    : __________
☐ 경비·순찰        : __________
☐ 기타            : __________

[브로커 서명]                [러너 서명]
___________________     ___________________

═══════════════════════════════════════════
   본 의뢰는 러너 길드 ____________ 의 중개로
   체결되었으며, 의뢰자·러너 양측의 신뢰 위에
              유효합니다.
   분쟁 시 길드 중재 절차에 따릅니다.
═══════════════════════════════════════════
```

## J-3. 미모타임즈 신문 기사

세션 도입에서 *오늘 아침 신문*으로 배포. 한 면 한 기사 양식.

```
═══════════════════════════════════════════
              MIMO TIMES
       미모타임즈 — 144년 ___월 ___일
═══════════════════════════════════════════

          [사회면 톱]

   새우거리, 흑공 농도 4단계 측정에도
       대피 명령 늦춰져 — 시민 12명 일시 격리

═══════════════════════════════════════════

  본지 단독 입수: 어제 새우거리 일대에서
  발생한 흑공 분출은 측정 직후 농도 6단계로
  치솟았으나, 대사관 환경관리국의 공식 발표는
  4단계로 보고됐다. 인근 (주)신성의 자체 측정은
  6단계를 기록한 것으로 알려져, 측정 기준의
  격차를 둘러싼 논란이 재점화될 전망이다.

  현장에서 격리된 시민 김 모씨(34)는
  본지에 다음과 같이 증언했다:

    "공기에서 그 *맛*이 났습니다. 4단계는
     거짓말입니다. 우리는 그 자리에 있던 사람들이고,
     숨이 막힌다는 게 무슨 뜻인지 알아요."

  대사관 환경관리국은 "측정 시점의 차이일 뿐
  공식 수치에 오류는 없다"고 밝혔으나,
  (주)신성 측은 별도의 입장 표명을 거부했다.

  러너 길드 *롱사*의 공식 성명은:
    "분출 직후 30초의 측정값과 30분 후의
    측정값이 다른 것은 자연스럽다. 그러나
    *어느 수치를 시민에게 알리는가*는
    선택의 문제다."

  관련: 새우거리 강제 이주 논의 5면 / (주)신성 채굴권 8면

           기자 — 박ㅇㅇ
═══════════════════════════════════════════
       MIMO TIMES — Truth Speaks Quietly
═══════════════════════════════════════════
```

## J-4. 익명의 러너 게시판 스레드

세션 중 PC가 *정보 검색*에 성공했을 때 화면 출력 형태로 배포.

```
═══════════════════════════════════════════
   [러너넷] 익명 러너 게시판 — 144년 ___월
═══════════════════════════════════════════

[제목] 어제 새우거리 흑공 분출 — 누가 거기 있었나
[작성자] @noir_runner_99
[조회] 2,341  [댓글] 47

> 어제 분출 현장에 있던 러너들. 거기 *세 명*은
> 분명 우리 쪽이었다. 한 명은 맥주병 삥 들고
> 진짜 분출 직후 30초 안에 표지판 옆에 서 있더라.
> 농도 측정기 들고. 그 자가 누군지 안다.
>
> 의문 하나 — *그 자가 누구의 의뢰로 거기 있었는가*.

  └─ @scrap_dancer 그건 묻지 마라. 묻는 자가
     다음 날 다음 분출 현장에 들어가게 된다.
  └─ @glass_eye_jin (주)신성 직속이다. 5년 전부터
     분출 직후 측정 전담. 민간 러너 가운으로 위장.
  └─ @noir_runner_99 그 자의 측정 데이터가
     공식 발표보다 *2단계 높았다*고 들었다.
  └─ @brown_ash 그게 (주)신성의 사업 모델이다.
     민간 데이터는 시장가, 공식 데이터는 안전가.
     두 데이터 격차가 채굴권 가격을 만든다.
  └─ @noir_runner_99 그러니까 — 우리가 *정확한*
     측정값을 알고 있으면, 그게 (주)신성에게
     *위협*이라는 거지.
  └─ @scrap_dancer 그 위협이 어제 새우거리에
     거주자 세 명을 *조용히* 사라지게 했다.
     말은 안 했지만 거리에서 다 알고 있다.

═══════════════════════════════════════════
   [러너넷] 본 게시판은 익명이며 검색·인덱싱
   되지 않습니다. 그러나 *모니터링*은 됩니다.
═══════════════════════════════════════════
```

## J-5. 광전 한 닢 인증서

PC가 광전을 입수했을 때 *동전과 함께* 건네지는 인증서. 박물관·환전상의 양식.

```
═══════════════════════════════════════════
           OFFICIAL CERTIFICATE
        광전(光錢) 진품 인증서 #_______
═══════════════════════════════════════════

   본 광전은 광제 시대(수평 ____년경) 주조된
   진품 금화임을 인증합니다.

[물리 사양]  (※ 캠페인별 GM이 채움 — 광전 사양 캐논 미정)
  중량      : ____ g
  순금 함량 : ____ % (광제 시대 주조 표준)
  지름      : ____ mm
  두께      : ____ mm

[각인]  (※ 권장 — GM 캠페인에 맞춰 변경 가능)
  앞면      : 광제(光帝) 좌상 또는 황실 인장
  뒷면      : 4대 신수 사괘 또는 광제사직계 문양
  측면      : 주조 연도 + 광제사직계 인장

[저장 바이트]
  내장 바이트 : 약 60만자 (한 사람 차원 이동 분량)
  바이트 봉인 : 광제 시대 봉인 — 현대 기술 미해독
  활성화      : 광전 매개체 + 차원 이동 의식 필요
              (현대에는 사용 권장하지 않음 —
               초차원적 존재화 위험)

[현재 시세]
  금값 기준 시세  : ___________ 오얏
  희귀성 프리미엄 : 200~300%
  실거래 추정가   : ___________ 오얏

[감정]
  감정사    : _______________________
  감정일    : _______________________
  감정 기관 : _______________________

═══════════════════════════════════════════
  ⚠ 본 광전을 *실제 차원 이동에 사용*하실 경우
       반드시 자격 보유 차원 이동 시술자와
              상담하십시오.
   부작용: 다인격화·매팅·진사 위험.
═══════════════════════════════════════════
```

## 인쇄·디자인 가이드

### 권장 양식

- **A4 또는 B5 1매** — 한 핸드아웃 한 면
- **흑백 인쇄** — 톤은 사이버펑크답게 검정 위주
- **종이질** — 일반 A4 인쇄로 충분 (몰입 위해 일반 종이가 더 나음)
- **글꼴** — 신문은 명조체, 광고는 굴림체, 게시판은 모노스페이스, 인증서는 명조체+영문 세리프

### 디자인 협업

그래픽 디자이너와 협업 시 본 부록의 텍스트를 그대로 전달. 디자이너가 in-world 비주얼(로고·테두리·레이아웃)로 변환. 디자인 완성본은 PDF/PNG로 별도 배포.
