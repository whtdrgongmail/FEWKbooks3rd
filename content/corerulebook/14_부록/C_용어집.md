# 부록 C. 용어집 (Core+Pack+World 통합)

---

FEWK 3판의 모든 용어를 한 곳에 모은 통합 사전. 각 용어 뒤에 **[Core]·[Pack]·[World]** 중 출처 표기.

## C-1. 정렬 규약

- **가나다순** 우선
- **알파벳**은 별도 섹션
- 각 용어: 정의 1~2문장 + 출처
- → 표시: 관련 용어 링크

## C-2. 주요 용어 (요약, 전체는 10~12p)

### ㄱ

- **간수호사(看守護士) [World]**: 수호사의 하위 계급. 순찰·경비 담당.
- **개척 차원 [World]**: 뇌네 네트워크 등록된 차원. 코핀 진입 가능. → **파이오니아**
- **경계 (BP% 70~51) [Pack]**: BP% 중간 구간. 사회 대우는 정상이나 일부 경계. → **생체순도**
- **공단계 [World]**: (주)신성 운영 대표 인공 차원. 수평 101년 건설.
- **궁정구 [World]**: 6스타일지구 중 하나. 왕실·귀족 중심. BP% 100% 표준.

### ㄴ

- **뇌네 (NOENE) [Pack/World]**: 차원 이해 주파수 보정 네트워크. 코핀 4층 팩시밀리의 1층.
- **뉴로덱 [World]**: 뇌네 인터페이스 개인 덱. 해커 표준 장비.

### ㄷ

- **다이브 [Pack]**: 코핀을 통한 차원 이동. → **코핀**
- **단계 승급 [Core]**: 스탯 Tier 변화 (d4 → d6 → d8 등). "경지 돌파"의 수학적 표현.
- **대실패 [Core]**: 판정값이 TN보다 −10 이하. 실패 + 큰 대가. → **Fumble**
- **대아크 [Core]**: 20~30 세션 이상의 중심 서사. → **아크**
- **돌칩 (Dolchip) [World]**: 신기 저장 크리스탈. 4 등급 (흰·회·흑·홍).
- **동백국 [World]**: 동방의 하얀 나라. (v0.2 교정: 이전 "백룡국")
- **동쟁 [World]**: 10대사국 대전쟁. 원전 사건.
- **드랍아웃 [Pack]**: 차원에서 강제 이탈 (6부 §07).
- **DSR (Dynamic Scene Resource) [Pack]**: FEWK 장르 자원. 8+PRE 한도. → **장르 자원**

### ㅁ

- **만진법 [World]**: 4자리 어퍼스트로피 구분 숫자 표기.
- **매팅 [Pack]**: 흑공 침식 트랙. 10 도달 시 종결.
- **먹개비 [World]**: 인과율 균열에서 태어난 존재. 5 등급.

### ㅂ

- **바이트 계산기 [World]**: 흑공·바이트 밀도 정밀 측정기.
- **배틀마커 [Pack]**: 러너 태그. Critical +5·Initiative +1 Die.
- **백공(白空) / 엑토플라즈마 [World]**: 바이트만 있는 정보 물질. 흑공의 대칭 쌍.
- **벽사종 [World]**: 신기 범위 무기 (3d4).
- **BP% (Body Purity) [Pack]**: 생체순도. 0~100% 7구간. → **생체순도**

### ㅅ

- **샤이닝 [Pack]**: 신기 공명 트랙. 10 도달 시 각성.
- **생체순도 [Pack]**: BP%의 다른 이름. 육체·정신·사회 순도. → **BP%**
- **수호사(守護士) [World]**: 치안·경비 직업. 순찰·대응. → **간수호사**
- **신기(神氣) [World]**: 의도에 형태 부여하는 정보 물질. 신사가 조작.
- **신단 [World]**: 신기 응축 공간. 30m 반경 활성.
- **신사(神士) [World]**: 신기 조작 직업. 신성재단 소속 다수.
- **신성재단 [World]**: 신기 관리 종교·기업 조직.

### ㅇ

- **Ace (폭발 주사위) [Core]**: 개별 주사위 최대값 → 재굴림 누적.
- **abyssi [World]**: 상위 1 차원 계층. 흑공의 원천.
- **에고 [Core]**: 자의식 내구. max(PRE×2, 4). 1/10 감쇠 법칙.
- **엑토플라즈마 / 백공 [World]**: → **백공**
- **오얏 [World]**: 화폐 단위. 1 오얏 = 10,000 크레딧 = 10,000원.
- **오지 차원계 [World]**: 미개척·극단 환경 차원.
- **운 토큰 [Core]**: LCK 기반 자원. LCK × 2 최대. 9 용도 (Spec §6-1 v0.3.1).
- **유령 상태 [World]**: 육체 재구성 불완전 상태. 물리 행동 제약.
- **윤회 [World]**: 혼백 21자의 재순환 (미래 확장).

### ㅈ

- **장르 자원 [Core]**: Pack별 이름 (FEWK = DSR). 8+PRE 한도. → **DSR**
- **진사 (永死) [Pack]**: 육체·에고 완전 소멸. 복귀 불가.

### ㅊ

- **차원 계층 [World]**: 상위 2 → abyssi → 물질계 → 하위 1~6 8단 구조.
- **첨작 [World]**: 4대 신수 중 1. 하늘 신수.
- **청호 (靑虎) [World]**: 4대 신수 중 1. 호랑이 신수. (Pack v0.3 E-2 교정: 이전 "청린")

### ㅋ

- **코핀 (Coffin) [World]**: 차원계 팩시밀리 장치. 4층 구조 (뇌네·흑공·엑토플라즈마·혼백 21자).
- **코핀 4층 팩시밀리 [World]**: 코핀의 구조·원리. v0.2.4 확립.
- **Critical [Core]**: 판정 차이 20+ 또는 폭발 체인. 최대 효과.
- **크레딧 (credit) [World]**: 화폐 단위. 1 크레딧 = 1 KRW.

### ㅍ

- **Partial [Core]**: 판정 차이 0~4. 성공 + 대가.
- **파이오니아 [Pack]**: 미개척 차원 최초 진입자. 코핀 불가.
- **패러독스 [Pack]**: 인과율 왜곡 트랙. 10 도달 시 먹개비 조우 + 드랍아웃.
- **팩시밀리 [World]**: 코핀 비유. 차원계 복사·전송·재구성. → **코핀**
- **페이 [World]**: 러너 브로커. 원전 NPC.
- **퓨어로열 [World]**: 8도 지방 중 흑공저택 출현 지역.

### ㅎ

- **혼백 21자 [World]**: 영혼의 정보 무게. 영혼 21g 과학적 밈 유래.
- **Full [Core]**: 판정 차이 5~9. 온전한 성공.
- **흑공(黑空) [World]**: 상상에 반응하는 정보 물질. 세계의 기본 재료.
- **흑공앰플 [World]**: 최초 진입용 흑공 주입제. 4 등급.
- **흑사 [World]**: 흑공 조작 직업. 신사의 대칭.

## C-3. 영문 용어

- **Ace** → 폭발 주사위
- **Bond** → 유대
- **BP (Body Purity)** → 생체순도
- **Clock** → Faction Clock
- **Critical** → 판정 차이 20+
- **Downtime** → 휴지기 행동
- **DSR** → Dynamic Scene Resource
- **Effect** → 결과 크기 3~4 단계
- **Faction Clock** → 세력 진행도 트랙
- **Fumble** → 모든 주사위 1
- **Grade Die** → 스탯 주사위
- **IC / OOC** → In-Character / Out-of-Character
- **NOENE (뇌네)** → 차원 이해 주파수 네트워크
- **NPC** → Non-Player Character
- **ORPG** → Online Roleplaying Game (실시간 온라인 세션)
- **PBeM** → Play by eMail
- **PC** → Player Character
- **Pool** → d6 Pool
- **Position** → 위험 수준 3 단계 (Controlled/Risky/Desperate)
- **SSOT** → Single Source of Truth
- **TN** → Target Number
- **Tier** → Grade Die 등급 (1~6)
- **TPK** → Total Party Kill

## C-4. 폐기 용어 (이전 버전)

이전 FEWK 버전에서 쓰였으나 v0.3에서 **폐기**된 용어:

- **DV (Difficulty Value)**: TN으로 대체 (v0.2 → v0.3)
- **AC (Armor Class)**: 방어 수정치 대립 판정으로 대체
- **d20 단독 판정**: Grade Die로 대체
- **6+PRE Pool 공식**: 8+PRE로 대체
- **Effect +0.5 누적식**: +1 단순 단계로 대체
- **스탯 비용 2/5/10/18/31/52p**: 새 누적 비용표로 대체
- **청린(靑鱗)**: 4대 신수 중 하나가 "청호(靑虎)"로 교정 (Pack v0.3 E-2)
- **백룡국**: "동백국"으로 교정 (Pack v0.3 E-2)
- **수호자(守護者)**: "수호사(守護士)"로 통일 (원전 확인, v0.2.4)
- **6구역**: "6스타일지구"로 변경 (v0.2 2차)
