# 부록 B. 표 요약·치트 시트

---

테이블에 두고 참조하는 **요약 표 모음**. PDF에서 분리 인쇄 권장.

## B-1. 판정 치트 시트 (1페이지)

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 FEWK v0.3 Grade Die 치트 시트 — 판정
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

◆ 주사위 3 종류
  Grade Die (d4~d20) — 내 능력
  d100              — 세계 확률
  d6 Pool           — 대량 소비

◆ 스탯 → Grade
  1~4   → 1~4d4   (초심자)
  5~8   → 3~6d6   (일반인)
  9~12  → 5~8d8   (숙련자)
  13~16 → 7~10d10 (고수)
  17~20 → 9~12d12 (대가)
  21+   → 8d20+   (전설)

◆ 판정 공식
  판정값 = Grade 합 + 스킬 + 수정치
  vs TN (5/10/15/25/40/60/100)

◆ 차이값 해석
  모든 1    : Fumble (자동 실패 + 최악)
  ≤ -10    : 대실패
  -9 ~ -1  : 근소 실패
  0 ~ 4    : Partial (성공 + 대가)
  5 ~ 9    : Full (온전)
  10 ~ 19  : Great (보너스)
  20+      : Critical (최대 + 드라마)

◆ 폭발 주사위 (Ace)
  주사위 최대값 → 재굴림 + 누적 (무한 체인)

◆ 파생 수치
  HP          = BOD × 5
  에고        = max(PRE×2, 4)
  저항        = 에고 ÷ 3
  DSR Max     = 8 + PRE
  운 토큰 Max = LCK × 2
  d6 Pool Max = BOD + PRE
  Initiative  = REF Grade + SEN Grade (합산)

◆ TN 감각
  5:쉬움 10:표준 15:어려움 25:매우 40:극한 60:전설 100:신화

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## B-2. 자원·전투 치트 시트 (1페이지)

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 FEWK v0.3 치트 시트 — 자원·전투
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

◆ 운 토큰 9 용도 (Spec §6-1 v0.3.1)
  1 토큰: 재굴림 1회
  1 토큰: +스탯 Tier 주사위 1개
  1 토큰: 에고 1d6 회복
  1 토큰: d6 Pool +1d6 임시
  2 토큰: 자동 크리티컬 (+20)
  2 토큰: Fumble 회피 (자동 재굴림)
  2 토큰: 스케일 전환 선언
  3 토큰: 운 판정 강제 성공
  5 토큰: 진사 직전 회피 → 드랍아웃

◆ 장르 자원 (DSR) 소모
  +1: Position 1단계 완화
  +1: Effect 1단계 상승
  +1: 짧은 플래시백
  +2: Effect 2단계 상승
  +2: 중간 플래시백
  +3: 확장 플래시백

◆ d6 Pool (대량 소비 전용 — 1개씩 금지)
  장전: Grade Die 굴림 ÷ 5 올림 = 공급 d6
  소모: N개 동시 굴림, Hit(5~6) 수 = 효과 배수

◆ 방어 수정치 (최대 +13)
  경장 +1 · 중장 +3 · 중갑 +5
  엄폐 1/2 +2 · 엄폐 3/4 +3 · 완전 +5
  임플란트 (피부) +1~3

◆ 피해 배수 (차이값)
  빗나감 (-) / Partial ÷2 / Full ×1 / Great ×2 / Critical ×3 + 부위

◆ 부위 효과 d6 (Critical 시)
  1 추가 피해 / 2 무장 해제 / 3 이동 제한
  4 상태 이상 / 5 공격자 +행동 / 6 대상 에고 -1d4

◆ HP 단계
  0 기절 → -1~-BOD 빈사 → -BOD×2 이하 진사 (운 5 회피)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## B-3. GM 치트 시트 (1페이지, 선택)

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 GM 치트 시트
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

◆ 세션 5단계
  도입 → 전개 → 전환 → 절정 → 수습

◆ NPC 4 등급 (주 특기 스탯)
  D: 2d4~4d4    C: 3d6~5d6
  B: 6d6~7d8    A+: 8d8~9d12

◆ Position × Effect 매트릭스
         Limited  Standard  Great
  Ctrl    작음    보통      이상
  Risky   비효율  기본      풍부
  Desp    비효율  무거움    영웅

◆ 즉석 NPC 10초 생성
  이름 + 역할 + 행동 1줄

◆ 사기 체크 트리거
  D급 HP 50%: 도주 체크
  C급 HP 25%: 항복 체크
  B급 HP 10%: 최후 저항

◆ Clock 크기
  4 segs: 짧은 사건 (1~2 세션)
  6 segs: 중간 작전 (3~5)
  8 segs: 대작전 (5~10)
  12 segs: 대아크 (20~30)

◆ 문제 대응
  규칙 충돌 → 확인 → 결정 (2분 이내)
  감정 사건 → X-Card → 개인 대화
  시간 초과 → 세션 저장

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```
