# 부록 E. World Book 참조 가이드

---

FEWK Core Rulebook과 World Book은 **자매 권**입니다. 같은 주제가 두 책에서 다른 각도로 다뤄집니다. 이 부록은 양권을 함께 쓰는 실무 도구입니다.

## E-1. 양권 분담 다이어그램

```

               "코핀"이라는 한 주제
```

## E-3. 양권 사용 시나리오

### 시나리오 A: GM의 첫 세션 준비

```
1. Core Rulebook 1부~3부 읽기 (판정 체계)
2. Core Rulebook 10~12부 읽기 (GM 가이드·NPC·시나리오)
3. World Book 1부~3부 훑기 (세계관 톤)
4. World Book 해당 지구 (예: 서민구 6부 §01/06) 정독
5. Core Rulebook 12부 §05 or §06 시나리오 선택
6. 양권 치트 시트 (Core 부록 B) 준비
```

### 시나리오 B: 플레이어 PC 생성

```
1. Core Rulebook 1부 (게임 기초) 읽기
2. Core Rulebook 2부 (캐릭터) 정독
3. Core Rulebook 2부 §06 프리셋 선택
4. World Book 해당 출신 지구 읽기
5. World Book 9부 (직업·역할) 원전 참고
6. Core Rulebook 2부 §09 유대 설정
```

### 시나리오 C: 캠페인 설계

```
1. Core Rulebook 8부 §05 (후기 캠페인)
2. Core Rulebook 10부 §07 (Faction Clock)
3. Core Rulebook 12부 §04 (아크 템플릿)
4. World Book 10부 §05 (주요 작전 연표)
5. World Book 3부 (역사) — 시점 설정
6. Core Rulebook 11부 (NPC 카탈로그)
```

### 시나리오 D: 설정 질문 해결

```
"왜 흑공 농도 7에서는 판정이 어려워지나?"

1. Core Rulebook 6부 §01 (수치·규칙)
2. World Book 2부 §01 (흑공의 본질·역사)
3. World Physics (엔진) PHYS-T01~T10 (물리 원리)
```

## E-4. 양권 간 교차 링크 규약

Core Rulebook 본문에서 World Book 참조 시:
- **링크 형식 (템플릿)**: 대괄호 안에 `World Book §X/Y`, 괄호 안에 World Book 실 경로 (상위 두 단계 + 부 디렉토리)를 넣습니다.
- **"→ World Book §XX 참조"** 표기 권장
- 세계관 설명 **4문장 이상**은 World Book 위임

World Book이 Core Rulebook 참조 시:
- "→ Core Rulebook §XX 참조"
- 규칙 판정 세부는 Core 위임
