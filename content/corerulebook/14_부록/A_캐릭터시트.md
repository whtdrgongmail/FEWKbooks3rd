## A-2. 웹용 YAML 시트

상세는 13부 §09 참조. 간략 포맷:

```yaml
character:
  name: ___
  preset: G-___
stats: {BOD: _, REF: _, INT: _, SEN: _, PRE: _, LCK: _}
derived: {HP: _, ego: _, ego_max: _, DSR: _, luck_tokens: _, d6_pool: _}
skills: {hacking: _, stealth: _, ...}
tags: {body: "", personality: "", background: "", runner: []}
interactions:
  power: {name: "", level: _}
  skill: {name: "", level: _}
  function: {name: "", level: _, bp_cost: _}
bp: _
tracks: {shining: 0, matting: 0, paradox: 0, corporeal: "locked"}
heat: {}
bonds: []
equipment: {}
wealth: {cash: 0, credit: 0, debt: 0}
trauma: []
```

## A-3. 프리셋 G-0~G-8 완성 시트 9종

각 프리셋(§2-6)의 30p 자동 구성 + 기본 장비 완성본. 플레이어가 **남은 70p만** 커스터마이징하면 됨.

### G-0 사회인 (완성 예)

```yaml
preset: G-0
stats: {BOD: 4, REF: 5, INT: 4, SEN: 4, PRE: 7, LCK: 3}
starting_bp: 95
implants: ["매팅 억제기 Lv 1"]
external_weapon: "까마귀 P-21 (3p 고정)"
tags: {body: "언어 능력", personality: "친화력"}
recommended_skills: ["사교 +3", "설득 +2"]
```

(G-1~G-8도 같은 포맷, 상세는 2부 §06 참조)

## A-4. 예시 빌드 5종 완성 시트

2부 §08의 5 빌드 (민·수덕·레이라·카이·야조) 완성 시트가 포함됩니다. 각 1페이지 분량, 그대로 플레이 가능.

## A-6. 인쇄 팁

- B5 앞뒤 2페이지 = 1 PC
- A4로 확대 가능
- 플라스틱 시트 프로텍터 권장 (장기 사용)
- 연필 사용 권장 (수정 용이)

## A-7. 시트 사용 가이드

### 생성 시
1. 상단 기본 정보
2. 스탯 → 파생 수치 자동 계산
3. 스킬 분배
4. 태그 선택
5. 인터렉션 3 슬롯
6. BP% 산출
7. 장비 기록

### 세션 중
- HP·에고·DSR·운 토큰 수시 갱신
- 트랙·트라우마 애프터매스에 갱신

### 세션 후
- 자원 상태 스냅샷
- 성장 포인트 기록
