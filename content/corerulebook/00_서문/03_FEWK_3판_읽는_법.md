# 00-3. FEWK 3판 읽는 법

---

FEWK 3판은 **두 권**입니다. 당신이 손에 든 이 책이 **Core Rulebook** (규칙서), 옆에 있는 두꺼운 책이 **World Book** (세계관서). 두 권이 합쳐져 하나의 FEWK를 만듭니다.

## 3-A. 두 권의 구조

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
             FEWK 3판 (두 권)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ┌─ 본 책: Core Rulebook ─┐     ┌─ 자매 권: World Book ─┐
  │                        │     │                       │
  │ 14부 + 부록 A~F         │     │ 12부 + 부록           │
  │ 560~700p                │     │ 440~470p              │
  │                        │     │                       │
  │ "어떻게 플레이"         │     │ "누가·무엇·어디"       │
  │                        │     │                       │
  └────────────────────────┘     └───────────────────────┘

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

두 권은 **분담**하되 **같이 씁니다**.

## 3-B. 먼저 펴는 권 선택 가이드

### 당신이 플레이어로 처음 참여한다면

**Core Rulebook 1부·2부**를 먼저. 그리고 **World Book 1부** (세계 톤).

이후 **Core Rulebook 2부 §06** (프리셋)을 보고 PC 선택. **World Book 해당 출신 지구**를 읽어 배경 잡기.

### 당신이 세계관 팬으로 읽는다면

**World Book 1부** (세계 톤) → **World Book 2부** (세계 법칙) → **World Book 4부** (지리) → **World Book 9부** (존재도감).

Core Rulebook은 **부록 C 용어집**과 **부록 E 양권 참조**만 훑기. 게임 안 할 거면 나머지는 읽지 않아도 됨.

### 당신이 GM이 되려 한다면

두 권 교차 독서:
1. Core 00 서문·1부 (게임 기초)
2. World Book 1~2부 (세계관 기반)
3. Core 3~7부 (규칙 본문)
4. World Book 4~7부 (지리·사회·러너)
5. Core 8~12부 (성장·NPC·시나리오)
6. World Book 9~10부 (인물·세력)
7. Core 13부·부록 (디지털·도구)
8. World Book 3부·8부 (역사·차원)

총 독서: 약 25~30시간. 캠페인 시작 전 완독 권장.

## 3-C. 양권 링크 표기 규약

이 책에서 World Book을 참조할 때:

### 구체 링크

```
[World Book 2부 §01 흑공](../../[3판%20FEWK%20World%20Book]/02_2부_세계법칙/01_흑공/)
```

### 서술 중 참조

```
"흑공의 역사·문화는 → World Book 2부 §01 참조."
```

### 분담 선언

```
"본 Core Rulebook은 흑공의 판정 규칙만 다룸. 흑공의 종교·문화적 의미는 World Book이 전담."
```

## 3-D. 엔진 3종은 무엇인가

이 책·World Book 뒤에는 **엔진 3종**이 있습니다:

1. **MEJE Framework Core v0.3**: 장르 중립 엔진 (다이스 체계·3 Layer)
2. **FEWK Cyberpunk Pack v0.3**: FEWK 장르 팩 (러너·흑공·신기)
3. **FEWK World Physics v0.1**: 세계 물리 법칙 SSOT

이들은 **설계 명세 문서**이며 **일반 독자는 읽지 않아도 됩니다**. 이 책은 엔진의 규칙을 **독자 친화 서술**로 재작성한 **출판 원고**입니다.

엔진 접근 경로는 부록 F 참조. Pack·엔진 제작 관심 있는 GM·작업자만 참고.

## 3-E. 웹 동반 자료 (계획)

**fewk.meje.kr** (개발 중):
- 온라인 캐릭터 시트
- Roll20·Foundry 세팅 가이드
- 공식 FAQ
- 공식 AI 보조
- 커뮤니티 공유 자료

출시 시점에 본 Core Rulebook 개정판에 반영됩니다.

## 3-F. 치트 시트 사용

**부록 B 치트 시트**는 세션 중에 펼쳐 놓기 위한 용도:

- 치트 1: 판정 요약
- 치트 2: 자원·전투 요약
- 치트 3: GM 치트
- 치트 4: FEWK 특화 치트

각 1페이지. 인쇄·투명 파일 보관 권장.

첫 세션 참가자 준비
