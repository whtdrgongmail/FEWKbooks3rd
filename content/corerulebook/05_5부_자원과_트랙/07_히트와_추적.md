# 05-7. 히트·추적 시스템

---

"오늘의 러닝이 내일의 보복이 됩니다." FEWK에서 PC가 큰일을 저지르면, 누군가가 **기억**합니다. 그 기억은 세력별 **히트 트랙**으로 누적되고, 특정 수준에 이르면 세력이 움직이기 시작합니다. 히트는 장기 캠페인의 리듬을 만드는 외부 압력입니다.

## 7-A. 히트 트랙 개요

각 **세력별로 독립된 트랙**을 관리합니다. PC는 일반적으로 다음 세력들의 히트를 감당합니다:

- 23기업 각 사 (기업별 독립 트랙)
- 10대사국 각국 (국가별 독립 트랙)
- 러너 길드·갱단 (조직별 독립 트랙)
- 종교재단 (재단별 독립 트랙)

```
트랙 범위: 0 ~ 10
세션 시작: 유지 (세션 넘나들며 장기 누적)
기본: 0 — 세력의 레이더 밖
```

## 7-B. 히트 누적 트리거

### +1 (경미)

- 해당 세력 하부 조직 파괴·도난
- 소속 NPC 경미한 피해
- 세력 관련 정보 유출 소량

### +2 (중급)

- 소속 NPC 사망
- 자산 파괴 (거점 1개)
- 대외 이미지 타격 이벤트

### +3 (심각)

- 고위 NPC 사망
- 주요 자산 손실 (거점 여러 개·대형 장비)
- 공개적 도전·선언

### +5 (극한)

- 최고 경영자·대사급 NPC 사망
- 세력 근간 훼손 (본부 파괴 등)

## 7-C. 히트별 효과

| 히트 | 세력 반응 | PC 영향 |
|---|---|---|
| 1~2 | 주목 | 정보 수집·주시. PC 체감 없음. |
| 3~4 | 정보 수집 | 세력 NPC가 PC 신상 파악. 길거리 오해·미행 이벤트. |
| 5~6 | 파견 | 세력이 C~B급 요원 파견. 세션 중 정찰·추격 시도. |
| 7~8 | 본격 추적 | B~A급 추적팀. 안전 지대가 위험해짐. |
| 9~10 | 제거 명령 | 세력의 최정예·대규모 작전. PC 정면 타격. |

히트 5 이상에서 PC의 일상이 흔들립니다. 히트 9~10은 **캠페인 아크의 절정**입니다.

## 7-D. 히트 감소 방법

### 도피·잠적 (시간)

- 해당 세력 영향권 이탈 1개월 → −1
- 차원 이탈 1주 (차원 체류 비용 수반) → −1
- 매달 누적 시 효과적 (그러나 PC는 행동을 멈춰야 함)

### 뇌물·협상 (비용)

- 세력 내부 인맥 로비 → −1 (오얏 단위 상당)
- 세력 하부 청탁·화해금 → −1 (작은 세력에)
- 계산상: 히트 3을 0으로 → 약 10~30 오얏 비용 예상

### 다른 세력 전환 (히트 이관)

- "이건 저 세력이 한 짓이다" 설득 → 원 세력 히트 −2, 타 세력 히트 +1~2
- 드라마 서술 필요 (DSR +1 플래시백 결합 가능)

### 드랍아웃·차원 이탈 (일시 리셋)

- 드랍아웃 성공 후 재진입 → 이전 히트 −5 (세력이 PC를 "죽은 자"로 간주하는 기간)
- 완전 잠적 1년+ → 히트 0 리셋 (캠페인 전환)

## 7-E. 다세력 히트 관리

한 PC가 여러 세력의 히트를 동시에 가지면 매트릭스가 됩니다:

```
예시 — 민의 히트 상태 (세션 20 기준):
  ㈜신성       ████ 4 (정보 수집)
  ㈜뻑더레스    ██░░ 2 (주목)
  피스메이커    ██████ 6 (파견 시작)
  신성재단      █░░░ 1 (경미 주목)
  기타         0
```

GM은 세션 계획 시 **가장 높은 히트 세력**의 추적 이벤트를 준비하고, **보조 세력**을 배경 긴장으로 활용합니다.

## 7-F. Faction Clock과의 연동 (10부 §07)

Faction Clock은 **세력 자체의 대아크 진행도**입니다. 히트 트랙은 **PC와 세력의 관계**를 측정합니다. 두 트랙이 교차합니다:

- Faction Clock이 진행되어 세력이 확장 중이면 히트 증가 가속
- 세력이 내부 위기(Faction 클록 역전)면 히트 감소 가속
- 세력 붕괴 시 히트 자동 0 (세력이 사라짐)

## 7-G. GM 운영 가이드

### 세션 준비

- 전 세션 히트 변동 확인 (PC 행동별 +N 계산)
- 히트 3+ 세력의 **이번 세션 이벤트** 준비
- 히트 7+ 세력의 **다음 세션 추적 시나리오** 준비

### 히트 이벤트 종류

- **주목** (1~2): NPC의 "어디서 본 얼굴이군" 언급
- **정보 수집** (3~4): 미행·감시 카메라 포착·거리 루머
- **파견** (5~6): 세력 요원 1~2명 PC 조우
- **본격 추적** (7~8): B급 추적팀 고정 스토킹
- **제거** (9~10): 시나리오 자체가 세력 공격

### 히트 시각화

PC 시트에 히트 테이블 칸. 세력별 색상 구분 권장.

## 7-H. 예시 — 민의 히트 여정

> **세션 1~3**: 신참 러닝. 히트 모두 0.
>
> **세션 5**: ㈜신성 지점 침투 성공. ㈜신성 +2.
>
> **세션 8**: 침투 중 경비원 부상. ㈜신성 +1 (=3). 정보 수집 단계 진입.
>
> **세션 12**: ㈜신성 본사 공격 (대형 작전). ㈜신성 +3 (=6). 파견 시작.
>
> **세션 15**: 피스메이커 갱단 거점 파괴. 피스메이커 +2.
>
> **세션 18**: ㈜신성 추적팀 공격 받음. 전투 승리. 후속 작전으로 ㈜신성 +1 (=7). 본격 추적 진입.
>
> **세션 22**: 민이 신성재단에 접근. 신성재단 정화 의식으로 ㈜신성 히트 경감 시도 (협상). ㈜신성 −2 (=5). 다시 파견 수준.
>
> **세션 25**: ㈜신성 제거 작전 예고됨. 민은 결정 — 차원 이탈 vs 정면 대결.

히트 트랙은 **캠페인 아크의 길이와 무게**를 만듭니다.
