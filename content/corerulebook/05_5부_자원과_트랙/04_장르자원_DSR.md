# 05-4. 장르 자원 DSR

---

FEWK Cyberpunk Pack의 장르 자원은 **DSR (Dynamic Scene Resource)**입니다. 서사적 개입에 쓰는 점수이며, **Position 협상·Effect 강화·플래시백**의 세 축을 전담합니다. DSR은 하루 저녁의 드라마를 결정하는 핵심 자원입니다.

## 4-A. 공식

```
DSR 상한 = 8 + PRE
세션 시작: 0으로 리셋
```

PRE 5 → DSR 11. PRE 7 → DSR 15. PRE가 높을수록 장르 개입의 여력이 큽니다.

## 4-B. 증가 조건

DSR은 **세션 중 이벤트로 획득**합니다. 주요 증가 계기:

- **드라마 순간 제공**: 감정적 장면·중요한 선택·희생 → +1
- **장르 톤 유지**: 사이버펑크 감성의 구체적 행동·대사 → +1
- **GM 주도 시험**: GM이 "선택지 둘 다 대가 있음"을 제시하고 PC가 선택 → +1
- **팀 상호작용**: 유대 상대와의 깊은 대화·협력 → +1

GM은 세션당 PC마다 평균 **3~5 DSR 획득**을 목표로 합니다. 너무 적으면 Position 협상이 불가능, 너무 많으면 트라우마 리스크.

## 4-C. 용도

```
DSR 소모 옵션:

+1: Position 1단계 완화 (Desperate→Risky or Risky→Controlled)
+1: Effect 1단계 상승 (Limited→Standard or Standard→Great)
+2: Effect 2단계 상승 (Limited→Great 단계로)
+1: 플래시백 짧은 장면 (1 장면 · 5분)
+2: 플래시백 중간 (2~3 장면 · 15분)
+3: 플래시백 확장 (임무 전체 재구성)
+1: 장비 소환 ("사실 이 순간을 위해 준비해뒀습니다")
```

**Effect 2단계 상승 비용**: Pack v0.2 M-1 해결로 DSR +2 확정 (Core §6-5 준수). 이전 +3은 폐기.

## 4-D. 가득 차면 — 트라우마 트리거

DSR이 **상한에 도달**하면 **트라우마 1을 자동 획득**합니다(§5-5 트라우마). 상한 초과는 불가능하므로, 더 이상 DSR을 얻을 수 없는 상태가 되면 GM이 "이 장면의 감정을 감당 못하는 순간"을 트리거합니다.

따라서 DSR은 **소비해야 합니다**. 쌓아두면 폭발합니다. 아껴도 손해, 너무 써도 손해. 세션 중 **평균 60~80% 수준**을 유지하는 것이 권장입니다.

## 4-E. Position 협상의 문법

판정 직전, 플레이어가 선언:

```
"근거 한 줄. DSR +N 소모해서 Position/Effect [요청]합니다."
```

GM 판단:
- **수락**: 서사적 타당성 OK → DSR 소모, 조정 적용
- **거절**: 타당성 부족 → DSR 소모 없음, 원안 유지

**예시**:
> 플레이어: "저 임원과는 작년 동쟁에서 같이 싸운 사이입니다. DSR 1 쓰고 Risky를 Controlled로 낮출게요."
> GM: "과거 인연 좋네요. 허용. DSR −1, Controlled."

> 플레이어: "DSR 1 쓰고 Effect를 Great로. 성공하면 뇌물 정보도 같이 흘리는 걸로."
> GM: "뇌물은 과합니다. Great까지는 좋지만 정보는 별도 판정."
> 플레이어: "그럼 Great만 쓸게요."

## 4-F. 플래시백의 문법

플래시백은 **과거 장면을 지금 삽입**합니다. 판정 직전·장면 중 선언 가능.

### 짧은 플래시백 (+1 DSR)

```
플레이어: "사실 어제 저는 경비원 존과 술을 마셨습니다."
GM: "좋아요. 이번 사교 판정에 +1 Die."
```

1 장면, 약 5분 분량. 간단한 사실 하나 확정. 판정 보너스.

### 중간 플래시백 (+2 DSR)

```
플레이어: "일주일 전 이 장소를 사전 답사했습니다. 함께 간 해커가 백도어를 설치했어요."
GM: "좋아요. 2~3 장면으로 과거를 재현합시다."
```

2~3 장면, 15분. 구체적 사전 준비. 여러 판정에 보너스.

### 확장 플래시백 (+3 DSR)

```
플레이어: "사실 이 임무 전체를 1년 전부터 준비했습니다."
GM: "좋아요. 한 회 일부를 그 준비 과정으로 쓰죠."
```

임무 전체 재구성. 과거 한 회를 따로 플레이할 수도 있음 (Downtime으로 처리).

## 4-G. 장비 소환

DSR +1로 "사실 준비해뒀습니다" 선언. 단, **현실적 한계** 안에서만.

- 허용: 등급상 가질 수 있는 소품·정보
- 불허용: 등급을 초월하는 무기·대량 장비

**예시**:
> 허용: "사실 마스터키를 가지고 있었습니다. DSR 1 쓰겠습니다."
> 불허용: "사실 미사일 발사기를 가지고 왔습니다. DSR 1 쓰겠습니다." → GM 거절.

## 4-H. 관리 전략

### 플레이어 관점

- **상한 80%를 경계선으로**: 그 이상은 트라우마 트리거 위험.
- **DSR은 주기적으로 소모**: 쌓아두지 않고 Position·Effect·플래시백에 꾸준히.
- **GM의 드라마 기회를 적극 활용**: 감정적 선택에 주저하지 마세요. DSR은 재획득됩니다.
- **중요 판정에 DSR**: 플래시백 확장(+3)은 대아크 중요 판정에.

### GM 관점

- 세션당 PC마다 3~5 DSR 지급 목표
- DSR 상한 80% PC에게 "트라우마 위험 장면" 준비
- 플래시백 시간 예산 고려 (확장 플래시백은 10분 이상 소요)

## 4-I. d6 Pool과의 분리

DSR과 d6 Pool은 **서로 다른 자원**입니다. 이름이 비슷해 보이지만 축이 다릅니다.

| 구분 | DSR | d6 Pool |
|---|---|---|
| 한도 | 8 + PRE | BOD + PRE |
| 단위 | 정수 점수 | d6 주사위 |
| 용도 | 서사 개입 | 대량 소비 판정 |
| 세션 시작 | 0 리셋 | 50% 충전 |
| 가득 차면 | 트라우마 트리거 | 문제 없음 |

이 둘은 Pack 09 · Pack 08에서 각각 정리되며, Core Rulebook에서도 **절대 섞이지 않습니다**.
