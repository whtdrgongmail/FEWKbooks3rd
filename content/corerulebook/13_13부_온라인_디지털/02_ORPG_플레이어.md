# 13-2. ORPG — 실시간 플레이어 가이드

---

ORPG 세션은 오프라인 테이블과 **유사하지만 다른** 문화가 있습니다. 음성·채팅·공유 시트의 조합에서 플레이어가 몸에 익혀야 할 실무를 정리합니다.

## 2-A. ORPG 정의

**ORPG (Online Roleplaying Game)**: 실시간 음성 채팅 기반 온라인 TRPG 세션.

- 한국 TRPG 커뮤에서 사실상 표준 온라인 양식
- Discord Voice가 주류 (2020년대 이후)
- 대안: Zoom, Mumble, IRC 음성

## 2-B. 준비물

### 필수

- **음성 장비**: 마이크·헤드셋 (품질 권장)
- **인터넷**: 안정적 연결
- **플랫폼 계정**: Discord·Zoom 등
- **캐릭터 시트**: 공유 문서 접근 (Google Sheets·Roll20 등)

### 권장

- **듀얼 모니터 or 창 분리**: 화상·시트 동시 보기
- **주사위 봇**: Discord 다이서 봇
- **BGM 환경**: 집중 가능한 공간

### 선택

- **카메라**: 얼굴 공유 (커뮤 합의에 따라)
- **그림판·메모**: 즉석 스케치

## 2-C. 세션 참여 예절

### 마이크 관리

- **음소거 기본**: 말하지 않을 때는 음소거
- **노이즈 게이트**: 배경 소음 차단
- **끼어들기 규약**: 채팅 ✋ or 손 올리기
- **식사·잡음 자제**: 음성 방해

### 선언 문법

IC (In-Character)와 OOC (Out-of-Character) 구분:

```
"민이 임원에게 묻습니다: '당신이 그 사람을 죽였죠?'" (IC)

OOC: "죄송, 스킬 판정 규칙 헷갈리는데요..." (OOC)

"민은 임원의 반응을 주의 깊게 살핀다." (IC 서술)
```

### 차례 대기·개입

- **자기 차례가 아닐 때**: 다른 PC의 행동에 반응만
- **개입 필요 시**: 채팅 ✋ 표시 → GM이 "네, 말씀하세요" 호출
- **긴급 개입**: "잠깐, X-Card입니다" (즉시)

## 2-D. 주사위 굴림 규약

### Discord 다이서 봇 (표준)

```
!roll 5d6+3  → 5d6에 +3
!roll 5d6!    → 폭발 (Ace)
!roll 1d100   → d100 저항
!pool 10      → d6 Pool 소모 (count 5+)
```

### 투명성

- **공개 굴림 기본**: 모든 판정 봇으로 공개
- **비공개 굴림**: GM 비밀 굴림만 (별도 봇 커맨드)
- **물리 주사위**: 허용하나 **굴림 결과 사진** 업로드 권장

## 2-E. 공유 시트 사용

### Google Sheets·Notion·Roll20

- 실시간 갱신
- 자원 변화 즉시 반영
- 다른 플레이어도 확인 가능

### 자원 변화 선언

- 운 토큰 사용 시: 시트 즉시 변경 + 음성 선언
- HP 감소: 시트 갱신
- 트랙 변동: 애프터매스에 일괄

## 2-F. 접속 장애·지각·조기 이탈

### 접속 장애

- 5분 이내 재접속: 잠시 대기
- 10분 초과: 남은 PC들 계속 진행, 부재 PC는 NPC 처리
- 30분 초과: 세션 저장 검토

### 지각

- 사전 OOC 공지
- 도착 시 간략 요약 받음
- 장면 도입에 합류

### 조기 이탈

- "Open Door" 원칙 — 이유 없이 이탈 가능
- OOC 간단 공지
- PC는 남은 세션 동안 NPC 처리

## 2-G. 집중력 관리

### 세션 중

- **2시간 단위 휴식**: 10~15분 쉬기
- **음식·화장실**: 휴식 시간에
- **화면 전환 방지**: 다른 앱 알림 차단

### 피로 신호

- 발언 감소
- 반응 느려짐
- 실수 증가
→ GM에게 OOC로 "잠깐 쉬고 싶어요" 요청

## 2-H. 세션 후

### 후기·감사

- 짧은 후기 (5분 내)
- 좋았던 순간 나누기
- GM·다른 플레이어에게 감사

### 일지 작성

- 개인 플레이 로그 (선택)
- PC 관점 일기
- 다음 세션 준비 메모
