# 13-10. AI 병행 플레이

---

2020년대 중반 이후, AI는 TRPG 플레이의 **유용한 보조 도구**입니다. FEWK는 AI를 적극 활용하되 **최종 판정·서사 결정은 인간**이라는 원칙을 고수합니다.

## 10-A. AI 활용 영역 5종

### 규칙 해석 참조

- 의심스러운 규칙·모호한 상황
- AI가 관련 조항 빠르게 찾음
- 단, **AI는 최종 결정자 아님** — GM이 최종

**사용 예**:
> GM: "운 토큰으로 Position 완화 되나?"
> AI 응답: "Core §7-2-D 분담 원칙. 운 토큰은 판정 보조 전용. Position 완화는 장르 자원 전담. 불가."

### NPC 목소리·전술 제안

- NPC 성격 설정 후 AI가 대사 생성
- 전술 선택 보조
- 감정 묘사 풍부화

**사용 예**:
> GM: "청지기 백 (냉정·PRE 9·숨겨진 과거)이 심문당하는 상황. 대사 3개 제안?"
> AI 응답: 3개 옵션 제시, GM이 선택.

### 판정 보조

- 주사위·TN 자동 계산
- 수정치 합산
- Ace 체인 자동 추적

### 세션 요약·일지

- 긴 세션을 요약본으로 압축
- 일지 자동 생성
- 주요 이벤트 추출

### 설정 확장

- FEWK 세계관 기반 NPC·장소 생성
- 이름·외모·배경 아이디어
- 시나리오 훅 생성

## 10-B. AI 주의사항

### 최종 판정·서사 결정은 인간

- AI는 **제안자**이지 **결정자**가 아님
- GM·플레이어가 모든 판정의 **최종 재량**
- AI 제안이 이상하면 무시·수정

### 원전·엔진 SSOT 준수 필수

- AI는 잘못된 수치·규칙 환각 가능
- **반드시 원전 교차 확인**
- 엔진 3종 + World Book을 SSOT로

### 사생활·안전 툴 준수

- 플레이어의 실명·민감 정보 AI에 입력 금지
- 세션 내용 외부 AI에 유출 시 동의 필요
- X-Card 등 안전 툴은 AI와 무관하게 유지

### AI 응답 검증

- 규칙 관련 AI 응답은 원전 확인
- 수치 자동 계산도 재검산
- 서사 제안은 GM의 창의 필터

## 10-C. FEWK 3판 공식 AI 보조 (fewk.meje.kr, 계획)

미래 계획:
- FEWK 전용 학습된 AI
- 규칙·원전 정합 보장
- NPC 생성·대사 제안
- Pack v0.3 E-3 등 Consistency Check 반영

## 10-D. AI 선택 가이드

### 범용 LLM (Claude·ChatGPT·Gemini)

- 장점: 접근성·범용성
- 단점: FEWK 전문 지식 없음 (환각 리스크)
- 사용: 일반 서술·대사 생성·요약

### 전문 TRPG AI (GM Buddy·Character AI)

- 장점: TRPG 특화
- 단점: FEWK 특화 아님
- 사용: 보조 판정·일반 NPC

### 자체 학습 AI (커뮤 자제작)

- 장점: FEWK 맞춤
- 단점: 개발 필요
- 사용: 커뮤 공용 봇

## 10-E. AI 활용 워크플로

### 세션 전

```
1. 세션 계획 AI 검토 (NPC 개요·TN 설정)
2. 부족한 NPC 대사 생성
3. 랜덤 테이블 결과 예측
4. 복선 아이디어 브레인스토밍
```

### 세션 중

```
1. 규칙 즉석 질의 (모호 상황)
2. NPC 즉석 대사 (긴급 시)
3. 판정 계산 보조
4. 서술 풍부화 제안
```

### 세션 후

```
1. 요약본 자동 생성 → GM 편집
2. NPC 상태 업데이트 정리
3. 다음 세션 훅 아이디어
4. 로그 아카이브 분류
```

## 10-F. AI 활용 금지 영역

- **플레이어 동의 없는** 실명·민감 정보 입력
- **엔진 SSOT와 모순되는** 규칙 생성 (검증 없이 적용)
- **중요 판정**의 AI 자동 결정 (GM 최종 결정 원칙 위반)
- **플레이어 간 갈등**의 AI 중재 (인간 개입 필요)

## 10-G. AI 병행 플레이어 체크리스트

### AI 활용 원칙

- [ ] AI는 보조, 인간이 최종
- [ ] 규칙·수치 AI 응답은 원전 교차 확인
- [ ] 사생활 보호
- [ ] 안전 툴 인간이 관리
- [ ] AI 제안 → GM/플레이어 검증 → 적용
