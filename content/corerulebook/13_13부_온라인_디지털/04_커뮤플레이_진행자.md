# 13-4. 커뮤플레이 — 진행자 가이드

---

**커뮤플레이**는 한국 TRPG 커뮤의 특유 문화입니다. 여러 GM·여러 파티가 **같은 세계관·타임라인**을 공유하며, 각자의 이야기가 서로 영향을 줍니다. 이 양식의 중심에는 **진행자 (Community Manager)**가 있습니다.

## 4-A. 커뮤플레이 정의

- 여러 GM·여러 파티가 같은 세계 공유
- 진행자 1명 (공식적·비공식적)
- 세션 간 정보·사건 교환
- 장기 캠페인 기본 구조

## 4-B. 진행자 역할

### 세계관 정합성 관리

- NPC·세력·사건 진행도 추적
- 설정 충돌 방지
- 타임라인 조정

### 파티 간 조율

- 겹치지 않는 임무 배분
- 파티 간 상호작용 조율
- 공통 NPC 관리

### 공용 NPC 데이터 관리

- 23기업·10대사국 주요 인물
- 시트·상태·관계 업데이트
- 여러 GM과 공유

### Faction Clock 전역 운영

- 10부 §07 Clock 개념을 **전체 커뮤 규모**로
- 여러 파티의 개입 누적 반영
- 세계 규모 사건 조율

### 세션 스케줄링

- 파티별 세션 일정 관리
- 겹침 방지
- PC 간 교차 출연 조율

## 4-C. 공용 자료 관리

### 플랫폼 선택

- **노션 (Notion)**: 위키·데이터베이스 통합
- **위키 (MediaWiki·Obsidian)**: 문서 관리
- **Discord**: 채널 분리 구조
- **Google Drive**: 공유 문서

### 핵심 문서

```
공용 자료 구조:
/세계관
  - 세력 현황 (23기업·10대사국)
  - 주요 인물 시트
  - 사건 연표
/파티별
  - 파티 1 로그
  - 파티 2 로그
  - ...
/관계망
  - NPC 관계 그래프
  - PC 간 연결
/규약
  - 커뮤 규약
  - 플레이 양식
```

## 4-D. 커뮤 규약 (세션 0의 커뮤 버전)

### 세계관 재정의 권한

- 개별 GM·PC의 재정의 한도
- 진행자 최종 재정의 권한
- 충돌 시 조정 프로세스

### 파티 간 충돌 조정

- PC 간 대립 시 개입 여부
- 세력 간 전쟁 시 다중 파티 반영

### 탈퇴·장기 결석 PC 처리

- PC 은퇴 or NPC 전환
- 자산·장비 처리

## 4-E. 공유 플랫폼 구조 샘플 (Discord)

```
# FEWK 커뮤 Discord 서버

📢 공지 채널
  - 규약
  - 주간 업데이트
  - 중요 이벤트

📖 세계관 채널
  - 세계관 위키 (고정 글)
  - NPC 카드
  - 지도·설정

🎭 파티별 채널 (각 파티 별도)
  - 파티 1 세션
  - 파티 1 IC 대화
  - 파티 1 OOC

🌐 크로스오버 채널
  - 파티 간 교류
  - 공용 NPC 대화

🎲 주사위 봇 채널
  - 굴림 기록

📝 로그 아카이브
  - 세션 요약
  - 주요 사건 기록
```

## 4-F. FEWK 세계관 활용 — 커뮤플레이 샘플 설정

### "서민구 러너 연대"

- **커뮤 프리셋**: 모든 파티가 서민구 출신 러너
- **공용 NPC**: 소화 이모·페이·렌드
- **공용 거점**: 소화 본점·특정 아지트
- **공용 적**: 피스메이커 갱단·(주)신성 지점

### "6스타일지구별 파티 할당"

- 파티 A: 궁정구 (왕실 관련)
- 파티 B: 자본구 (기업 음모)
- 파티 C: 서민구 (거리 정의)
- 파티 D: 노마드 (차원 개척)

각 파티가 자기 지구 전문가, 다른 지구에 개입 시 협조.

## 4-G. 진행자의 일정 관리

### 주간

- 일요일: 주간 보고서 작성
- 평일: 파티별 세션 조율
- 수요일: 공용 NPC 상태 업데이트

### 월간

- 첫째 주: 월간 Faction Clock 갱신
- 둘째 주: 주요 사건 공지
- 셋째 주: 파티 간 교류 이벤트
- 넷째 주: 월간 요약 작성

## 4-H. 예시 — 커뮤플레이 사건 진행

> **사건**: (주)신성 대규모 공단계 확장 (세계 규모 Faction Clock 12).
>
> **파티 A**: 기업 내부 고발자 보호 (직접 개입). Clock −2.
> **파티 B**: 고급 정보 유출 (정치 공작). Clock −1.
> **파티 C**: 서민구에 피해자 돕기 (간접). Clock +0.
> **파티 D**: (주)신성과 대립 미미. Clock +0.
>
> **진행자 조율**: 3개 파티의 개입을 종합, Clock 진행 재계산.
> **결과**: Clock 8/12 → 5/12. 대아크 지연.
>
> **주간 보고**: 각 파티에게 "네 파티의 행동이 (주)신성 확장을 지연시켰다" 공지.
