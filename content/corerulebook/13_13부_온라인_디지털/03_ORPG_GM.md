# 13-3. ORPG — GM 진행 가이드

---

ORPG GM은 오프라인 GM과 **다른 감각**을 요구합니다. 얼굴이 안 보이는 상태에서 페이싱·감정·긴장을 조율해야 합니다.

## 3-A. ORPG GM의 특수성

### 제한된 감각 신호

- 얼굴 표정 (카메라 없으면 0)
- 분위기 전달 약함
- 침묵의 의미 해석 어려움

### 보완 도구

- **음성 톤**: GM이 감정 풍부하게 표현
- **BGM·효과음**: 분위기 보조
- **이미지·맵 공유**: 시각적 보충
- **채팅 이모지**: 보조 신호

## 3-B. 세션 전 준비

### 플랫폼 점검

```
[ ] 음성·다이서 봇 테스트
[ ] 맵·이미지 업로드
[ ] 캐릭터 시트 공유 권한
[ ] 음성·채팅 분리 (권장)
[ ] BGM 파일 준비
```

### NPC 음성 구분

여러 NPC 등장 시 GM이 **음성 변조** or **프레임 태그** 사용:

```
변조: "GM이 약간 높은 톤으로 페이를 연기"
프레임: "[페이] 당신들이 찾던 사람입니다."
```

### 세션 시작 체크리스트

```
[ ] 안전 툴 재확인
[ ] 출석 확인
[ ] 지난 세션 요약 준비
[ ] 운 토큰 리셋 안내
[ ] 세션 저장 상태 확인
```

## 3-C. 세션 중 페이싱

### 발언권 분배

- 침묵하는 PC 지명: "수덕 씨, 어떻게 하시겠어요?"
- 발언 독점 방지: 한 PC당 1~2분 이내
- 채팅 ✋ 신호 주목

### 긴 서술 vs 즉답

- **긴 서술**: 도입·절정·감정 장면에 사용 (2~3문단)
- **즉답 판정**: 빠른 행동·전투에 (30초~1분)

### 지루한 순간의 컷어웨이

플레이어가 침묵하거나 지루해 하면:
- "다음 장면으로 넘어가죠."
- "그 동안 [다른 PC] 쪽으로 전환합니다."

## 3-D. 주사위 투명성

### 공개 굴림 (기본)

- 모든 판정 봇 공개
- 플레이어가 GM의 굴림도 볼 수 있음

### 비공개 굴림 (선택)

- GM 비밀 정보 판정 (NPC 감지 등)
- 별도 DM·비공개 채널 사용
- 결과만 공지

### 플레이어 주사위 기록

- 중요 굴림 스크린샷
- 분쟁 시 참조

## 3-E. 플레이어 이탈·접속 장애 대응

### 5분 이내
- 잠시 대기
- 다른 PC 장면으로 전환

### 10분 이내
- 부재 PC NPC 처리 ("지금은 옆 방에 있습니다")
- 계속 진행

### 30분 초과
- 세션 저장 검토
- 플레이어 OOC 연락
- 필요 시 세션 조기 종료

## 3-F. 세션 후

### 일지·저장

- 로그 봇 자동 저장
- GM 요약 5분 작성
- 다음 세션 시작점 메모

### 다음 세션 예고

- "다음 세션에는 ___를 마주하게 될 겁니다"
- 플레이어 기대감 유도

## 3-G. ORPG 특화 NPC 연출

### 화면 공유

- NPC 일러스트 (AI 생성 가능)
- 장소 이미지
- 맵·다이어그램

### 사운드

- NPC 등장 시 BGM 변경
- 전투 시 긴장 음악
- 조용한 장면에 앰비언스

### 채팅 병용

- NPC 대사 채팅에 기록 (참조용)
- OOC 질문 채팅으로
- 중요 정보 채팅 고정

## 3-H. ORPG GM 체크리스트

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
ORPG GM 세션 전 1시간

[ ] 플랫폼·봇·맵·시트 점검
[ ] 세션 5단계 계획
[ ] NPC 3~5명 준비 (음성 톤 포함)
[ ] 주요 판정 TN 메모

[ ] 안전 툴 재확인
```
