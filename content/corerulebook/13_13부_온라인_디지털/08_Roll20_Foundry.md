# 13-8. Roll20·Foundry 세팅

---

ORPG·커뮤플레이에서 널리 쓰이는 두 플랫폼 — **Roll20**과 **Foundry VTT** — 의 FEWK 전용 세팅을 정리합니다.

## 8-A. Roll20 세팅

### 기본 기능

- 웹 기반, 브라우저만 있으면 접속
- 유료 (무료 티어 제한적)
- 캐릭터시트·맵·음성 통합

### FEWK 세팅 단계

```
1. 새 캠페인 생성
2. 시스템: "Custom" or "OGL" 선택
3. 캐릭터시트: 커스텀 HTML 시트 업로드 (FEWK v0.3 제공 예정)
4. 주사위 매크로:
   - /r 5d6!+3 (Grade Die + Ace)
   - /r 1d100 (d100 저항)
   - /r 10d6>4 (d6 Pool, count 5+ 유사)
5. 맵·토큰 준비
6. 플레이어 초대
```

### 추천 매크로 세트

```
#grade 5d6! + 3         → Grade Die 판정
#d100 1d100             → d100 저항
#pool 10d6>4            → d6 Pool 소모 (5+ 카운트)
#init /r (@{REF_grade} + @{SEN_grade})  → Initiative
```

## 8-B. Foundry VTT 세팅

### 기본 기능

- 자체 호스팅 (PC or 클라우드)
- 일회성 구매
- 광범위 모듈 지원

### FEWK 세팅 단계

```
1. Foundry 설치
2. 시스템: "Generic" or "Simple Worldbuilding"
3. FEWK 모듈 설치 (커뮤니티 제공 예정)
4. 캐릭터시트 커스터마이징
5. 주사위 매크로 설정
6. 맵 파일 업로드
```

### FEWK 전용 모듈 (계획 중)

- 자동 Grade Die 변환 (스탯 수치 입력 → 주사위 자동)
- Pool 게이지 UI
- 트랙 시각화 (샤이닝·매팅·패러독스·현화)
- Clock 관리 도구

## 8-C. 플랫폼 비교

| 기능 | Roll20 | Foundry |
|---|---|---|
| 가격 | 유료 구독 | 일회성 구매 (약 $50) |
| 호스팅 | Roll20 자체 | 자체 호스팅 필요 |
| 커스터마이징 | 중간 | 매우 높음 |
| 학습 난이도 | 낮음 | 중간 |
| 권장 인원 | 4~6명 | 4~8명 |
| 커뮤 | 국제 | 국제 + 일부 한국 |

- [ ] d6 Pool 매크로

- [ ] NPC 토큰

## 8-E. 플랫폼 간 캐릭터 이식

### Roll20 → Foundry

- JSON 내보내기
- Foundry에 변환 모듈 사용
- 매크로·주사위 재설정

### Foundry → Roll20

- 시트 PDF 생성
- Roll20에 수동 입력
- 매크로 재작성

### 공통 백업

- YAML 형식 권장 (§09)
- 플랫폼 독립
