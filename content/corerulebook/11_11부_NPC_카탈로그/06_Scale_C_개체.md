# 11-6. Scale C 개체 — 4대 신수·흑공해일·대형 장비

---

Scale C 개체는 FEWK 세계의 **거대한 것들**입니다. 신수, 흑공의 자연 현상, 대형 전투 장비. Scale B의 일반 전투와는 규칙이 다릅니다.

## 6-A. Scale C 정의

- **크기**: 건물 크기 이상
- **HP**: 100~500+ (다중 HP 블록)
- **공격**: 범위·고유 공격 다이스
- **방어**: 소구경 관통 저항 (Effect −1)
- **이동**: 25m+ 격자 단위

## 6-F. Scale C 전투 규칙

### 다중 HP 블록

```
개체의 부위를 나눠 각각 HP:
- 공격은 부위별 지정 가능
- 부위 파괴 시 효과 (이동 불가·공격 불가 등)
- 전체 HP 대신 주요 부위 HP
```

### 소구경 관통

- Scale A/B 소구경 무기: Effect −1 (Limited)
- Scale C 무기·범위: 정상

### 범위 공격

- 영향 반경 25m+ 격자
- PC 다수 동시 피해
- 회피 판정 각자

### 팀 전술

- 여러 PC 동시 공격 필수
- 부위별 역할 분담
- 원조·공동 행동 자주 사용

## 6-G. Scale C 조우 설계

### 등장 페이스

- 대아크당 1~2회 (보스급)
- 세션 자체의 중심 이벤트
- 준비·정찰 단계 포함 필수

### 승리 조건

- 주요 부위 모두 파괴 or
- 전체 HP 0 or
- 특수 조건 (4대 신수 조우 시 "항복 판정" 가능)

### 팀 자원 소모

- 대부분의 PC 자원 소모
- DSR·운 토큰·d6 Pool 전액 사용
- HP 심각 감소, 드랍아웃 리스크
