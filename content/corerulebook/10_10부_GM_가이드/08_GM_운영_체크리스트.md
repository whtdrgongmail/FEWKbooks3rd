# 10-8. GM 운영 체크리스트

---

GM은 테이블 앞에서 **수십 가지 정보**를 동시에 관리합니다. 체크리스트는 그 부담을 줄이고, 필수 사항 누락을 막습니다. 이 장은 세션의 네 단계 — **전·중·후·위기** — 각각에 대한 GM 체크리스트를 제공합니다.

## 8-A. 세션 전 (준비) 체크리스트

- [ ] 안전 툴 상태 확인 (Lines·Veils·X-Card)
- [ ] 플레이어 출석·결석 확인
- [ ] 지난 세션 요약 준비 (2~3분)
- [ ] 세션 목표 1~2개 (시나리오·서사)
- [ ] 주요 NPC 3~5명 준비 (이름·욕망·전술)
- [ ] 주요 장소 1~3개 세팅
- [ ] 예상 판정 TN 목록
- [ ] Faction Clock 진행도 갱신
- [ ] 자료 준비 (지도·사진·BGM)
- [ ] 온라인: 플랫폼·음성·다이서 봇 점검
- [ ] 치트 시트 인쇄 (물리) or 화면 (온라인)

## 8-B. 세션 중 (운영) 체크리스트

### 전투 운영

- [ ] Initiative 기록 (판·화면)
- [ ] HP 추적 (각 PC·주요 NPC)
- [ ] 상태 이상 기록 (출혈·중독·공포 등)
- [ ] Clock 진행도 (히트·Faction·패러독스)
- [ ] 자원 소모 기록
  - 운 토큰 (각 PC)
  - DSR (각 PC)
  - d6 Pool (각 PC)
  - 트라우마 (누적 시)
- [ ] 탄약·장비 손실
- [ ] 사기 체크 트리거 (NPC)
  - D급: HP 50% 시 도주 체크
  - C급: HP 25% 시 항복 체크
  - B급: HP 10% 시 최후의 저항

### 소셜·서사 운영

- [ ] 발언권 분배 (침묵 PC 개입)
- [ ] NPC 동기 일관성
- [ ] Check-in (감정 확인)
- [ ] 서사 리듬 (5단계 진행)
- [ ] 플레이어 창의 수용
- [ ] DSR 지급 기회 (드라마 순간마다 +1)
- [ ] 감정 무게 조절

## 8-C. 세션 후 (정리) 체크리스트

- [ ] 애프터매스 6단계 완료 (4-9 참조)
- [ ] PC 자원 상태 기록
  - HP·에고·DSR·Pool·운 토큰
  - 트라우마·트랙 변동
  - 히트 변동
- [ ] 다음 세션 훅 전달
- [ ] 세션 포인트 지급 (기본 4p + 이정표)
- [ ] 세션 로그 작성 (§11)
- [ ] Faction Clock 진행 반영
- [ ] 플레이어 피드백 수집
- [ ] 다음 세션 일정 확정

## 8-D. 위기 상황 체크리스트

### TPK (전원 사망·드랍아웃)

- [ ] 플레이어 동의 확인 (계속 / 전환 / 종료)
- [ ] 운 토큰 5 회피 옵션 제시
- [ ] 드랍아웃 vs 진사 선택
- [ ] 새 PC 생성 준비 (필요 시)
- [ ] 캠페인 전환 검토

### 플레이어 간 갈등

- [ ] 즉시 세션 중단
- [ ] X-Card 우선 처리
- [ ] 개별 대화 (세션 외)
- [ ] GM 중립 유지
- [ ] 합의 후 복귀 or 종료

### 규칙 해석 충돌

- [ ] 규칙서 확인 (이 Core Rulebook or 엔진 SSOT)
- [ ] 빠른 결정 (2분 이내)
- [ ] 플레이어 유리 원칙
- [ ] 세션 후 재검토

### 세션 초과

- [ ] 남은 시간 확인
- [ ] 절정 우선 (수습은 다음 세션)
- [ ] 세션 저장 절차 (§06)
- [ ] 복귀 포인트 메모

## 8-E. 1 페이지 요약 (부록 B 치트 시트)

부록 B 치트 시트에 압축 버전 수록:

### GM 체크리스트 1페이지

- **【전】** 안전 툴·출석·요약·목표·NPC·장소·TN·Clock
- **【중】** Initiative·HP·상태·Clock·자원·사기·발언·Check-in·DSR 지급
- **【후】** 애프터매스·자원 기록·훅·포인트·로그·피드백
- **【위기】** TPK·갈등·규칙·초과 각각 대응 프로토콜
