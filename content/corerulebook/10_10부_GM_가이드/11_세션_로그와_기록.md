# 10-11. 세션 로그·기록 관리

---

기록 없는 캠페인은 **기억의 캠페인**입니다. 짧으면 좋지만, 장기 캠페인은 기억의 한계를 초과합니다. 로그가 이야기를 지킵니다.

## 11-A. 세션 로그의 목적

1. **연속성**: 다음 세션 복귀 지점 확인
2. **복선 관리**: NPC·단서 일관성 유지
3. **감정 보존**: 돌아본 즐거움
4. **커뮤플레이 공유**: 다중 파티 간 정보 교환
5. **PBeM·게시판**: 아카이브 자체가 기록

## 11-B. 4 로그 유형

### 요약본 (반 페이지)

**용도**: 다음 세션 도입용, 빠른 복귀.

```
세션 N 요약:
- 일자: ___
- 주요 사건: ___ (2~3줄)
- PC 상태: HP / 에고 / DSR / 트라우마 요약
- 다음 시작점: ___
```

### 상세본 (1~3 페이지)

**용도**: 판정·자원·이벤트 시간순 기록.

```
세션 N 상세:
[시간] 사건
[판정] 판정값·결과
[자원 변화] ...
[NPC 등장] ...
[이벤트] ...
```

### 감정본 (자유 형식)

**용도**: 인상적 장면·대사·감정 모음.

"기억에 남는 순간"을 GM·플레이어가 짧게 적음.

### GM 비밀 로그 (플레이어 비공개)

**용도**: 복선·NPC 비밀·미래 이벤트 계획.

§9 NPC 비밀 파일과 연계.

## 11-C. 기록 서식 템플릿

### 세션 N 로그

**【헤더】**
- 일시: ___
- 회차: N
- 참여자: ___
- 플레이 시간: 3시간

**【요약 4줄】**
- 도입: ___
- 주요 행동: ___
- 전환: ___
- 수습: ___

**【자원 상태 스냅샷】**
- PC 민: HP 25/35, 에고 5/8, DSR 4/11, 운 7/10
- PC 수덕: ...
- PC 카이: ...

**【트랙 변동】**
- 민: 매팅 +1 (총 3)
- 수덕: 샤이닝 +1 (총 4)

**【히트 변동】**
- ㈜신성 히트: 3 → 4

**【남은 훅】**
- "청지기 백이 제안한 비밀 의뢰"
- "페이의 남편 단서 1개 추가"

**【다음 세션 시작점】**
- 민의 아파트, 다음 날 아침

## 11-D. 양식별 로그 권장

### 오프라인 세션

- GM + 서기 플레이어 1명
- 요약본 + 감정본 (간단)
- 세션 후 1~2일 내 작성

### ORPG (실시간 온라인)

- 자동 녹화·로그 봇 (Discord·Roll20·Foundry)
- GM 요약 5분 내 작성
- 로그 봇이 판정·대화 자동 기록

### 커뮤플레이

- 공유 위키·노션 주간 업데이트
- 파티 간 정보 교환 중심
- 진행자(Community Manager)가 통합

### PBeM

- 이메일 스레드 자체가 로그
- 월 1회 요약본 작성
- 플래시백·반전 시 전체 로그 재검토

### 게시판 플레이

- `[자료]` 스레드 자동 축적
- 주요 판정·이벤트 섹션 구분
- 관전자도 참조 가능

## 11-E. 아카이브·백업

### 백업 주기

- 주 1회 (권장)
- 세션 10 단위 마일스톤 저장
- 대아크 종료 시 전체 아카이브

### 플랫폼 이전

- 다른 툴로 이전 시 로그 포맷 변환
- 검색 가능한 형식 (markdown·txt·pdf)

### 장기 캠페인 요약

- 월 1회 또는 아크 종료 시 **대요약** 작성
- 10~30 세션 분량을 1~3 페이지로 압축

## 11-F. 로그의 플레이어 공유

### 공개 로그

- 요약본·상세본 (GM 비밀 제외)
- 다음 세션 전 배포
- 플레이어 복귀 보조

### 비공개 로그

- GM 비밀 (NPC 비밀·미래 계획)
- 개별 PC 비밀 (해당 플레이어만 공유)

## 11-G. 예시 — 세션 15 요약본

### FEWK 캠페인 "흑공의 그림자" — 세션 15 요약 (2026-04-23)

- 참여자: GM 수현 + 민(지우), 수덕(민재), 카이(서연)
- 플레이: 3시간

**【주요 사건】**
- ㈜신성 서울 지점 침투 성공
- 청지기 백 첫 대면 (B급 NPC)
- 서버룸 데이터 탈취
- 탈출 중 경비대 추격, 카이 리브 스트레이트로 따돌림

**【자원 상태】**
- 민: HP 22/35, 에고 4/8, DSR 7/11, 운 5/10
- 수덕: HP 30/35, 에고 10/14, DSR 5/15, 운 4/6
- 카이: HP 25/30, 에고 6/6, DSR 3/11, 운 8/10

**【트랙】**
- 수덕 샤이닝 5 (샤이닝 오라 발현 상태)
- 민 매팅 3 (흑공 감응 상태)

**【히트】**
- ㈜신성: 4 → 6 (파견 단계)
- 피스메이커: 1 유지

**【남은 훅】**
- 청지기 백이 "다음 번엔 대가를 치를 것"
- 탈취한 데이터에 "페이의 남편 이름" 발견

**【다음 세션】**
- 민의 아파트. 데이터 분석 시작.

이 로그가 세션 16 도입의 기초가 됩니다.
