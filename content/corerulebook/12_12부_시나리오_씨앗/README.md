# 12부. 시나리오 씨앗 (Scenario Seeds)

**예상 분량**: 40~55p (B5)

---

12부는 GM이 바로 굴릴 수 있는 세션 재료입니다. 짧은 도입 훅·3~4시간 원샷·이벤트 판정 테이블·캠페인 아크 템플릿. 각 씨앗은 FEWK 세계관의 6스타일지구와 주요 작전 연표를 활용합니다.

## 구성

| 순서 | 항목 | 요지 |
|---|---|---|
| 01 | [도입 훅 6선](01_도입_훅_6선.md) ◕ | 6스타일지구별 1개씩 |
| 02 | [원-샷 시나리오 3종 (3~4시간)](02_원샷_시나리오_3종.md) ◕ | 초보 GM용 즉석 운영 |
| 03 | [이벤트·판정 테이블](03_이벤트_판정_테이블.md) ◕ | 흑공 농도별·인과율 d100 |
| 04 | [캠페인 아크 템플릿](04_캠페인_아크.md) ◕ | 에피소드·아크·대아크 |
| 05 | [입문 시나리오 「신성의 카드」](05_입문_시나리오_신성의_카드.md) ◕ | 1세션 완성 시나리오 (첫 세션 가능) |
