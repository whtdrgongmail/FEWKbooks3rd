# 12-5. 입문 시나리오 「신성의 카드」

---

FEWK의 **첫 세션으로 최적**인 완성 시나리오. 1판 원전 Book3 17-1a를 v0.3 규칙으로 재작성.

## 5-A. 시나리오 개요

- **분량**: 3~4시간 (1 세션)
- **대상**: 신참 GM·PC (TRPG 첫 세션 가능)
- **배경**: 공단계 (23기업 (주)신성) — 표준 러닝 임무
- **주제**: 기업 카드 탈취 — 전형적 러너 업무
- **난이도**: 도전 (표준 러닝 경험)

## 5-B. 세션 0 체크리스트

시나리오 시작 전 세션 0 (30분~1시간):

```
[ ] 안전 툴 확인 (Lines·Veils·X-Card·Open Door)
[ ] 장르 톤 (사이버펑크 + 약한 오컬트)
[ ] 플레이어 기대치 (전투·수사·사회?)
[ ] PC 4명 추천 프리셋 배정
[ ] 기본 장비·팀 구성 합의
```

### 추천 PC 4명 (100p 프리셋)

1. **G-1 은밀자** — 잠입 담당
2. **G-3 반사 전투** — 경비 교전 담당
3. **G-4 네트워크** — 해킹·정보 담당
4. **G-7 사교 전문** — 브리핑·협상 담당

## 5-C. 5 단계 진행표

### ① 도입 (30분): 의뢰인 접선·브리핑

**장면**: 서민구 힐링카페 (소화 본점). PC들 접선.

**GM 서술 예시**:
> "오후 3시. 소화 이모의 힐링카페에서 국값을 치르고 있을 때, 옆자리에 앉은 여성이 조용히 말을 건다. '이 카페 주인이 추천하신 분들 맞죠? 제 이름은 페이입니다. 의뢰가 있습니다.'"

**의뢰 내용**:
- (주)신성 공단계 지점의 **금고 카드 1장** 탈취
- 지점장 강 이사의 개인 집무실에서
- 24시간 내
- 보수: 각 100 오얏 (팀 400 오얏)

**PC 결정**: 수락 / 협상 / 거절

### ② 전개 (60분): 공단계 잠입·정찰

**장면**: 공단계 빅포레스트 지역 → Pack v0.3 E-9 교정 반영 **캐치먼트 브라이트캐슬** (신성 본사).

**Note**: 세션 진행 중 "공단계 빅포레스트" → Pack v0.3 E-9에서 "캐치먼트 브라이트캐슬"로 본사 교정. GM 재량으로 선택.

**활동**:
- 다이브 판정 (INT + 차원 이해 vs TN 10)
- 공단계 도착. 흑공 농도 5. 매팅 저항 d100 (BP% 70% → 목표 40).
- 지점 외곽 정찰. 잠입 판정.
- 경비원 (D급) × 3 통과 (은밀 or 전투).

### ③ 전환 (30분): 예상치 못한 변수 — 히트 추적

**변수**: 또 다른 러너 팀 (경쟁자)도 같은 카드 노리는 중. C급 × 3.

**GM 서술**:
> "서버룸 앞 복도. 당신들은 이미 다른 침입자의 흔적을 발견합니다. 최근, 길지 않은 시간 전."

**선택지**:
- 충돌 → 전투
- 잠복 → 타이밍 경쟁
- 동맹 → 협상 (리스크)

### ④ 절정 (45분): 카드 탈취·탈출 전투

**장면**: 강 이사 집무실. 금고 해킹 시도.

**판정**:
- 해킹 판정 (INT + 해킹 vs TN 15)
- Partial 시 경보 작동 → 경비 출동
- Full 시 조용히 탈취

**경보 발동 시**:
- B급 청지기 출동 (Pack 12 §12-3-C)
- D급 경비 × 4 추가
- 탈출 대립 판정 다수

### ⑤ 수습 (15분): 보수 정산·다음 훅

**결말 갈래**:
- **성공**: 페이에게 카드 전달. 400 오얏 수령. 페이의 다음 의뢰 제안.
- **Partial 성공**: 카드는 있으나 추격자 히트 증가. 보수 절반.
- **실패**: 페이 실망. 향후 의뢰 제한.

**히트 변동**:
- (주)신성 히트: 0 → 3 (정보 수집 단계)

**다음 세션 훅**:
- 카드 내용에 "페이의 남편 이름" 있음
- 페이의 진짜 목적 단서

## 5-D. NPC·장소 카드

### 페이 (의뢰인, 브로커, C급)
- §02 C급 표준 예시 참조
- 욕망: "남편 찾기" (숨김)
- 외모: 40대 여성, 단정한 정장

### 강 이사 (B급, 임원)
- BOD 6, INT 8, PRE 8
- HP 30
- 집무실 함정 알고 있음
- 욕망: "개인 자산 보호"

### 청지기 백 (B급, 보안 책임자)
- §11-3 참조
- HP 40
- 전술: 체계적 방어

### 경비원 (D급, 다수)
- §11-1 참조

### (주)신성 캐치먼트 브라이트캐슬 (본사)
- 공단계 거점
- 흑공 농도 5
- 다층 보안
- Pack v0.3 E-9 교정된 지명

## 5-E. GM 운영 팁

### 판정 난이도

- TN 10~15 중심 (신참 친화)
- 운 토큰 여유 있게 제공
- Critical·Fumble 드라마 강조

### 드랍아웃 방지

- HP 관리 세심히
- 운 토큰 5 회피 안내
- 필요 시 D급 2명 감소 (조정)

### 시간 초과 시

- 절정 우선
- 수습 축약 (요약 서술)
- 다음 세션에 이어 가능

### 첫 세션 특수 조언

- Check-in 자주
- 규칙 설명 인내심
- 실수 허용 · 학습 중심

## 5-F. 세션 후 확장

이 시나리오가 성공하면:
- **단편 캠페인** (3~5 세션): 페이·(주)신성 테마 확장
- **장기 캠페인** (20~30 세션): 페이의 남편 비밀·FHI 연결

다음 시나리오 「흑공의 저택」 (§06)과 연결 가능 — 페이의 의뢰가 확장됨.
