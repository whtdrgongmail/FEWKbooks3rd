# 04-8. 도주·추격

---

모든 전투가 정면 대결로 끝나지는 않습니다. PC는 도주를 선언할 수 있고, 적도 도망칩니다. FEWK의 추격은 **연속 대립 판정**과 **Scale 전환**으로 표현됩니다.

## 8-A. 도주 선언

자기 턴에 "도주 시도" 선언 가능:
- 주요 행동 사용
- 최대 이동 거리 적용
- 추격자가 있다면 다음 라운드부터 추격 판정 시작

도주를 막으려면 **제압 공격**(§06) 또는 **물리 저지**(BOD 대립)로 붙잡아야 합니다.

## 8-B. 추격 판정 — 연속 대립 판정

추격은 **라운드마다 연속 대립 판정**입니다.

```
라운드마다:
- 추격자: REF + 운전 또는 운동 vs 차이
- 도망자: REF + 운전 또는 운동 vs 차이

차이 +N: 거리 N격자 (Scale 단위) 좁힘
차이 −N: 거리 N격자 벌림
차이 0: 거리 유지
```

### 거리 변화 규칙

- 초기 거리: Scale B 20m, Scale C 100m 등 (상황별)
- 거리 0m 도달: 근접. 추격 종료, 전투 재개.
- 거리 최대치 초과: 도망자 탈출. 추격 종료.

## 8-C. Scale 전환 (A → B → C)

추격은 Scale을 바꾸며 진행됩니다:

### Scale A → B 전환

- 조건: 건물 밖으로 튀어나옴, 좁은 골목에서 큰 거리로
- 효과: 격자 크기 변경 (1m → 5m). 거리 재측정.
- 이점: 차량 접근 가능.

### Scale B → C 전환

- 조건: 대로·고속도로 진입, 도시 경계 이탈
- 효과: 격자 크기 변경 (5m → 25m).
- 이점: 추격 전술의 대규모화 (헬기·드론·장비 활용).

### 전환 비용

자연 전환은 무료 (GM 선언). PC 주도 전환은 운 토큰 2 + GM 승인.

## 8-D. 장애물·지형 효과

추격 중 장애물이 등장:

| 장애물 | 판정 | TN |
|---|---|---|
| 낮은 장벽 | REF + 운동 | 10 |
| 지붕·옥상 | REF + 운동 | 15 |
| 좁은 다리·간격 | REF + 운동 | 15 |
| 차량 사이 | REF + 운전 | 15 |
| 군중·사람 틈 | REF + 은밀 | 10 |
| 흑공 농도 지대 | REF + 저항 | 15 + 농도 |

**성공**: 통과, 거리 유지 또는 +1 확보.
**실패**: 통과 실패, 거리 −1 또는 −2 (Partial 시 통과 + 대가).

## 8-E. 탈것

FEWK의 주요 탈것 (상세: 7부 §05 차량):

| 차량 | 속도 우위 | 특징 |
|---|---|---|
| 리브 스트레이트 (오토바이) | 대다수 차량 대비 +1 | 도심 추격 최강 |
| 옐부르스 | 고급 바이크 | 고속도로·거리 강세 |
| 일반 차량 | 기준 | 중간 |
| 산악보행트럭 | 지형 이점 | 비포장·산악 |

PC의 드라이버 태그·운전 스킬이 판정에 직접 적용됩니다.

## 8-F. 추격 종료 조건

- **거리 0m 도달**: 근접 전투 재개.
- **거리 최대치 초과**: 도망자 탈출.
- **한 쪽 Fumble**: 사고 발생 (1d6 피해 or 차량 고장).
- **장애물 치명 실패**: 추락·충돌 (심각 피해).
- **GM 재량**: 서사적 종료 (제3자 개입 등).

## 8-G. 예시 — 시가 추격

> **상황**: 민(리브 스트레이트, REF 7·운전 +5) vs 갱단 추격차(REF 5·운전 +3).
> 시작 거리: Scale B 30m.
>
> **라운드 1**:
> - 민: 5d6 + 5 = 굴림 20 + 5 = 25
> - 갱단: 3d6 + 3 = 굴림 11 + 3 = 14
> - 차이 +11 → 민이 11격자 (55m) 벌림. 현재 거리 85m.
>
> **라운드 2 (장애물)**:
> - 좁은 골목 진입. REF + 운전 TN 15.
> - 민: 5d6 + 5 = 22 → 성공, 추격차 TN 15: 14 → 실패. 사고 (−1 피해 + 거리 +2).
> - 현재 거리 105m → 최대치 (100m) 초과 → **민 탈출**.
