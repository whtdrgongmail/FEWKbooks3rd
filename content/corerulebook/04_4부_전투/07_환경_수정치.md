# 04-7. 환경 수정치·은밀·감지

---

전투는 진공에서 일어나지 않습니다. 조명, 날씨, 거리, 그리고 FEWK 특유의 **차원 환경**이 매 판정에 수정치를 더합니다. 이 장은 그 수정치 일람과 은밀·감지 대립 판정을 정리합니다.

## 7-A. 조명 수정치

| 상태 | 수정치 |
|---|---|
| 밝은 낮 | 0 |
| 흐린 낮 | 0 |
| 야간 (가로등) | −1 |
| 야간 (조명 없음) | −3 |
| 완전 어둠 | −5 |
| 눈부심 (역광·섬광) | −2 |

**임플란트 보정**: 시각 강화 Lv 2+ 보유 시 조명 페널티 절반.

## 7-B. 날씨·환경 수정치

| 상태 | 수정치 |
|---|---|
| 맑음 | 0 |
| 비 | −1 |
| 폭우 | −2 |
| 안개 | −3 (10m 이상 거리) |
| 눈 (적설) | −1 이동, −1 감지 |
| 폭설 | −3 시야 |
| 강풍 | −2 원거리 |
| 열대 고온 | −1 지구력 (긴 전투) |
| 극한 저온 | −1 모든 판정 |

## 7-C. 거리 수정치

유효 거리 초과 시 (§05 무기 Tier 참조):

| 상태 | 수정치 |
|---|---|
| 근거리 (권총) 초과 | −2 |
| 중거리 초과 | −4 |
| 원거리 초과 | −6 (또는 판정 불가) |

## 7-D. 차원 환경 — 흑공 농도

FEWK 고유 수정치. 흑공 농도 1~10이 판정에 직접 영향:

| 농도 | 모든 판정 | 매팅 저항 | 신기 판정 |
|---|---|---|---|
| 1~3 | 0 | +5 | +1 |
| 4~6 | −1 | 0 | 0 |
| 7~9 | −2 | −10 | −2 |
| 10 (원시) | −3 | −20 | −5 |

흑공 농도 규칙의 세부는 [6-1. 흑공 농도와 노출](../06_6부_FEWK_고유규칙/01_흑공_농도와_노출.md).

## 7-E. 엄폐 수정치

| 엄폐 | 수정치 |
|---|---|
| 엄폐 없음 | 0 |
| 부분 엄폐 (1/4) | +1 |
| 엄폐 (1/2) | +2 |
| 강한 엄폐 (3/4) | +3 |
| 완전 엄폐 | +5 (공격 불가에 가까움) |

엄폐는 방어 판정에만 적용됩니다. 공격 시엔 자신의 엄폐 밖으로 일시 노출됩니다.

## 7-F. 복합 환경

수정치는 **누적**됩니다. 각 축별로 계산 후 합산:

**예시 — 공단계 흑공 농도 7 야간 잠입**:
- 조명: 야간 조명 없음 = −3
- 차원: 흑공 농도 7 = −2
- 엄폐: 1/2 엄폐 = +2
- **총**: −3 + (−2) + 2 = **−3**

상황 페널티 누적 −15 이하면 판정 자체가 불가 (§3-6 수정치 상한).

## 7-G. 은밀·감지 대립 판정

### 은밀 (숨기·잠입)

- **판정**: REF + 은밀 스킬 + 환경 수정치
- **대상**: 모든 감지 자에 대한 대립 판정

### 감지 (발견·탐지)

- **판정**: SEN + 감지 스킬 + 환경 수정치
- **능동 감지**: 의도적 탐색
- **수동 감지**: 자동 판정 (GM 비공개)

### 대립 처리

두 판정의 차이값으로 결정:

```
은밀자 판정값 − 감지자 판정값 = 차이

양수: 은밀 성공 (은폐 유지)
음수: 감지 성공 (은밀자 발견)
차이 0: 경계 (감지자 수상함 느낌, 결정 불가)
```

### 은밀·감지 예시

> **상황**: 민(REF 7·은밀 +4)이 공단계 흑공 농도 7 야간 경비(SEN 5·감지 +3) 옆을 지남.
>
> **민 은밀**: 5d6 + 4 + (−3 야간) + (−2 농도) = 5d6 − 1
> 굴림: 3+5+2+6+4 = 20 − 1 = **19**
>
> **경비 감지**: 3d6 + 3 + (−3 야간) + 0 = 3d6
> 굴림: 4+2+5 = 11 → 차이 −3 (페널티 상쇄) = **11**
>
> **차이**: 19 − 11 = +8 → **은밀 성공**
> 민은 감지 없이 통과.

## 7-H. 환경 이벤트 통합

GM은 환경 수정치를 **장면별로 묶어** 관리합니다. 전투 시작 시:

```
환경 선언:
- 조명: ___
- 날씨: ___
- 차원 환경 (농도): ___
- 기본 엄폐 가용: ___
```

매 턴마다 각 PC가 자기 엄폐·위치에 따라 수정치 적용.
