# 4부. 전투 (Combat)

**예상 분량**: 55~65p (B5)

---

4부는 총과 신기가 교차하는 순간의 규칙입니다. 3 Scale (A 실내 · B 거리 · C 확장)로 공간을 나누고, 6 단계 턴 흐름으로 시간을 나눈 뒤, 피해·방어·상태 이상·환경 수정치로 결과를 산출합니다. 죽음·중상·드랍아웃의 경로도 이 단원이 결정합니다.
