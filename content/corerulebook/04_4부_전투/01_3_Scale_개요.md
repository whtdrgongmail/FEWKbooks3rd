# 04-1. 3 Scale 개요

---

FEWK의 전투는 **세 가지 공간 스케일**에서 벌어집니다. 같은 규칙의 세 변형이며, 상황에 맞는 스케일을 고르고 필요시 전환합니다.

## 1-A. 3 Scale 비교

| Scale | 거리 | 맵 단위 | 전형 상황 |
|---|---|---|---|
| **Scale A** | 실내·근접 (5~20m) | 1m 단위 | 복도·방·지하실·좁은 골목 |
| **Scale B** | 거리·조우 (20~200m) | 5m 격자 | 거리·광장·건물 외부 |
| **Scale C** | 확장·대형 (200m~) | 25m+ 격자 | Scale C 개체·탈것·대지 전투 |

같은 무기도 Scale에 따라 효과가 달라집니다. 권총은 Scale A·B에 효과적이지만 Scale C에서는 부적합 장비로 Effect −1단계.

## 1-B. Scale A — 실내·근접

- 5~20m 범위
- 밀집된 공간, 엄폐가 많음
- 근접 무기 유효 (단검·주먹 피해 배수 정상)
- 원거리 무기는 좁은 공간 페널티 없음, 유효
- 벽·장애물 활용 주요 전술

**전형**: 해킹 중 경비 돌입, 좁은 술집 싸움, 아파트 급습.

## 1-C. Scale B — 거리·조우

- 20~200m 범위
- 5m 격자 기본
- 원거리 무기가 주력 (중거리 페널티 권총 −2)
- 차량·추격 시작 가능
- 엄폐·거리 조절이 전술 핵심

**전형**: 거리 총격전, 기업 건물 외부 전투, 광장 소요.

## 1-D. Scale C — 확장·대형

- 200m 이상
- 25m+ 격자
- 저격·폭발 무기 주력
- Scale C 개체 (4대 신수·흑공해일·대형 장비)
- 소구경 무기 부적합 (Effect −1)

**전형**: 4대 신수 조우, 기업 본사 대규모 작전, 흑공해일 대응.

## 1-E. Scale 전환

### 자연 전환

상황 변화로 Scale이 바뀔 때 GM이 선언:
- 거리 전투 중 건물 진입 → Scale B → Scale A
- 건물에서 탈출 → Scale A → Scale B
- 거대 개체 등장 → Scale B → Scale C

### PC 주도 전환

운 토큰 2 + GM 승인으로 PC가 Scale 전환 선언 (§5-2 운 토큰 참조).

**예시**:
> 민: "Scale B 총격전 중. 운 토큰 2 써서 이 건물 내부로 돌입합니다. Scale A로 전환."
> GM: "합리적이네요. Scale A 시작."

### 전환 시 조정

- 이동 거리 재계산 (격자 단위 변경)
- 엄폐 상태 재평가
- 일부 무기 효과 변경 (Scale A→C 이동 시 권총 Effect −1 등)

## 1-F. Scale 간 무기 효과

| 무기 | Scale A | Scale B | Scale C |
|---|---|---|---|
| 단검·주먹 | 정상 | 불가 (근접 불가) | 불가 |
| 권총 | 정상 | −2 (중거리) | Effect −1 |
| 카빈·돌격총 | 정상 | 정상 | −1 (장거리) |
| 저격총 | −2 (근거리) | 정상 | 정상 |
| 수류탄 | 과도 위험 | 정상 | 정상 |
| 신기 무기 | Pack 지정 | Pack 지정 | Pack 지정 |
| 기관포 | −1 (과잉) | 정상 | 정상 |

Pack 11 무기 카탈로그(7부 §01)에서 각 무기의 최적 Scale 명시.

## 1-G. 예시 — 3 Scale 한 세션

> **장면 1 — 의뢰 수락**: 사무실. Scale A. 판정·대화 중심. 한두 번 굴림.
>
> **장면 2 — 이동·정찰**: 거리. Scale B. 은밀 판정·군중 속 NPC 만남.
>
> **장면 3 — 본 작전 침투**: 건물 내부. Scale A. 경비 전투 2~3 회차.
>
> **장면 4 — 발각·탈출**: 건물 외부로 터져 나감. Scale A → B 전환. 추격 전투.
>
> **장면 5 — 도심 추격**: 거리. Scale B 유지. 차량 등장.
>
> **장면 6 — 안전 지대**: 카페. Scale A. 애프터매스.

한 세션이 세 Scale을 모두 거치는 것이 일반적입니다.
