# 04-5. 무기 Tier (1d4~2d6)

---

무기의 기본 피해는 Tier별 다이스로 표현됩니다. 여기에 스탯·숙련 보너스가 붙고, 차이값 배수가 적용됩니다.

## 5-A. 무기 Tier

| 다이스 | Tier | 전형 | FEWK 예 |
|---|---|---|---|
| **1d4** | 소형 | 단검·주먹 | 무투가 기본 |
| **1d6** | 표준 | 권총 | 까마귀 P-21 (1d6+1) |
| **1d8** | 중간 | 카빈·돌격총 | 여우 S건 (1d8+2) |
| **1d10** | 중형 | 기관단총·마법 무기 | 만트라 건 (1d10) |
| **1d12** | 대형 | 저격총·격투 무기 | 무투가 연타 (1d12) |
| **2d6·3d4** | 다중·범위 | 수류탄·범위 신기 | 벽사종 (3d4) |

## 5-B. FEWK 6계열 체계 (Pack v0.3 E-2)

1판·wiki 원전 검증 통과한 6계열:

### M건 — 동력·물리 (Mechanical)
- 까마귀 P-21 (표준 권총) · 여우 S건 (카빈)
- 제조사: (주)액살·(주)뻑더레스 등
- 탄약: 물리 탄
- 특징: 신뢰성 높음·고장 낮음

### V건 — 소리·진동 (Vibration)
- 소리 총·초음파 무기
- 인체 내부 공진 피해
- 방어구 관통 특수

### S건 — 자기·전기 (Sonic/Static)
- 전자·전기 무기
- 임플란트 대상 추가 피해 +1d4
- 신기 교란 효과

### 광선포
- 빔 계열
- 고정·휴대형 혼재
- 대형 장비 (Scale C 주력)

### 전격총
- 무력화 무기
- 피해는 낮으나 마비·기절 확률 높음

### 서국 화약
- 구식 화약 무기
- 차원 체류 중에도 작동 (현대 탄약은 실패 가능)

### 특수: 크로코다일 Closing Pistol

30cm 권총. Scale A 귀속 (Pack v0.3 E-2 크로코다일 Scale C 오분류 교정).
- 피해: 1d8
- 유효: 50m 내
- 가격: 50,000+ 오얏

## 5-C. 신기 무기

만트라 건·벽사종·신단칩 연동 무기 (Pack 11 §11-3):
- **BP 0 예외**: 신기 무기는 임플란트 아님, BP% 영향 없음
- **신기 탄약**: 돌칩 소모
- **외부 휴대**: Pack Spec §4-5 외부 무기 3p 고정비 대상 가능

## 5-D. Scale별 무기 효과

Scale A/B/C별 효과는 §01 3 Scale 개요 표 참조. 요약:
- **권총류**: A·B 정상 / C에서 Effect −1
- **카빈·돌격총**: 전 Scale 유효
- **저격총**: A 근거리 페널티 / B·C 정상
- **수류탄·범위**: A 과도 위험 / B·C 정상
- **크로코다일**: A 귀속

## 5-E. 숙련도·유효 거리

### 숙련도 보너스 (피해에 적용)

```
피해 = 무기 다이스 + 스탯 수정치 + (스킬 ÷ 2 내림)
```

사격 스킬 +4이면 피해 +2 추가.

### 유효 거리

각 무기별 유효 거리 구간. 거리 초과 시 공격 판정 −2.

| 무기 | 근거리 | 중거리 | 원거리 |
|---|---|---|---|
| 까마귀 권총 | 5m | 20m | 50m |
| 여우 S건 | 10m | 50m | 150m |
| 저격총 | (페널티) | 100m | 500m |
| 만트라 건 | 5m | 20m | 60m (신기 집중) |

## 5-F. 탄약 소모

무기 사격 시 d6 Pool을 소모합니다. 상세는 §07 환경 수정치·탄약과 7부 §04 참조.
