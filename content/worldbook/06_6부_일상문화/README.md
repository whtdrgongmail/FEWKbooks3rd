# 6부. 일상·문화

**예상 분량**: 25p (B5)

---

4부가 공간을 펼치고 5부가 권력의 지도를 그렸다면, 6부는 그 위에서 사람들이 실제로 **어떻게 하루를 사는가**를 다룹니다. 궁정구의 종소리 알람과 노마드의 건조한 바람, 성인식이 결정하는 평생 진로, 호칭 하나로 드러나는 계급, 오얏죽 한 그릇의 위로, 그리고 DDOGG 이별식이라는 가장 조용한 의례까지. 이 부의 모든 항목은 FEWK가 **살아지는 세계**라는 사실을 일상의 결로 보여줍니다.

| 01 | [여섯 아침](01_여섯아침/) | 6스타일지구 주민 각각의 하루 — 알람에서 저녁까지 |
| 02 | [임플란트 문화](02_임플란트_문화/) | 성인식·생체순도·야메가 만드는 몸의 정치 |
| 03 | [언어와 일상](03_언어와_일상/) | 호칭 체계·계층별 화법·FEWK 특유의 완곡어 |
| 04 | [음식과 의복](04_음식과_의복/) | 오얏 중심 식문화·계층별 식탁·복장과 장신구 |
| 05 | [의례와 축제](05_의례와_축제/) | DDOGG 이별·결혼·장례·종교재단의 예배 |
