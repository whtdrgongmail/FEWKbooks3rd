# FEWK: World Book 제3판 — 동방의 하얀나라 백과사전

**FEWK 세계의 외형과 속을 다루는 공식 백과사전.** "읽기·창작·몰입·수집·소설 독자"를 위한, 비플레이어도 단독으로 열람 가치가 있는 월드북입니다.

저작은 하얀용 (WhtDrgon · 김동은) · MEJE Works Corp. 판형은 B5 양장 하드커버 약 440~470p을 목표로 합니다. 자매 권은 [FEWK: Core Rulebook 제3판](../corerulebook/) (규칙서).

> "읽기·창작·몰입·수집·소설 독자"를 위한 FEWK 세계의 공식 백과사전.

---

## 이 책이 다루는 것

FEWK 세계의 **외형과 속**을 한 권에 묶습니다.

- **외형** — 10대사국 · 23기업 · 8도(道) · 6스타일지구 · 러너 · 코핀 · 사이버펑크 거리
- **속** — 흑공 · 신기 · 현화 · 차원 · 바이트 · 에고 같은 FEWK 자연 법칙

FEWK는 "가상(?)세계"라는 물음표의 세계입니다. VR·매트릭스처럼 보이는 이 공간은, 그러나 **실재하는 차원 밀도의 공학적 결과**입니다. 본 백과사전은 그 이중성을 유지하며 기술합니다.

## 사용 안내

### 어떻게 읽는가

- **처음 방문하는 독자**: [00 서문](#worldbook/00_서문/README) → [1부 세계 톤](#worldbook/01_1부_세계톤/README) → 관심 가는 부로
- **FEWK 플레이어**: 관심 주제를 직접 검색하거나 해당 부로 이동
- **세계관 작가·팬**: 궁금한 항목 바로 링크 이동
- **GM·제작자**: 부록의 용어집·색인에서 출발 → 필요한 항목으로 진입

### 각 페이지의 구조

대부분의 항목 페이지는 다음 세 부분으로 구성됩니다.

1. **정보 (위키)** — 구조화된 사실
2. **산문 (책 원고 재료)** — 문학적 서술
3. **교차 참조** — 관련 항목·법칙·원전

### Core Rulebook과의 분담

본 백과사전은 **세계관**을 다루고, 게임 규칙·수치는 자매 권 [Core Rulebook](../corerulebook/) 에서 찾을 수 있습니다.

- 본 책: "흑공은 어떻게 보이는가"
- Core Rulebook: "흑공 농도 X단계 시 환경 수정치 -Y"

수치 기재가 필요한 곳은 **"→ Core Rulebook §N-N 참조"** 로 위임합니다.
