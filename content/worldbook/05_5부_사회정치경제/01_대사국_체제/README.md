# 01장. 대사국 체제

FEWK의 정치는 10대사국(10大使國)의 분할통치로 작동합니다. 어느 한 국가도 단독으로 동백국 전체를 지배하지 않으며, 10개의 주권이 느슨한 공존 속에서 질서를 유지합니다. 이 장은 그 체제의 원리와 10개 국가 각각의 성격을 차례로 다룹니다.
