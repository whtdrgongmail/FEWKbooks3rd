# 02장. 비인간 존재

---

FEWK에는 인간이 아닌 존재가 여럿 있습니다. 흑공에서 태어난 생물, 이름조차 붙일 수 없는 미지의 존재, 흑공을 흡수해 변한 전직 인간, 그리고 인간이 만든 의사(擬似) 존재들. 이 장은 이 네 범주를 한자리에 펼쳐 봅니다.
