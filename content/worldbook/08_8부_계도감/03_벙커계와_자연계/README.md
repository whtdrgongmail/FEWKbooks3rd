# 03장. 벙커계와 자연계

---

인공계 너머에는 **벙커계**라는 독특한 지하 네트워크와 여러 **자연계**가 펼쳐져 있습니다. 벙커계는 동쟁 시대의 유산으로 만들어진 인공계이지만, 수천 개의 노드가 연결된 독립 생태로 발전해 별도의 장으로 분류됩니다. 자연계 중에서는 자원 풍부한 버그계, 개인용 월드 임플란트 차원계인 재민 월드, 그리고 가장 어두운 불법 차원계 언카날 밸리를 다룹니다.
