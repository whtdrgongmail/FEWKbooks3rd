# 04장. 오지 차원계

---

대사국이 공식 지도에 올리지 못한, 개척이 미완료되거나 개척 자체가 금기시된 **오지 차원계(Uncharted Dimensions)**가 있습니다. 이 장은 원전에서 보고된 대표적인 다섯 오지 차원계를 하나씩 소개합니다. 각 계의 풍경은 낯설고 아름답고 동시에 위험하며, 그 조합이 러너 문학의 가장 풍부한 재료를 제공합니다.

이 다섯 계는 모두 **최초 진입 비용 1,000+ 오얏** 이상이며, 파이오니아 수당·뇌네 보정 비용·보험을 더하면 수천~수만 오얏 규모의 임무가 됩니다(→ §01 이동 비용과 경로). 그만큼 탐사 성공 시 보상도 최상위이며, 이 계들의 귀환 기록은 대사국·기업·종교재단 모두에게 값비싼 정보 자산입니다.
