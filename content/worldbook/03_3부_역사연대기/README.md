# 3부. 역사·연대기

**예상 분량**: 40p (B5)

---

2부가 FEWK 세계의 법칙을 다뤘다면, 3부는 그 세계가 현재의 모습에 이르게 된 과정을 다룹니다. 검은 수평선이 있기 전의 시대로부터 지금 우리가 있는 수평 144년까지, 이 땅을 거쳐간 변곡점과 사람들의 이야기입니다.

본 부는 과거의 기록인 동시에 현재의 배경입니다. 이후의 모든 부 — 지리, 사회, 러너 생태, 세력 도감 — 에 등장하는 모든 요소의 뿌리가 여기 있습니다.
