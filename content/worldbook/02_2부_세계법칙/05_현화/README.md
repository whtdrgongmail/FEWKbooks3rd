# 05장. 현화(玄化) — 흑공과 존재의 반응

현화는 FEWK 세계에서 가장 드라마틱한 법칙입니다. 흑공과 주체가 만날 때 벌어지는 반응 — 상상이 실체가 되고, 사람이 괴물이 되거나 영웅이 되는 — 이 일어나는 과정입니다. 이 장은 현화의 원리, 두 갈래(샤이닝·매팅), 그리고 기록자의 눈을 통해 수집된 구체 사례를 다룹니다.
