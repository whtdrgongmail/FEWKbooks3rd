# 02장. 신기(神氣) — 하얀 연기

흑공이 검은 연기라면, 신기는 하얀 연기입니다. 동백국의 이름 — "동방의 하얀나라" — 이 이 연기에서 나왔습니다. 이 장은 신기가 어떤 에너지이고, 어디서 솟아나며, 어떻게 활용되는지를 다룹니다.
