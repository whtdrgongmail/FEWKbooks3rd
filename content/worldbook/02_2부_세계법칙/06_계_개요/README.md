# 06장. 물질계와 차원계 — 계(界)의 개요

FEWK 세계는 하나의 세계가 아닙니다. 물질계와 그 이웃한 여러 차원계가 복잡한 지도를 이룹니다. 이 장은 그 지도의 개괄입니다. 상세한 차원계 지리는 8부 계도감에서 다루지만, 여기서는 "몇 가지 계가 있으며 각각 어떤 성격인가"의 밑그림을 제시합니다.
