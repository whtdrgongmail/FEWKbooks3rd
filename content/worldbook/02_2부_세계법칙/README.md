# 2부. 세계 법칙 — 흑공·신기·현화·인과율

> 검은 수평선 사건 이후, 흑공과 신기를 기반으로 바이트와 계의 시대가 열렸다.
>
> 비논리적인 미지의 학문인 흑학과 신학, 마법 같은 기술의 발달이 이뤄졌지만 — 여전히 인과율이 질서를 유지한다.
>
> 인과율을 어기면 패러독스가 발생하고, 패러독스는 대자연에 의해 반드시 보정된다. 작게는 손저림이나 두통, 머피의 법칙. 그러나 문제가 커지면 지역 전체에 화산폭발, 지진, 해일이 일어나 패러독스를 묻어버린다.
>
> 그 이전에 — 능력자들과 러너들의 공포의 존재, 일명 먹개비가 등장한다. 본래의 이름은 있지만, 인식하는 것만으로도 패러독스가 발생하기 때문에 그저 '먹개비'라 불린다.
>
> — *FEWK Core #0040 「인과율」*

---

2부는 이 세계가 작동하는 자연 법칙을 다룹니다. 거리에서 마주치는 임플란트 러너와 신단의 신사가 같은 물리 법칙 안에서 살고 있는 이유, 흑공이 의지에 반응하는 이유, 신기가 동방의 백국에만 존재하는 이유, 인과율이 패러독스를 묻어 보정하는 방식 — 이 세계의 모든 변덕이 자의적인 것이 아니라 한 덩어리의 법칙에서 나온다는 사실을 정리하는 자리입니다.

본 부가 다루는 것은 **법칙의 서사**입니다. 같은 법칙의 **수치** — 농도 단계·트랙 수치·판정 적용 — 는 자매 권 FEWK Core Rulebook(특히 6부)이 담당합니다. 본 부에서는 "흑공은 어떻게 거리를 물들이고 어떻게 사람을 바꾸는가"를 말하고, Core Rulebook이 "흑공 농도 몇 단계에 환경 수정치 얼마"를 제공합니다 — 두 책은 같은 법칙을 두 해상도로 기술하며, 함께 펴실 때 비로소 한 그림이 됩니다.

2부의 톤은 차분하면서도 명확합니다. 흑공·신기·인과율·바이트·현화·계·핵심기술 — 일곱 장 각각이 하나의 큰 법칙을 다루지만, 모두 **이 세계가 어떻게 보이는가**의 답을 제공합니다. 한 자리의 법칙이 다른 자리의 법칙과 어떻게 맞물리는지 — 흑공이 신기와 만나는 자리에서 현화가 일어나고, 그 현화를 인과율이 보정하며, 그 보정의 손이 닿지 않는 자리에서 먹개비가 등장하는 — 그 한 덩어리의 그림이 2부의 목표입니다.

이 부의 일곱 장을 다 읽으신 다음에는, 이 세계의 거리에 서 있는 한 인물이 무엇을 두려워하고 무엇을 욕망하는지가 자연스럽게 짐작이 됩니다. 그것이 2부가 후속 부들에게 건네는 토대입니다.

## 구성

본 부는 7개 장으로 나뉩니다. 각 장은 독립적으로 읽을 수 있으며, 관련 주제는 서로 교차 참조됩니다.

| 순서 | 장 | 요지 |
|---|---|---|
| 01 | [흑공(黑空) — 검은 연기](01_흑공/) | 상상과 반응하여 물질을 만드는 검은 구름 |
| 02 | [신기(神氣) — 하얀 연기](02_신기/) | 신단에서 솟아나는 동백국 고유의 에너지 |
| 03 | [인과율 — 세계의 질서](03_인과율/) | 패러독스와 먹개비가 집행하는 대자연의 법칙 |
| 04 | [바이트 — 만물의 척도](04_바이트/) | 물질·정보·혼백을 측정하는 단일 단위 |
| 05 | [현화(玄化) — 흑공과 존재의 반응](05_현화/) | 상상이 실체화되는 과정, 괴물과 능력자의 탄생 |
| 06 | [물질계와 차원계 — 계의 개요](06_계_개요/) | FEWK 세계가 갖는 여러 계(界)의 분류 |
| 07 | [핵심 기술 개관](07_핵심기술/) | 와치엔진·복제기·코핀이 돌아가는 원리 |

## 읽는 순서

처음 읽는 독자는 01 흑공 → 02 신기 → 03 인과율 → 04 바이트 순서로 읽으시길 권합니다. 이 네 장은 FEWK 자연 법칙의 뼈대이며, 이후 장들의 기반이 됩니다. 그다음 05 현화는 흑공과 존재의 상호작용을 다루므로 01·04를 읽고 보시면 더 풍부하게 이해됩니다. 06 계 개요와 07 핵심 기술은 앞 장들의 응용으로, 순서 없이 필요에 따라 읽으셔도 좋습니다.

특정 주제에서 출발하고 싶은 독자는 이 표에서 바로 해당 장으로 이동하시면 됩니다.

## FEWK Core Rulebook와의 경계

본 부의 법칙은 FEWK Core Rulebook의 아래 섹션과 짝을 이룹니다.

- 흑공 환경 수정치·농도 표 → FEWK Core Rulebook §5-12 (Pack 01 §1-10)
- 신기 셀·충전 규칙 → FEWK Core Rulebook §6-6 (Pack 11 §11)
- 패러독스 트랙 → FEWK Core Rulebook §6-10 (Pack 08 §8-3)
- 샤이닝·매팅 트랙 → FEWK Core Rulebook §6-9·§6-11 (Pack 08 §8-4)
- 바이트 환산·화폐 → FEWK Core Rulebook §8-13 (Pack 01 §1-11)
- 임플란트·동력 → FEWK Core Rulebook §8-11·Pack 11

수치가 필요한 순간 FEWK Core Rulebook를 함께 펼치시면 됩니다.
