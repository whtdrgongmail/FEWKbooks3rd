# 05장. 장비와 기술

---

러너의 몸에는 임플란트가 있고, 허리에는 총이 있으며, 발 밑에는 오토바이가 있고, 집에는 코핀이 있습니다. 이 네 가지가 러너의 기본 도구이며, 각각이 러너 세계의 기술·문화·경제와 얽혀 있습니다. 이 장은 러너가 쓰는 장비들을 하나씩 살펴봅니다.
