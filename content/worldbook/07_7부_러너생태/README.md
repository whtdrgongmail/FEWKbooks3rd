# 7부. 러너 생태

**예상 분량**: 30p (B5)

---

6부가 대사만국 시민의 평범한 하루를 따라갔다면, 7부는 그 하루의 경계를 넘어 일하는 사람들 — **러너(Runner)** 의 내부로 들어갑니다. 러너는 차원을 넘나드는 자이자, 기업·대사국·종교재단의 공식 경로 바깥에서 의뢰를 처리하는 자이며, FEWK 경제의 뒷면을 실제로 움직이는 자들입니다. 이 부는 러너를 "누구인가"에서 시작해 "어떻게 일하고, 어디에 모이고, 무엇을 타며, 어떻게 은퇴하는가"까지 따라갑니다.

모든 러너는 다르지만 몇 가지를 공유합니다. 러너전(走者錢)이라 불리는 신용 기반 화폐를 주고받고, 레드 몽키 클럽·힐링카페·식스센스 같은 거점에 들르며, 동력총·소리총·자기총의 세 종류 중 하나를 자기 총이라고 이름 붙여 부르고, 숙련된 야메 한 명을 평생의 비밀로 삼습니다(→ 6부 §02 임플란트 문화). 이 공통의 결이 러너라는 직업을 하나의 **생태계**로 만듭니다.
