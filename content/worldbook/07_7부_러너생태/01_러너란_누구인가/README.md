# 01장. 러너란 누구인가

---

"러너(Runner)"라는 한 단어 안에는 서로 다른 여러 정체성이 포개져 있습니다. 차원을 넘나드는 자, 의뢰를 처리하는 자, 정보를 파는 자, 위험한 해결사, 기업 바깥의 프리랜서, 예술가 출신의 관찰자 — 이 모든 얼굴이 같은 이름으로 불립니다. 이 장은 러너를 정의에서 시작해 그 분류·기원·사회적 위치·진입 경로까지 따라갑니다.
