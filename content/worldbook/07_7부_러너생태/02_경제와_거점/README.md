# 02장. 경제와 거점

---

러너의 경제는 FEWK 공식 경제의 바깥에서 독립적으로 돌아갑니다. 러너전(走者錢)이라는 신용 기반 화폐가 있고, 의뢰와 보수를 중개하는 브로커 네트워크가 있으며, 그 경제가 물리적으로 흐르는 자리 — 카페와 클럽 — 이 있습니다. 이 세 축이 러너 생태의 운영 체계입니다.
