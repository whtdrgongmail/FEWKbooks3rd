# 04장. 조직과 갱단

---

러너는 홀로 일하기도 하고 팀을 이루기도 하며, 어떤 경우에는 갱단에 속해 공동 작전을 수행하기도 합니다. 이 장은 러너가 협업하는 세 가지 형태 — **솔로와 팀, 갱단, 신디케이트** — 를 정리합니다. 각 형태는 장단점이 명확하며, 한 러너가 평생 한 형태에만 머무는 일은 드뭅니다.
