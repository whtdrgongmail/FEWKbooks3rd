## 버전 정책

1. 이미 공개된 버전의 **CC BY 4.0 자유는 철회 불가**. 향후 버전 변경이 있어도 각 버전은 영구 유효.
2. FWOL 본문(루트: `content/license.md`)은 현재 최신 v2.1을 담고 있으며, 향후 버전 변경 시 이 디렉토리에 새 스냅샷을 추가합니다.
3. **버전 넘버링 규칙**:
   - 메커니즘·권리의 구조적 변경 → major 번호 증가 (v1.0 → v2.0)
   - 새 권리·새 조항 신설 → minor 번호 증가 (v2.0 → v2.1)
   - 거버넌스·연락처·표현 보강 → 같은 minor 안에서 개정 기록 (문서 footer 및 CHANGELOG에 명기)
4. 과거 버전 하에 공개된 FEWK 세계관 작품은 공개 당시의 조건을 계속 유지할 수 있으며, 원할 경우 최신 버전으로 재배포할 수 있습니다.
