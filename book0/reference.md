# 5. 빠른 참조표

> *FEWK LOREBOOK — Book 0: Quick Start*
> 관련 항목: [[book0/rules]]

---

| 항목 | 규칙 |
|------|------|
| 판정 | `2d6 + 스탯 수정치 + 스킬 ≥ 난이도 (6/8/10/12)` |
| 공격 | `1d20 + AB + 수정치 + 스킬 ≥ AC` |
| 치명 | 치명 주사위(폭발) ≥ 6 → 피해 ×2 |
| HP 0 | → 돌파. 이후 체(BOD) 직접 피해. |
| BOD 0 | 차원계 = 드랍아웃 / 물질계 = 진사 |
| 잡졸 | HP 1. 한 발에 쓰러짐. |
| 푸시 | 실패 시 재굴림. 반드시 대가. |
| 운 토큰 | 재굴림 / 피해 반감 / 치명 무효. |
| 환경 태그 | 활성화 시 +2 보너스. |
| 1 오얏 | = 1끼 식사 = 최저시급 1시간 |
