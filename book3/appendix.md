## A-1. 밸런스 프레임워크 -- 위협 등급

| 등급 | 예시 | HP | AB | 대응 PC 레벨 |
|------|------|----|----|-------------|
| 0 | 일반 시민, 동물 | 1–3 | +0 | Lv1 이하 |
| 1 | 거리 깡패, 하급 경비 | 1 | +1 | Lv1–2 |
| 2 | 기업 경비, 레인저 | 8–12 | +2–+3 | Lv2–4 |
| 3 | 수호사, 신전사 | 12–18 | +3–+5 | Lv4–6 |
| 4 | 엘리트 흑전사, 대신수 | 18–30 | +5–+7 | Lv6–8 |
| 5 | 전설급 NPC, 미명존재 | 30+ | +7+ | Lv8–10 |

잡졸(위협 0–1): HP 1. 어떤 피해든 전투 불능. 사기 체크로 도주.
네임드(위협 2+): 정규 HP. 치명타, 중상 테이블 적용.

### 전투 조합 예시

- Lv1 PC 4명 vs 잡졸 6명: 쉬움. 2라운드 이내.
- Lv3 PC 4명 vs 잡졸 4명 + 네임드 1명: 적정. 3라운드.
- Lv5 PC 4명 vs 네임드 2명 + 잡졸 8명: 도전적. 4–5라운드.
- Lv7 PC 4명 vs 위협 4 네임드 1명: 보스전. 5–7라운드.

## A-2. GM 체크리스트 -- 전투 운영

### 전투 시작 전

- [ ] 물질계인가 차원계인가? -> 사망 모드 결정.
- [ ] 환경 태그 1–3개 설정.
- [ ] 활성 시계 선택 (미션 태그 기반).
- [ ] 적 NPC: 잡졸 / 네임드 구분.
- [ ] 잡졸 사기치 설정.
- [ ] 네임드 NPC에 운(LCK) 토큰 부여 (보통 1–2개).

### 전투 중

- [ ] 매 라운드 시계 업데이트.
- [ ] 사기 체크 트리거 확인 (첫 사상 / 절반 쓰러짐 / 리더 쓰러짐).
- [ ] 푸시 대가 적절성 판단.
- [ ] 신기 사용 시 셀 소모 추적.
- [ ] 치명타 시 폭발 주사위 처리.
- [ ] 환경 태그 활성화 시 +2/-2 적용.

### 전투 종료 후

- [ ] 갑옷 HP 자동 복원.
- [ ] 허약 상태 부여 여부.
- [ ] 히트(Heat) 누적 판단 (과잉 살상 시).
- [ ] XP 부여 (전투가 아닌 미션 완수 기준).
- [ ] 시계 잔여 칸 -> 다음 전투에 이월 여부.

## A-3. 흑공분출 환경 변이 테이블 (d6)

| d6 | 변이 | 효과 |
|----|------|------|
| 1 | 지형 변이 | 바닥 늪화/벽에서 가시/천장 녹음. 이동력 절반. 새 환경 태그 [변이 지형] 자동. |
| 2 | 현화 파동 | 반경 10m 전원 현화 판정. 기연 내성 실패 시 매팅 +1. |
| 3 | 신기 역류 | 신사 신기용량 일시 x2 (1라운드). 다음 라운드 0 리셋. |
| 4 | 괴물 출현 | 현화 괴물 1d4체 출현 [잡졸 HP 1]. 무차별 공격. |
| 5 | 시야 차단 | 흑연기 확산. 10m+ 시야 0. 사격 불가. 1d4 라운드. |
| 6 | 소강 | 특이사항 없음. 분출 시계 +1 면제. |

## A-4. 중상 테이블 (d12)

같은 전투에서 치명타 + 돌파가 겹칠 때. 체력 내성 실패 시 굴림.

| d12 | 결과 | 효과 |
|-----|------|------|
| 1 | 즉사 | 이야기가 여기서 끝. |
| 2 | 내부 손상 | 1시간 내 치료 필요. 코마. |
| 3 | 뇌 손상 | 코마. 임플란트 치료 필요. |
| 4 | 시력 상실 | 사이버아이 필요. |
| 5 | 내장 파열 | 체(BOD) 수정치 -1 (치료 전까지). |
| 6-7 | 사지 파괴 | 해당 부위 임플란트 교체 필요. |
| 8-9 | 임플란트 파손 | 가장 비싼 임플란트 1개 고장. |
| 10 | 에고 손상 | 에고 -2(영구). 매팅 트랙 +1. |
| 11-12 | 피부만 찢어짐 | 흉터가 남지만 기능 손상 없음. |

## A-5. 미션 보상 가이드

| 미션 규모 | XP | 보수(오얏/인) | 히트 위험 |
|-----------|------|------------|----------|
| 소규모(배달 등) | 1–2 | 100–300 | 낮음 |
| 중규모(조사/잠입) | 3 | 500–1,000 | 중간 |
| 대규모(기업/전투) | 5 | 1,000–3,000 | 높음 |
| 도시급(위기 대응) | 7–10 | 3,000–10,000 | 최고 |

- 전투 회피 보너스: 전투 없이 해결 시 XP +1.
- 창의적 해결 보너스: GM 판단으로 XP +1–2.
- 히트 누적: 민간인 사상 +2, 건물 파괴 +1, 기업 적대 +3.
