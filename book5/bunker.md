# Ch.10 벙커계

[[book5/index|목차로 돌아가기]]

---

벙커(Bunker)는 동쟁시대에 만들어져
지금까지 곳곳에 남아 있는
인공계 게이트다.

모든 벙커는 사실 하나의 인공계의
공간을 사용하는 시설이다.
동쟁시대 이후 만들어진 적이 없는
특별한 종류의 차원 시설이며,
아직 발견되지 않은 벙커가 더 많다.

지금까지 발견된 가장 큰 번호:
5000대. 하지만 전부 찾지도 못했다.

---

## 벙커 탐사 가이드

| 항목 | 내용 |
|------|------|
| 환경 | 밀폐 공간. 조명 없음. |
| 공기 | 보통(밀봉 유지 시) 또는 유독(손상 시 흑공 유입) |
| 흑공 | 미량–보통 (벙커에 따라) |
| 위험 | 동쟁시대 자동 방어 시스템, 내부 붕괴, 흑공 존재 서식 |

### 준비물

- 야시 고글
- 반딧불 군체
- 필터 마스크
- 만능 공구
- 스마트 밧줄
- 오방색 결계끈 (야영 시)

### 보상

- 동쟁시대 유물
- 기술 데이터
- 두더지기관차 같은 대형 장비
- 또는 아무것도 없을 수 있다.

## 벙커379

리버힐 지방의 기술 고등학교
학생들이 발견. 내부에 철로를 가진
지중열차 정거장. 두더지 기관차가
다 썩은 상태로 발견되었다.
토리가 신호연결로 기관차를 작동시켰고
땅속으로 폭주했다.
— 시나리오 시드로 사용 가능.
