## GM 스크린용 참조표

### 흑공 농도별 장비 요구

| 흑공 농도 | 필요 장비 |
|----------|----------|
| 미량 | 없음 (장기 체류 주의) |
| 보통 | 필터 마스크 권장 |
| 높음 | 필터 마스크+스킨 필수 |
| 위험 | 보호복 필수 |
| 치명 | 환경복 필수 |

### 흑공 농도별 매팅 판정

| 흑공 농도 | 판정 주기 | DC |
|----------|----------|-----|
| 미량 | 8시간마다 | DC 10 |
| 보통 | 4시간마다 | DC 12 |
| 높음 | 2시간마다 | DC 14 |
| 위험 | 1시간마다 | DC 16 |
| 치명 | 30분마다 | DC 18 |

### 매팅 경감

| 수단 | 효과 |
|------|------|
| 필터 마스크 | DC -2 |
| 필터 스킨 | DC -1 |
| 향로 | DC -2 |
| 무복 | DC -1 |

### 매팅 트랙

| 트랙 | 상태 |
|------|------|
| 0–4 | 정상 |
| 5–6 | 현화 시작 (피부 검은 점) |
| 7 | 현화 진행 (가역 한계) |
| 8–9 | 비가역 (치료 불가) |
| 10 | 완전 현화 (인간이 아님) |

### 사건 판정

매 4시간마다 d6.
5–6이면 d66 사건 테이블 참조.

### 계 이동 비용

| 이동 수단 | 비용 |
|-----------|------|
| 게이트 | 10–50 오얏 |
| 공용 코핀 | 100 오얏 |
| 원거리 코핀 | 300 오얏 |
| 히어로폰 | 200+ 오얏 |
| 미탐험계 | 1,000+ 오얏 |
| 화물/kg | 5–50 오얏 |

### 뇌네 보정

- 미보정 시 적대도 +2단계 인식.
- 보정 비용: 시간 + 파이오니아 기술.
- 보정 후에도 환경 위험은 그대로.
