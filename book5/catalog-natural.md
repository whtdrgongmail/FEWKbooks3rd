## 버그계 (Bug界)

| 항목 | 내용 |
|------|------|
| 개척 | 홀로국 |
| 유형 | 자연계 |
| 대기 | 농밀(습도 극고) |
| 중력 | 0.8G (경) |
| 온도 | 35–45℃ (혹서) |
| 수분 | 습윤 |
| 흑공 | 미량 |
| 지형 | 밀림 |

곤충 몬스터로 가득한 정글 세계.
엑토플라즘의 원료인 만월 고무라는
특수 소재를 채굴하는 지역.
벌목장과 개척지가 존재한다.

| 항목 | 내용 |
|------|------|
| 자원 | 극대 (만월 고무, 엑토플라즘) |
| 위험 | ★★★★☆ |
| 뇌네 보정 | 완료 (곤충은 여전히 크다) |
| 인간 거주 | 채굴 기지 3개소 |

> 시나리오 훅:
> (주)초록손이 새 만월 고무 광맥을
> 발견했으나 채굴팀이 실종됐다.
> 러너 팀에게 수색 의뢰.

## 재민 월드 (Jemin World)

| 항목 | 내용 |
|------|------|
| 유형 | 인공계 (개인, 월드 임플란트) |
| 대기 | 표준 |
| 중력 | 1G |
| 온도 | 열대 (해변) |
| 흑공 | 무 |
| 지형 | 리조트 + 미탐험 내륙 |

가난하게 태어난 재민이
"죽지 말고 드랍하세요" 광고를 보고
월드 임플란트를 이식하여 만든
개인 차원계.

해변에 리조트 "재민 월드"가 있고,
바다에 유람선 "재민호"가 떠 있다.
레디메이드 의지 30명과 함께 산다.

레디메이드 킴에게 아이가 들어섰다.
재민 월드의 첫 네이티브가 될 것이다.

내륙 20만 미터 안쪽에는
회사가 정한 금지구역이 있고,
거대한 산 너머가 궁금하다.

> 시나리오 훅:
> 재민이 금지구역으로 떠났고
> 돌아오지 않는다.
> 킴이 러너에게 수색을 의뢰한다.
> — 고용주는 레디메이드 의지.
